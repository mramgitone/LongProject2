  • Set the Timing for Frames In a movie, a frame is displayed for a certain period of time. When you record a movie, the timing of frames is automatically added, so that one frame displays for a certain number of seconds before it disappears and the next frame appears. However, you can adjust how much time a given frame is displayed before it goes away. See Setting Timing for Frames.

    The timing for full-motion video (FMV) frames is exactly the same length as the action when you originally captured it. You cannot edit the timing of FMV frames in this version.

  • Set the Timing for Objects An object added to a frame is displayed for a certain period of time. However, you can control the timing of individual objects within frames. You can arrange objects in a frame so that they follow one after the other during the course of the frame, or you can overlap the timing of objects. A useful tool for doing this visually is the Timeline window pane. See Setting Timing for Objects.
