Setting Timing for Objects

how to set the timing for an object in the timeline window pane
  1. Open the frame.
  2. Open the Timeline window pane.

    • Ribbon In the Tool Windows section, select .

    • Menu Select View>Timeline.
    • Standard Toolbar Click .
    • Keyboard Shortcut Press CTRL+T on the keyboard.

    The Timeline window pane opens.

  3. Click at the beginning or end of an object bar and drag it to the right or the left. You can drag the left edge of an object to determine its starting point, and the left edge to set its ending point. In this example, the Timeline window pane shows that there are five objects in the frame. (If you are unable to drag an object bar, right-click on it and select Customize time span. Then try to drag the bar again.)

    • The first object is a bubble shape that starts playing at the beginning of the frame and plays to the end of the frame.
    • The second object is a cursor shape that starts playing at the beginning of the frame and stops after 2 seconds.
    • The third object is a rectangle shape that plays for 2 seconds, starting at 1 seconds and stopping at the end of the frame.
    • The fourth object is a polyline that starts playing at the beginning of the frame and plays to the end of the frame.
    • The final object is another rectangle shape that plays for 1.9 seconds, beginning .6 seconds after the start of the frame and ending .5 seconds before the end of the frame.
how to set the timing for an object in the object properties dialog
  1. Open the frame.
  2. In the Frame Editor, double-click the object whose timing you want to set.
  3. Select the Time Span tab.
  4. Change the settings in any of the following fields.
    • Time span If you want the object to be displayed for a specific period of time, leave the check mark in this box. If you want to disable timing for the object, click this check box to remove the check mark. Disabling the timing is useful if you want to ensure that the object continues to display as long as the frame does.
    • Start Time Enter the number of seconds after the start of the frame that you would like the object to first be displayed. If you want the object to display from the very start of the frame, leave 0 in this field.
    • Length Enter the number of seconds you would like the object to be displayed in the frame.
    • Display message for [ ] seconds Enter the total number of seconds that you would like the object to be displayed in the frame.
    note

    The Time span, Start Time, and Length fields are not displayed for success, failure, and hint captions. On the other hand, the Display message for [] seconds field is displayed only for success, failure, and hint captions.

 note

When you adjust the timing for a frame, the timing for the objects within it stay the same, unless you specify otherwise. You can use a setting on the Mimic tab of the Options dialog (File>Options in the ribbon interface or Tools>Options in the menu interface) to adjust time spans automatically for all frames and objects as you work in Mimic. See Options Dialog.
