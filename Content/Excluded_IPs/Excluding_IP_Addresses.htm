Excluding IP Addresses

You can specify that certain IP addresses should be excluded from Feedback data. This might be a useful feature, for example, if you want to test your online output but do not want your topic views and searches to be included in the data collected by the Feedback server.

how to exclude ip addresses
  2. In the Feedback Admin, select the Excluded IPs tab.

  3. Enter the IP addresses that you want to exclude. Use a separate line for each IP address, pressing Enter after you add an address.

    When entering IP addresses, you can use wildcards (i.e., asterisks) to exclude multiple IP addresses.

    examples

    examples

    172.16.0.100 will exclude that exact IP address.

    172.16.0.* will exclude 172.16.0.[0-255]. The numbers within brackets represent a range. So, for example, 172.16.0.144 will be excluded.

    172.16.*.* will exclude 172.16.[0-255].[0-255] The numbers within brackets represent a range. So, for example, 172.16.92.144 will be excluded.
