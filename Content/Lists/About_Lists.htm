lets you work with numbered and bulleted lists in a variety of ways.

ways to work with lists
  • Simple Lists Read more… You can create simple lists that are numbered or bulleted. See Creating Simple Lists.
  • Multi-Level Lists Read more… You can create numbered and bulleted lists with multiple levels (such as an outline). There are multiple ways to create a multi-level list. See Creating Multi-Level Lists.
  • Custom List Formats Read more… See Creating Custom List Formats.
  • Continue Sequence Read more… Use this to ensure that the next list you create in the topic starts with the next number in the sequence of the list above (even if the two lists are separated by other content). See Continuing the Sequence of Lists.
  • Merge Read more… Use this to combine a list with another list immediately before it. See Merging Lists.
  • Notes or Comments (Paragraph Item) Read more… Use this to include comments between items without interrupting the flow of the list. See Adding Notes or Comments Between List Items.
  • Restart Numbering Read more… Use this to specify a number to start a numbered list, or to start a selected item within a numbered list. See Restarting Numbering in Lists.
  • Reverse Read more… Use this to reorder the items in a list so that they appear in reverse order (i.e., first item is last, last item is first). See Reversing Lists.
  • Sort Read more… Use this option to reorder the items in a list alphabetically. See Sorting Lists.
  • Unbind (Remove) Read more… Use this to remove the list designation from the content so that it displays as regular text. See Unbinding Lists.
  • Edit List Styles Read more… You can modify the look of lists by making adjustments to the tags and style classes used for them. By editing list styles, you can affect things such as the alignment, background, bullet images, and indentation of lists. See Editing List Styles.
quick list drop-down menu

The easiest way to create a numbered or bulleted list is to use 's quick drop-down menu button in the Home ribbon, from which you can select various list formats. The image on the button depends on the most recent action that you have used. You can click the image to quickly apply that type of list format to the selected content. Otherwise, you can click the down arrow and select one of the other list formats available.

 advanced lists with autonumbers

In addition to using the quick list drop-down menu, you can use autonumbering to create more advanced types of lists. Autonumbering is an especially useful method for creating volume, chapter, section, and page numbering for printed outputs. For more information see About Autonumbers.
