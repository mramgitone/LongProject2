Creating Multi-level Lists

You can create numbered and bulleted lists with multiple levels (such as an outline) by using a combination of indentation and list styles.

examples

You can create multi-level lists that are numbered, bulleted, or both.

numbered

  1. Step one.
    1. Substep one.
    2. Substep two.
  2. Step two.
  3. Step three.

bulleted

  • Item one.
    • Subitem one.
    • Subitem two.
  • Item two.

  • Item three.

numbered and bulleted

  1. Item one.
    • Subitem one.
    • Subitem two.
  2. Item two.

  3. Item three.

 how to create a multi-level list
  1. (Optional) If you want to format styles to control the look of your lists, do either or both of the following:
    • Custom List Formats See Creating Custom List Formats.
    • Regular List Styles You can modify the look of lists by making adjustments to the standard CSS list styles and/or creating style classes. By using these list styles, you can affect things such as the alignment, background, bullet images, and indentation of lists. There are a handful of list styles, but the ones you are most likely to use to control lists are the <ol> and <ul> tags (for ordered and unordered lists, respectively). See Editing List Styles and Creating New Style Classes.
  3. Place your cursor in the document where you want to insert a list.

    • Ribbon In the Paragraph section click the list button . If you want to quickly apply the most recent list format, click the face of the button. If you want to apply a different list item type, click the down arrow and select it from the menu.

    • Menu Select Format>List>[Type of List Item].

    • Text Format Toolbar Click the list button . If you want to quickly apply the most recent list format, click the face of the button. If you want to apply a different list item type, click the down arrow and select it from the menu.

  5. Enter text for the first list item. If you selected a custom list format or style, the look associated with that format or style is already applied.
  6. Press Enter to move to the next list item and type content for that item.

  7. If you want to move to a second level of the list, press Tab or click the indent button in the Home ribbon.

    If you are using a custom list format, the style for that level is automatically applied.

    If you are using basic CSS list styles, you can right-click on the ol or ul structure bar to the left, and from the context menu select Style>[Name of Style Class]. See About Structure Bars.

    If you are on one type of list and want to switch to the other kind of list (e.g., you are on a numbered list and want to change to bullets), right-click on the ol or ul structure bar to the left, and from the context menu select Type. Then either click one of the list types or select Custom List Format and choose a style from the submenu.

    If you want to move back to a higher level list (e.g., you are on the third level and want to move back to the second level). click the outdent button twice in the Home ribbon (or you can click it once and then click the yellow arrow icon that is displayed temporarily in the list).

  8. If you are at the end of the entire list and want to exit the list altogether, press Enter again on a blank list item. You should now be outside of the list and on a regular paragraph.
 note

You can also create multi-level lists by using autonumbering. This is a more complex method, but it also has certain advantages. For example, the need to restart numbering manually is not an issue with autonumbering, as it can be with other methods. See About Autonumbers.
