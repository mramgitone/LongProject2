Restarting Numbering in Lists

You can create a numbered list in a topic by using the Numbered List button in the Home ribbon or Text Format toolbar. But what if you need to restart the list at a specific number? The following steps show you how to do this.

how to restart numbering in a list
  2. Click in the numbered list where you want to restart the numbering.

    • Ribbon In the Paragraph section click the List Actions button .

    • Menu Select Format>List>List Actions.

    • Text Format Toolbar Click the List Actions button .

    • If you want to change the numbering for the entire list, restarting the first item at a specific number, click Set List Start Number. Enter a value in the dialog and click OK.

    • If you want to change the numbering for a specific item in a numbered list, click Set Item Number. Enter a value in the dialog and click OK.

      Let's say you have a numbered list from 1 to 10. If you were to click in the paragraph with the number 7 and then use this option to start numbering at 23, your list would change so that the numbering displayed as 1 to 6 and then 23 to 26.
