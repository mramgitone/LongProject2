Inserting Mimic Movie Links in Browse Sequences

If you have used Mimic to produce a movie, you can use this feature to create a link in a browse sequence entry to the movie's output.

how to insert a movie link into a browse sequence
  1. Open the browse sequence where you want to insert the movie link.
  2. In the Browse Sequence Editor, double-click the browse sequence entry that you want to link to the movie. The Properties dialog opens. For help with creating browse sequence entries, see Creating a Browse Sequence.
  3. Select the General tab.
  4. Select Mimic Movie or Movie Collection as. The Open dialog opens.
  5. Find and select an individual Mimic movie or a movie collection to which you want to link the entry. You can select any of the following types of files.
  6. Click Open.

  7. (Optional) You can click in the drop-down field to the right and specify an output format.

  8. Click OK in the Properties dialog. The movie link is added to the browse sequence entry.
