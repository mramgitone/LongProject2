About Importing

You can import files into in these ways:

  • CHM Read more… You can create a new project by importing an HTML Help (CHM) file. See Creating a Project by Importing CHM Files.
  • DITA Read more… You can create a new project or topics by importing DITA files. See Creating a Project by Importing DITA File Content.
  • FrameMaker Read more… You can create a new project or topics by importing Adobe FrameMaker files. See Creating a Project by Importing FrameMaker Documents.
  • HTML Files Read more… You can create a new project or topics by importing HTML files. See Importing HTML Files.
  • HTML Help Projects Read more… You can create a new project by importing an HTML Help project (HHP) file. See Creating a Project by Importing an HTML Help Project (HHP).
  • Source Control Read more… You can create a new project by importing one located in source control. See Importing a Project from Source Control.
  • Word Read more… You can create a new project or topics by importing Microsoft Word files. See Creating a Project by Importing Word Documents.
  • Individual Files Read more… You can import a wide variety of individual files into an existing Flare project. See Importing Individual Files.
  • Global Project Linking Read more…  See About Global Project Linking.For more information about this feature, see the online Help or the Global Project Linking Guide.
