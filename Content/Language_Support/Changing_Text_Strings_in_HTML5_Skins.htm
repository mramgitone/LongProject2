Changing Text Strings in HTML5 Skins

how to change text strings in an html5 skin
  1. Open an skin.
  2. Select the UI Text tab.
  3. From the Language drop-down, select the language for the skin.

  4. In the grid below, type text in the Value field for each relevant row.

    For more information about the items in the grid, see HTML5 Skin Styles.
