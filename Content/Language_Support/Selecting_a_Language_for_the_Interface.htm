Selecting a Language for the Interface

how to select a language for the flare interface

  1. Launch . The Select UI Language dialog opens.

    If you have previously selected to disable this dialog when launching , you can overrule that selection. To do this:

      • Ribbon Select File>Options.

      • Menu Select Tools>Options.

      The Options dialog opens.

    2. Select the General tab.
    3. Click the check box Show Select UI Language Dialog on Startup.
    5. Launch again.
  2. Select the language that you want to move in the list. (The language displayed highest in the list will be the first choice in displaying the interface. If that language is not available for a particular element, the next language in the list is used, and so on.)
  3. Click the Up or Down button until that language is in the proper location in the list.
  4. Repeat steps 2 and 3 until the list displays the languages in the order that you want.
  5. (Optional) If you do not want the dialog to display the next time you launch , click Show this dialog on startup to remove the check mark. If you later decide that you do want to see this dialog when launching , open the Options dialog and click the box to open this dialog on startup.
  6.   is displayed according to your language selections.
 note

The MadCap Help Viewer also supports multiple languages in the interface. Therefore, if you build and distribute DotNet Help, users will be able to select from the available languages to view the interface for your online Help.
