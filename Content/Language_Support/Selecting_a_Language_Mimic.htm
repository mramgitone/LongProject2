Selecting a Language

See About Language Support.

how to select a language for a collection
  1. Open the collection.
  2. In the Collection Editor, select the Language tab.

  3. Select the language from the list. The language must be displayed in bold or have "Yes" displayed in the "Localized Skins" column. Spell check is supported only for languages displayed in bold.

    If the language that you want to use is not displayed in bold and does not have "Yes" displayed in the "Localized Skins" column, you can create a user-defined skin for that language. After that, you can select it from this list. See Creating User-Defined Language Skins.

 how to select a language for a single movie
  1. Open the movie.
  3. Select the Language tab.

  4. Select the language from the list. The language must be displayed in bold or have "Yes" displayed in the "Localized Skins" column. Spell check is supported only for languages displayed in bold.

    If the language that you want to use is not displayed in bold and does not have "Yes" displayed in the "Localized Skins" column, you can create a user-defined skin for that language. After that, you can select it from this list. See Creating User-Defined Language Skins.

If necessary, edit the language skin. You can change the look of the skin, as well as the text on the buttons and labels. See Editing Language Skins.
