Viewing Used Language Tags

how to view used language tags
  1. Open a project.

    • Ribbon  In the Dynamic Reports section, select Used Items>Used Language Tags.

    • Menu Select View>Used Items>Used Language Tags.

    The Used Language Tags window pane opens, listing all of the language tags used throughout the project.

    • Language Displays the abbreviation for the language in the tag. For example, the abbreviation for English is "en"; the abbreviation for German is "de"; and the abbreviation for Italian is "it."
    • Tag Displays the style tag for the content where the language tag is applied. For example, if the language is applied to a topic file, the tag is "html." If the language is applied to selected text within a paragraph, the tag is "span."

    • Title
    • Folder

  5. You can open any file in the list to view or modify it in an editor. You can open the file in the interface or in the project.
