Editing Topic Toolbar Proxies

After inserting a topic toolbar proxy, you can edit it, selecting a style class and/or specifying buttons for it.

basics of using topic toolbars

The generation of topic toolbars is based on two elements—(1) the topic toolbar proxy and (2) settings in the Skin Editor.

  • Topic Toolbar Proxy

  • Skin Editor Settings See Specifying Topic Toolbar Settings in a Skin—WebHelp Outputs and Specifying Topic Toolbar Settings in a Skin—WebHelp Mobile.

    See Standard Skin Styles and WebHelp Mobile Skins.

You can also use the Madcap|topicToolbarProxy style to change the look of the container holding the generated toolbar. See Using Styles to Determine the Look of Topic Toolbars.

note how to edit a topic toolbar proxy

Following are the steps for editing a topic toolbar proxy that you have inserted.

  1. Open a topic or master page where you have inserted a topic toolbar proxy.

  2. Right-click on the proxy and select Edit Topic Toolbar Proxy.The Topic Toolbar Proxy dialog opens.

  3. (Optional) In the Stylesheet class for proxy field, you can select a class to affect the look of the entire toolbar.

    You might create and use a proxy style class, for example, if you want to add a border around the toolbar. If you do not select a class from this field, the generated toolbar will use the style settings from the parent <MadCap|topicToolbarProxy> style. You have the option of creating a class for this proxy style in the Stylesheet Editor. To do this, select the MadCap|topicToolbarProxy style and click Add Class to create a class. The class will then be available from this field.
