About the Page Header Proxy

When you insert one of these proxies, a page number placeholder is added automatically. You can delete it if you do not want to include a page number in that location, or you can realign it (to the left, right or, center). If you want to specify a starting page number or change the format (e.g., alpha, Roman, uppercase, lowercase), you can do so by editing the body proxy and changing the settings in the Printer Page Properties dialog.

For information about inserting page header proxies, see Creating Master Pages, Creating Print Master Pages, and Adding Page Header and Footer Proxies.
