Creating Manifests for Administrators

One way to view Feedback information is to open Feedback Explorer and manually access the many reports in it. However, you can also work with manifests, which are periodic email summaries of various reports. This means that you do not even need to open Feedback Explorer at all; the information automatically comes to you in email format. In order to take advantage of this feature, you must create a manifest in the Feedback Admin, selecting the types of reports you wish it to include. You then create a custom schedule to send the emails at specific intervals (daily, weekly, monthly). You can also use filters to specify the time frame during which the manifest should be run.

  3. In the top portion of the tab, click the appropriate check boxes to select the information to be included in the manifest. For check boxes that let you show the top results for certain information, you can enter a number in the field under it. The default number is 10.
  4. Create at least one scheduled task.

    1. In the Frequency section at the bottom of the tab, click . The New Task dialog opens.

    2. In the Settings area, select how often you want the manifest to be sent.

      If you select Daily, a field displays so that you can specify a certain number of days for the process to recur.

      If you select Weekly, a field displays so that you can specify a certain number of weeks for the process to recur. In addition, check boxes are available so that you can select certain days of the week for the process to run.

      If you select Monthly, additional fields are displayed so that you can select certain months for the process to run, even on specific days during particular months.

    4. (Optional) If you want to add a date filter for that task, click . The Date Filter dialog opens. You can use this dialog to set the following.

      • Enable filter Click this check box to enable the filter, thus enabling the fields below.
      • Range You can click the down arrow in the field and select a starting date when the manifest will be sent.
      • Period of days You can specify how many of the most recent days you would like to include in the manifest. For example, if you enter 20, the manifest will include information from 20 days ago until the current date.

 note

You might want to add a name for your Help system on the General tab of the Feedback Admin. This will be used in emails that you receive to help identify the specific output. See Adding a Help System Name.

In addition to creating a manifest, you must make sure you are added as an administrator and that you have subscribed to the manifest. See Adding Administrators and Editing Administrator Settings.
