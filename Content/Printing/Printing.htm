Printing Charts

You can use one of the following methods to print a chart and its associated data.

how to print charts from the file menu
  1. Create the chart. The Chart window pane opens.
  2. Open a window pane that has a chart.
  3. Select File>Print.
  4. Complete the options in the Print dialog.
 how to print charts from the local toolbar
  1. Create the chart. The Chart window pane opens.
  2. Open a window pane that has a chart.
  3. Use the buttons in the local toolbar of the Chart window pane to specify settings and send the chart to the printer.
    • Print   After you make your selections in the dialog, click OK.
    • Print Preview When you are finished in the dialog, either click to send the chart to the printer or to return to the Chart window pane.
    • Page Setup When you are finished in the dialog, click OK.
