Ribbons

's user interface lets you use a ribbon layout, or the more traditional tool strip layout. The ribbon layout consists of a File drop-down with common tasks such as creating new movies, as well as opening and saving files. This is followed by the Edit ribbon, which is the default, and several others related to various features in .

 characteristics of ribbons

Following are some characteristics of ribbons.

  • Switch Views By default ribbons are used when you install and launch Mimic for the first time. However, you can use the Options dialog to switch back to the classic tool strip.

    See Switching Between Ribbons and the Tool Strip and Options Dialog.

  • Minimize You can minimize the ribbons so that you only see the top portion of the ribbons until you select an option.

    To do this, right-click anywhere in a ribbon and select Minimize the Ribbon from the context menu.

    See Minimizing Ribbons.

  • Icons/Text Ribbons consist of icons and text, which let you perform various tasks. Larger icons are shown for the most common tasks.

  • Sections Ribbons are split into sections to make it easier to locate the options on them. For example, the Frame ribbon contains the following sections: Navigate, Edit, Insert, Reuse, and Properties.

  • Reducing Application Window Size When you reduce the size of the entire application window, the ribbons are also reduced in a way that lets you continue to access hidden options by clicking on the section name.

file drop-down

Click the face of the button to save changes to the active file. You can also click the arrow and choose the following.

  • Save
  • Save As
  • Save All Saves changes to all of the open files.

Opens the Recent Collections dialog, which lets you remove collections from the list of recently opened collections. This allows you to keep the list shorter and more manageable. When you use this feature, the collection is merely removed from the list, but the actual collection is not deleted. See Recent Collections Dialog.

movie ribbon

Click the face of the button to build output files for the active movie using the output type that is selected as the primary target. You can also click the drop-down arrow and select to build the active movie's output files using any Mimic's output types. See Building Output (Interface).

Click the face of the button to view the movie output using the output type that is selected as the primary target. You can also click the drop-down arrow and select to view the movie output using any Mimic's output types. See Viewing Movie Output.

Click the face of the button to publish the output files for the active movie using the output type that is selected as the primary target. You can also click the drop-down arrow and select to publish the active movie's output files using any Mimic's output types. See Publishing Output.

collection ribbon

Click the face of the button to build output files for the active collection using the output type that is selected as the primary target. You can also click the drop-down arrow and select to build the active collection's output files using any Mimic's output types. See Building Output (Interface).

Click the face of the button to view the collection output using the output type that is selected as the primary target. You can also click the drop-down arrow and select to view the collection output using any Mimic's output types. See Viewing Movie Output.

Click the face of the button to publish the output files for the active collection using the output type that is selected as the primary target. You can also click the drop-down arrow and select to publish the active collection's output files using any Mimic's output types. See Publishing Output.
