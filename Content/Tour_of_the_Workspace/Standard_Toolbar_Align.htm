Standard Toolbar

Tools in the Standard toolbar let you start a new alignment project and open the File Map window pane. There are also shortcut buttons for performing basic functions, such as Save, Cut, Copy, Paste, Undo, and Redo. In order to see this toolbar, you can select View>Toolbars>Standard.

 

Description

Opens the Start Alignment Wizard, which you can use to create a new alignment project. See Creating Alignment Projects.

Opens the File Map window pane, which lets you view and align files if necessary. See Aligning Files.
