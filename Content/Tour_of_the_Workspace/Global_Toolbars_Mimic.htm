Global Toolbars

has four global toolbars ("global" meaning they are always available, regardless of where you are working in the interface). These are the Standard toolbar, the Format toolbar, the Frames toolbar, and the Layout toolbar.

standard toolbar

Tools in the Standard toolbar let you perform basic functions, such as Save, Cut, Copy, Paste, Undo, and Redo. To see this toolbar, select View>Toolbars>Standard.

format toolbar

Tools in the Format toolbar let you quickly apply formatting to content in your frames and objects. To see this toolbar, click in the local toolbar of the Frame Editor, or select View>Toolbars>Format.

Opens a menu that lets you select various options for rotating the selected object. See Rotating Objects.

frames toolbar

Tools in the Frames toolbar let you quickly perform actions related to movie frames. To see this toolbar, click in the local toolbar of the Frame Editor, or select View>Toolbars>Frames.

Tool

Description

Opens the Frame Properties dialog, which lets you edit settings for the current frame. See Frame Properties Dialog.

Opens the first frame in the movie.

Opens the previous frame in the movie.

Opens the next frame in the movie.

Opens the last frame in the movie.

Copies the selected frame.

Copies the background image of the selected frame.

Pastes the background image of the frame.

note

Inserts a blank frame after the selected frame.

Opens the Import PowerPoint Wizard, which lets you import slides from a Microsoft PowerPoint presentation. See Inserting PowerPoint Slides into a Movie.

Minimizes and displays a recording rectangle the same size as the frames in the movie. This lets you record new frames, inserting them immediately after the selected frame. See Recording New Frames.

Minimizes and displays a capture rectangle the same size as the frames in the movie. This lets you capture a screen shot on your computer, thereby creating a new frame and inserting it immediately after the selected frame. See Capturing New Frames.
layout toolbar

Tools in the Layout toolbar let you quickly align, sink, or float objects on a frame. To see this toolbar, click in the local toolbar of the Frame Editor, or select View>Toolbars>Layout.

Tool

Description

Sinks the selected object, sending it backward one level.

Floats the selected object, bringing it forward one level.

Sinks the selected object, sending it backward to the bottom level.

Floats the selected object, bringing it forward to the top level.

Aligns the selected objects so that the left border of each is at the same location on the frame. The alignment is based on the left border of the first object that you select. The first object selected has orange handles, while the other selected objects have white handles.

Aligns the selected objects so that the right border of each is at the same location on the frame. The alignment is based on the right border of the first object that you select. The first object selected has orange handles, while the other selected objects have white handles.

Aligns the selected objects so that the bottom border of each is at the same location on the frame. The alignment is based on the bottom border of the first object that you select. The first object selected has orange handles, while the other selected objects have white handles.

Aligns the selected objects so that the top border of each is at the same location on the frame. The alignment is based on the top border of the first object that you select. The first object selected has orange handles, while the other selected objects have white handles.

Resizes the width of the selected objects so that they are all the same. The resizing is based on the width of the first object that you select. The first object selected has orange handles, while the other selected objects have white handles.

Resizes the height of the selected objects so that they are all the same. The resizing is based on the height of the first object that you select. The first object selected has orange handles, while the other selected objects have white handles.

Resizes the width and height of the selected objects so that they are all the same. The resizing is based on the width and height of the first object that you select. The first object selected has orange handles, while the other selected objects have white handles.

Vertically aligns the selected objects so that the center of each is at the same location on the frame. The alignment is based on the center of the first object that you select. The first object selected has orange handles, while the other selected objects have white handles.

Horizontally aligns the selected objects so that the middle of each is at the same location on the frame. The alignment is based on the middle of the first object that you select. The first object selected has orange handles, while the other selected objects have white handles.

Centers the selected object(s) both vertically and horizontally on the canvas.

Centers the selected object(s) horizontally on the canvas.

Centers the selected object(s) vertically on the canvas.
