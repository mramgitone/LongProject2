Menus

The Feedback Explorer's user interface includes a menu bar at the top of the program window, containing several menu options. Of particular importance is the "View" menu, which is used to open the many different window panes that you will use to see and work with user statistics and information.

view menu

Toolbars

Topics

Opens the Topics window pane, which is used to view a list of all the topics that users have opened in your output. This includes a count of how many times each topic has been opened. See Viewing Topics Opened by Users.

Context-Sensitive Help Calls

Opens the Context-Sensitive Help Calls window pane, which is used to view all of the topics that users have opened through context-sensitive Help (CSH) calls that you integrated into your output. This includes each CSH ID, the topic it points to, and the number of times it has been accessed. See Viewing Context-Sensitive Help Topics Opened by Users.

Comments

Opens the Comments window pane, which is used to to accept or discard pending comments in Feedback Explorer. See Accepting Pending Comments and Discarding Comments.

Active Users

Opens the Active Users window pane, which is used to view a list of all of the users who have registered and submitted at least one comment in your Flare output. See Viewing Active Users.

Search Phrases

Opens the Search Phrases window pane, which is used to view all search keywords that users have entered when looking for information in your output. This includes a count of how many times each keyword has been used. See Viewing Used Search Phrases.

Search Phrases with No Results

Opens the Search Phrases with No Results window pane, which is used to view all the search phrases that users have entered when looking for information in your output, but results were not found when the keywords were used. See Viewing Searches Without Results.

Browser Statistics

Opens the Browser Statistics window pane, which displays all the browsers used to view information in your output. See Viewing Browser Statistics.

Operating System Statistics

Opens the Operating Statistics window pane, which displays all the operating systems used to view information in your output. See Viewing Operating System Statistics.

Synonyms

Opens the Synonyms Editor, which is used to create synonyms for phrases in your online output. This way, users will find the results they need when they search with either phrase. See Creating Synonyms to Enhance Search Results.
