When you move window panes, explorers, or editors around in the interface, the configuration (or "layout") of the workspace is changed. You can do several things with layouts, including the following.

When you move window panes, explorers, or editors around in the interface, the configuration (or "layout") of the workspace is changed. You can do several things with layouts, including the following.

how to save a window layout

You can save different layouts of the interface, in case you want to use them for different purposes.

  1. Configure the workspace how you want it.

    • Ribbon In the Window Layout section select Save Layout>Save Layout As.

    • Menu Select Window>Layouts>Save Window Layout As.
  3. In the Rename Window Layout dialog, enter a name for the layout.

how to auto-save a window layout

You can automatically save the layout of the workspace when you exit . The next time you launch , that same layout will be displayed.

  • Ribbon In the Window Layout section select Auto-save Layout.
  • Menu Select Window>Layouts>Auto-save Layout.
 how to select a saved window layout

You can quickly change the configuration of your workspace by selecting a window layout that you have saved previously.

  • Ribbon In the Window Layout section select Layout>[Name of Layout].
  • Menu Select Window>Layouts>Select Layout>[Name of Layout].
 how to reset the window layout

You can return the workspace configuration to the original layout that you saw when you first installed and launched .

  • Ribbon In the Window Layout section select Reset Layout.
  • Menu Select Window>Layouts>Reset Window Layout.
 how to reload the window layout

You can return the workspace to the saved configuration of the layout. In other words, if you are working in a particular layout and have opened different interface elements or moved interface elements around, you can select this option to go back to the saved configuration.

  • Ribbon In the Window Layout section select Reload Layout.
  • Menu Select Window>Layouts>Reload Window Layout.
 how to delete a window layout

You can open the Manage Window Layouts dialog, which lets you select or delete an existing layout.

    • Ribbon In the Window Layout section select Layout>Layouts.
    • Menu Select Window>Layouts>Layouts.

    The Manage Window Layouts dialog opens.

  2. Select the layout that you want to delete.
  3. Click Delete.
  4. Confirm the deletion by clicking OK.
  5. Click Close.
