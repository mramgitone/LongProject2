Log File Viewer

If you save an error report at the end of the process for building a target, you can then open the report to review and address the error messages. When you open an error report, it displays in the Log File Viewer.

how to open this viewer
  2. Double-click the Reports folder.
  3. Double-click the error report that you'd like to open.

 

Description

Description

Displays a description of the log file. For example, if you save an error report after building a target called "Fairy Tales," this field would display "Generate Fairy Tales."

Date

Displays the date and time that the log file was created.

Messages

Displays messages from the generation of the target that was built.

Errors

Displays errors that were encountered during the build process. A description of each error is given. If you scroll to the right in the Log File Viewer, you will see additional information about the error (e.g., the file, line, and character where the error was found).
