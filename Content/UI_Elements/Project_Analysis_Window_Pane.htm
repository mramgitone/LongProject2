Project Analysis Window Pane

This window pane is used to display information collected by the built-in Analyzer. See About Analyzer.

Following are the types of information you can see in this window pane. Click in the drop-down field at the top of the window pane to select a different view.

  • Broken Bookmarks Read more… See Viewing and Fixing Broken Bookmarks.
  • Broken Links Read more… You can See Viewing and Fixing Broken Links.
  • Database Errors Read more… You can See Viewing Database Errors.
  • Files With Changes Read more… See Viewing Files with Changes.
  • Files With Annotations Read more… See Viewing Files with Annotations.
  • Topics Not in Index Read more… You can See Viewing Topics Not Indexed.
  • Topics Not in Selected TOC Read more… You can See Viewing Topics Not in a TOC.
