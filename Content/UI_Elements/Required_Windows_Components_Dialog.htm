Required Windows Components Dialog

This dialog opens when you are installing MadCap Feedback Server Admin Pulse. It lets you know if there are Windows components that you still need to install or allow. If you see an item with red text, it still means you need to click the button next to it in order to complete the action. If the item has green text, it means that the item is fully installed.
