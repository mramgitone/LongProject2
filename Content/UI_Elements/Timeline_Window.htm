Timeline Window Pane

The Timeline window pane gives you a visual method for adjusting the timing for the following.

  • Frames You can change the timing for the entire frame by clicking and dragging the vertical border to the right of all the objects. See Setting Timing for Frames.
  • Objects You can change the timing for an object by clicking and dragging either the left or right side of the object bar and dragging it. If you want to change the timing of an object so that it matches the frame, you can right-click on the object bar and select Same time span as frame. In addition, you can click in the middle of the object bar and drag it to adjust when it starts and ends. See Setting Timing for Objects.
