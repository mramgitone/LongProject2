Hint Caption Properties Dialog

This dialog is used to modify the appearance and behavior of an input hint caption that has been added to a movie frame in conjunction with an input typing box. There are several tabs in the dialog, used for various purposes.

Following are the primary tasks that you can perform in this dialog in regards to hint captions.

  • Shadow Effect Read more… See Adding Shadow Effects.
