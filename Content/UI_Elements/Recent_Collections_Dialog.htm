Recent Collections Dialog

Use this dialog to remove collections from the list of recently opened collections. This allows you to keep the list shorter and more manageable.

  • Ribbon Select File>Manage Recent Collections.

  • Menu Select File>Recent Collections>Manage Recent Collections.

 

Description

Recent Collections

Lists the names of the most recently opened collection files.

Path

Provides the location of the collection.

Open File

Opens the first selected collection in the list.

Remove All

Removes all collections from the list.

Remove

Removes the selected collection(s) from the list. You can select multiple collections in the list by holding down the SHIFT or CTRL keys and clicking a range of collections or individual ones.
note

When you use this feature, the collection is merely removed from the list, but the actual collection is not deleted.
