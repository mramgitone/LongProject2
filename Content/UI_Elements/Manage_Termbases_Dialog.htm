Manage Termbases Dialog

This dialog lets you work with termbases in the following ways.

  • Add Termbases You can add a new termbase using this dialog. See Adding Termbases.
  • Remove Termbases If you no longer need a termbase, you can remove it. See Removing Termbases.
  • Rename Termbases At any point, you can rename a termbase. See Renaming Termbases.
  • Edit Terms See Editing Termbases.
  • Import TBX Files If you have created a termbase in , you can import a TermBase eXchange (TBX) file into it. See Importing TBX Files.
  • Export TBX Files If you have created a termbase in , you can export it to a TermBase eXchange (TBX) file. See Exporting TBX Files.
