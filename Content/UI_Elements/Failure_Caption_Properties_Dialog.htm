Failure Caption Properties Dialog

This dialog is used to modify the appearance and behavior of an input failure caption that has been added to a movie frame in conjunction with an input typing box. There are several tabs in the dialog, used for various purposes. See Adding Input Failure Captions and Adding Input Boxes.
