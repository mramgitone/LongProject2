Index Results Window Pane

This window pane displays links to topics that are related to an index keyword that you click in the Help Index window pane. When you click an index keyword that is related to more than one Help topic, those topics are listed in the Index Results window pane. You can then click any of those Help topics to display it.

  • Ribbon In the Panes section select Index>Index Results.

  • Menu Select Help>Index.
  • Keyboard Shortcut Press ALT+SHIFT+F2 on your keyboard.

 

Description

Title

Displays the title of the topic.

Path

Displays the location where the topic is found. The anchor at the end of the path indicates where the index marker has been inserted in the topic.
