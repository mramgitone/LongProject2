New Updates Available Dialog

This dialog is used to find and download any program updates.

  • Ribbon In the Product section select Check for Updates.

  • Menu Select Help>Check for Updates.

 

Description

Download

Displays the update available (if any).

Description

Displays a description of the update.

Download Update(s)

Downloads the updates from the list above. Follow the instructions on the screen to install the updates.

Close

Closes the dialog.
