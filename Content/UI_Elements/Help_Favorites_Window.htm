Help Favorites Window Pane

If you open a Help topic in 's online Help and plan to refer to it often in the future, you may want to click the "Add topic to Help Favorites" button , which adds the topic to this window pane. Also, if you open the Help Search window pane to perform a search, you can click the "Add search string to Help Favorites" button . This adds the search string to this window pane.

  • Ribbon In the Panes section select Favorites.

  • Menu Select Help>Favorites.

 

Description

Favorite Topics

Displays links to the topics that you have added to your favorites. If you double-click a link, the topic opens to the right.

Favorite Searches

Displays search strings that you have added to your favorites. If you double-click a search string, the search results open in the Help Search window pane.
