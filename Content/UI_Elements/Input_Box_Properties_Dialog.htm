Input Box Properties Dialog

This dialog is used to modify the appearance and behavior of an input box that has been added to a movie frame. There are several tabs in the dialog, used for various purposes.

Following are the primary tasks that you can perform in this dialog in regards to input boxes.

  • Rectangle Properties Read more… See Setting the Rectangle Properties for an Object.
  • Success Action (Add) Read more… See Adding Input Boxes, About Actions, Selecting Actions for Objects, and Adding Input Success Captions.
  • Text Read more… See Adding Text.

  • Transitions Read more… See Setting Transitions for Objects.
