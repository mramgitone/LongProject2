New Pseudo Class Dialog

This dialog is used to create the following pseudo classes to produce certain kinds of effects:

  • after If you want to automatically add content after other content, you can use the after pseudo class. See Pseudo Classes.
  • before If you want to automatically add content in front of other content, you can use the before pseudo class. See Pseudo Classes.
  • first-letter If you want to create a drop caps or initial caps effect in a paragraph, you can use the first-letter pseudo class. See Creating Drop Caps Effects and Creating Initial Caps Effects.
  • first-child If you want to apply style settings to the first child of another element (e.g., the first <p> tag that occurs within a <div> tag) so that it looks different from the other elements, you can use the first-child pseudo class.
