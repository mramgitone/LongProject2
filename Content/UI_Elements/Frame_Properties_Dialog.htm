Frame Properties Dialog

This dialog is used to edit frames that you add to a page layout. See Editing Pages and Frames.

  • Columns Read more… See Creating Columns.
  • Floating and Sinking Read more… See Floating and Sinking Frames.
  • Frame Flow (Type) Read more… See Specifying Frame Flow.

  • Heading and Running Head Variables Read more… See Inserting Heading Variables Into Frames and Inserting Running Head Variables into Frames.
