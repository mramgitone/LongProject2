Page Properties Dialog

This dialog is used to edit individual pages that you add to a layout. See Editing Pages and Frames.

Following are the primary tasks that you can perform in the Page Properties dialog.

  • Background Read more… See Specifying Page Background Settings.
  • Page Types Read more…  See Specifying Page Types.
  • Empty Pages Read more… If you need to force some pages in the output to be empty on purpose, you can use the "Empty" page type. See Using Empty Pages.
  • Bleed Size Read more… The bleed is the area beyond the edge of the page where printing occurs. Professional printers use the bleed to prevent a final trimmed document from having unprinted areas along the edge of the document. You can set the size of the bleed in the Page Properties dialog. See Specifying the Page Bleed Size.
  • Margins Read more…  See Specifying Page Margins.
  • Page Names Read more…  See Specifying Page Names.
  • Size and Orientation Read more… You can select a pre-set size for a page or create a custom size. You can also determine the orientation (portrait or landscape). See Specifying Page Size and Orientation.
