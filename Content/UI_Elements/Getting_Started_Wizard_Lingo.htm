Getting Started Wizard

This wizard is the first window that you see when you launch for the first time. It provides links to the most common high-level tasks and is designed to help new users begin working with the product right away.

how to open this wizard
  • Select File>Getting Started Wizard.

 

Description

Opens the Start New Project Wizard, which guides you through the basic steps for creating a new project. See Creating New Projects.

Opens a dialog that lets you find and select an existing project. See Opening Projects.

Opens the MadCap Align utility, which lets you create alignment projects. See About Alignment.

Recent Projects

Lists projects that you have already created in . They are listed in order of how recently they were opened. Click on a project link to open that project.

If you want to remove links from this list, go to File>Manage Recent Projects (if using ribbons) or File>Recent Projects>Manage Recent Projects (if using menus). See Recent Projects Dialog.

Show this dialog on launch

If there is a check mark in the box, this wizard will open each time you open in the future. If you do not want to see the Getting Started Wizard each time you open , remove the check mark from the box. You can always open the Getting Started Wizard later by selecting File>Getting Started.
