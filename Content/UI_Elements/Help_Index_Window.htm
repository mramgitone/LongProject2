Help Index Window Pane

This window pane displays 's online Help index, which lets you find information about by clicking index keywords. When you click an index keyword, the first related Help topic displays.

What happens if an index keyword is related to multiple topics? To see and open additional Help topics that are related to the index keyword, open the Index Results window pane. Then, when you click an index keyword that is related to more than one Help topic, those topics are listed in the Index Results window pane. You can then click any of those Help topics to display it.

  • Ribbon In the Panes section select Index.

  • Menu Select Help>Index.
  • Keyboard Shortcut Press ALT+F1.

 

Description

Type in the keyword to find

Enter a word or phrase that you want to find in the Help index. As you start typing, the keywords below change to display the nearest terms to your entry.

[Index Keywords]

Displays the index keywords in 's online Help. Each keyword is linked to one or more Help topics. When you click on a keyword, the first related topic opens. If you click on a keyword linked to multiple topics, the Index Results window pane lists links to all of the related topics. Simply click on a link to open that topic.
