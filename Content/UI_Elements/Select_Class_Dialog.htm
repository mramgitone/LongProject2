Select Class Dialog (Tables)

This dialog is used to apply a CSS style to a table. You can create classes of the <table> style in the Stylesheet Editor; those classes will then become available in the Select Class dialog. You might use this dialog when taking either of the following actions. See Stylesheet Editor.

  • Edit Table Properties After you insert a table into a topic, you can easily change the properties of that table. This includes changing the number of columns and cells, the style associated with the table, alignment, borders, and more. See Editing Table Properties.
  • Insert Table A table can easily be inserted into a topic for various purposes, such as comparing elements or giving providing descriptions. See Inserting Tables into Topics.
