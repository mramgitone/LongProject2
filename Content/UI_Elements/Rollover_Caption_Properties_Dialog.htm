Rollover Caption Properties Dialog

This dialog is used to modify the appearance and behavior of a rollover caption that has been added to a frame. A rollover caption is an effect where you designate an area on a frame for the user to hover the mouse over. When the user does this, a caption with text that you provide appears on the frame. When the user moves the mouse away from the designated area, the caption disappears. There are several tabs in the dialog, used for various purposes.
