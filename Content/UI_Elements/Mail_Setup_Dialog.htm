Mail Setup Dialog

This dialog can be used to specify your outgoing email server and your email address. This is used as part of the file review process, letting you email files to other individuals. When you first attempt to send a topic for review, will search for your email server and enter it automatically. However, you can open this dialog whenever you want in order to make changes to the server or your email address. See About Reviews and Contributions and Sending Files for Review.

  1. Open a topic.

    • Ribbon In the Packages section select Send For Review.

    • Menu Select Review>Send for Review.

    The Send Topic for Review dialog opens.

  3. Click Next.
  4. Click Setup Mail.

 

Description

Outgoing Mail Server

Displays your outgoing email server.

Your E-mail Address

Enter your email address, which others will see when you send them topics for review.

Lookup in Active Directory

Searches for your server and enters it into the Outgoing Mail Server field.
note

This works with 32-bit versions of Microsoft Office, not 64 bit.
