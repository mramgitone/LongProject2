Movie Properties Dialog

This dialog is used to modify the appearance and behavior of all frames in a movie. To do the same for just an individual frame in a movie, see Frame Properties Dialog.

Following are the primary tasks that you can perform in this dialog in regards to movies (or all frames in a movie).

  • Audio (Add) See Adding Audio to Movies.
  • Audio (Record) See Recording Audio.
  • Blur See Editing Blur Effects.
  • Border See Adding Borders.
  • Color See Setting the Color for a Movie Background.
  • Condition Tags (Associate) See Associating Condition Tags.
  • Condition Tags (Create) See Creating Condition Tags.
  • End Action See Selecting an End Action for a Movie.
  • Flare Project Links See Linking to Flare Projects.
  • Home Page—for Browser-based Output See Setting the Home Page for Browser-based Output.
  • Language See Selecting a Language.
  • Looping See Looping Movies.
  • Mark of the Web—for Browser-based Output See Adding Mark of the Web.
  • Master Frames (Apply) See Applying Master Frames.
  • Output Files—Single See Selecting Single or Multiple Files for the Output.
  • Output Format See Selecting the Movie Format.
  • Padding See Adding Padding.
  • Resizing See Resizing Movies.
  • Shading Effect See Editing Shading Effects.
  • Skins—for Browser-based Output See Editing Skins.
  • Targets (Create) See Creating Targets.
  • Targets (Edit) See Editing Targets.
  • Transitions See Setting Transitions for Frames.
  • Variables (Create) See Creating Variables.
