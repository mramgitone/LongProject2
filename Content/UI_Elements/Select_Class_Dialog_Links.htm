Select Class Dialog (Links)

This dialog is used to apply a CSS style to a link (such as a text hyperlink). For hyperlinks, you can create classes of the <a> tag in the Stylesheet Editor; those classes will then become available in the Select Class dialog.

  2. Start the process for inserting a link. For example, select Insert>Hyperlink. The Insert Hyperlink dialog opens.
  3. Next the Style Class field, click Select.

 

Description

Class

Displays the style class when you select it from the list of Defined Classes.

Defined Classes

Lists the available link styles in your stylesheet. This includes the <a> tag, as well as any classes that you have created under it (e.g., a.exampletext).
