Text Box Properties Dialog

This dialog is used to set properties for a text box. Following are the primary tasks that you can perform in this dialog. For more information see About Text Boxes, Inserting Text Boxes, and Editing Text Boxes.

  • Background Read more… You See Adding Backgrounds to Text Boxes.
  • Borders Read more… You See Adding Borders to Text Boxes.
  • Hyphenation Read more… You can adjust the way that the text within a text box is hyphenated. See Setting Hyphenation in Text Boxes.
  • Margins Read more… You See Adding Margins to Text Boxes.

  • Padding Read more… You See Adding Padding to Text Boxes.

  • Position Read more… You can adjust the positioning of a text box on a page. This includes the ability to float text boxes to the left or right, or even outside of page frames. See Positioning Text Boxes.
  • Size Read more… You can adjust the width and height of a text box. See Specifying the Size of Text Boxes.
