Image Viewer

This viewer is used to see images or preview movies (SWF files) in your project.

how to open this viewer
  1. Navigate to the image or movie file (usually located in the Resources\Images subfolder).
  2. Double-click the image or movie file that you want to open.

    • Ribbon In the Dynamic Reports section select More Reports>Unused Items.

    • Menu Select View>More Windows>Unused Items.

    The Unused Items window pane opens.

  2. Click the drop-down field at the top of the Unused Items window pane and select Unused Images.
  3. Double-click an unused image in the list.

 

Description

Opens the image in MadCap Capture (if you have it installed) so you can edit it.

Regenerates (updates) the image. This is useful for removing watermarks and modifying PROPS files. See Regenerating Images.
