Comments Window Pane

You can view all comments in the Comments window pane. If you want to see only comments that have a particular status (e.g., Accepted, Pending, Hidden), you can use the filter field in the local toolbar of the window pane.

Following are some of the primary tasks that you can perform in this editor.

  • Accept pending comments Read more… See Accepting Pending Comments.
  • Hide comments Read more… See Hiding Comments.
  • Discard comments Read more… See Discarding Comments.
