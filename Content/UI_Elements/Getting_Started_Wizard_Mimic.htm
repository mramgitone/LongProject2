Getting Started Wizard

This wizard is the first window that you see when you launch for the first time. It is designed to help you become familiar with quickly and begin using the product right away.

how to open this wizard
  • Select File>Getting Started Wizard in either the ribbon or menu interface.

 

Description

minimizes and a rectangle with a red border appears on your screen, along with a task bar, so that you can record a new movie. See Recording Movies.

Opens the Start New Collection Wizard, which guides you through the basic steps for creating a new movie collection. When you finish providing information in the wizard, the new collection is loaded into . See Creating a Movie Collection.

Lets you open an existing movie or collection. See Opening a Collection.

Recent Files

Lists recent movie collections and standalone movie files. You can click on a link to quickly open that collection or movie. The maximum number of recent files saved in the registry is 30.

Show this dialog on launch

If there is a check mark in the box, this wizard will open each time you open in the future. If you do not want to see the Getting Started Wizard each time you open , remove the check mark from the box.
