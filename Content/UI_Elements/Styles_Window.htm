Styles Window Pane

This window pane displays style classes contained in the stylesheets that are associated with the current topic. See About Styles and Stylesheets.

  • Ribbon In the Styles section select Style Window.

  • Menu Select View>Style Window.
  • Keyboard Shortcut Press F12.
 what you can do in this window pane

Following are the primary tasks that you can perform in the Styles window pane.

  • Apply You can apply a style to the selected content in the topic. To do this, click the style in the Styles window pane to apply it to the content. See Applying Styles to Topic Content.

    This is supported in the following workflows: Review (Full Registered), Contribution, and New Document. It is not supported if you are working in the Free Review (Annotation Only) mode.

  • Create You can create a new style class, which you can then apply to content in the topic. To do this, click the Create Style button. See Creating New Style Classes.

    This feature is designed for the following workflows: Contribution and New Document.

  • Edit You can modify the settings for the style currently applied to the selected content. To do this, click the Edit button. See Editing Styles.See Editing Styles.

    This feature is designed for the following workflows: Contribution and New Document.

 

Description

[Style]

Displays the style applied to the selected content in the XML Editor.

Edit Style

Opens the Stylesheet Editor, with the current style already selected, allowing you to quickly edit it. See Editing Styles.

Create Style

Opens the Create Style dialog, which lets you create a style based on the local properties already applied to the selected content. See Creating New Style Classes.

[Local Values]

This area displays any local formatting values applied to the selected content in the XML Editor. This is an indication that those local values will override values set on the style.

Available Styles

Displays the styles that can be applied to the selected content in the XML Editor. To apply a style, simply click on it once.
