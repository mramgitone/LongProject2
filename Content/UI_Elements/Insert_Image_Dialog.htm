Insert Image Dialog

This dialog opens when you attempt to insert an image in an element such as a page layout or a stylesheet property, or a background element (e.g., in a paragraph). You can use the built-in tree view to find and select an image file in your project. Otherwise, you can click to select an image file outside of your project; that image file will then be copied into your project. If you want to link to an image stored on a website, rather than adding it to your project, you can enter the path directly into the field. You can click to find and select an image file.
