Help Search Window Pane

This window pane provides a way for you to quickly search for information in 's online Help. Simply enter one or more keywords in the field at the top and click the Search button. Links to topics containing the keywords are then listed below. You can click on a link to open that topic. The keywords are highlighted in the topic.

In addition to searching for regular terms, you can perform wildcard searches. This is possible if you are using Microsoft Windows XP (or a newer operating system) and have SQL Compact installed (see Installing SQL Server).

example

example

If you type Mad* in the Search field, will list all topics that contain words starting with "Mad" (e.g., MadCap, Madagascar, madness, Madrid).

 note

Wildcard searches are supported in output only.

If you want to find a specific combination of words that are always next to each other in the same order (e.g., Options dialog), you can enter the search keywords within quotation marks (e.g., "Options dialog").

  • Ribbon In the Panes section select Search.

  • Menu Select Help>Search.
  • Keyboard Shortcut Press CTRL+F1.

 

Description

[Search Keywords]

Enter one or more keywords in the field at the top. When you initiate the search, looks for these keywords in the Help topics.

Search

Initiates the search. Links to topics containing the keywords are then listed below.

Adds the search keywords to your Help favorites.

Filter

You can limit the topics to include in your search by selecting a subject from this drop-down list.

Title

Displays the title of the topic in which the search keywords were found.

Rank

Displays the rank of the topic in the search. The topics containing the most occurrences of the keywords are ranked higher.

File

Displays the topic file.
