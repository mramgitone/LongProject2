Copy to Project Dialog

This dialog displays the files to be copied to your project when you add certain types of files, such as those from SharePoint or external resources.

 

Description

The following files will be copied to the project

Displays the files to be copied. If the file being added is dependent on other files to operate correctly (e.g., if it is linked to another topic file), those necessary files are also listed in the dialog (and therefore copied as well).

Keep file(s) synchronized (create mapping)

This check box displays if you are copying an external resource or SharePoint file to your project. You can select this check box if you want to immediately create a connection (mapping) between the copy of the file in your project and the original file in the external mapped location.
