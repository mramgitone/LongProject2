Waiting for Database Refresh Dialog

This dialog opens if you are attempting to perform an action in the interface but the current project is still being scanned by for issues. Typically, larger projects take more time to refresh than smaller ones.

Although you have the ability to click the Close button and continue working, it is recommended that you allow the refresh process to finish. This will ensure that the database is up-to-date. Once complete, this dialog closes and you can continue working. After finishes updating the database, you can press F5 to refresh any pane and show the latest results.
