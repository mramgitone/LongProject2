The Start Page

how to open the start page

The Start Page automatically opens when you first launch . If you close the Start Page, you can reopen it.

  • Ribbon In the Common section, select Start Page.
  • Menu Select View>Start Page.
  • Standard Toolbar Click .

Sections of the Start Page

Following are explanations of the different sections in the Start Page.

Getting Started

This contains links to open an existing project (see Opening Projects) or create a new one (see Creating New Projects). You can also launch MadCap Align (see About Alignment).

 recent projects

This lists projects that you have already created. They are listed in order of how recently they were opened. Click on a project link to open that project.

If you want to remove links from this list, you can do so from the Recent Projects dialog. See Recent Projects Dialog.

 Help resources

This tab contains links to additional resources, such as the Help system, PDFs, webinars, video tutorials, the online community, and the MadCap blog.

 News and Updates

This section is at the bottom of the Start Page. It contains images and links to events, MadCap services, and other information.
