Manage Movie Sound Library Dialog

This dialog lists any audio files (such as MP3 and WAV files) that you have imported into the movie or recorded. You can use this dialog to do the following.

  • Add Audio to Frames See Adding Audio to Frames.
  • Add Audio to Movies See Adding Audio to Movies.
  • Add Audio to Objects See Adding Audio to Objects.
  • Rename Audio Files See Renaming audio Files.
