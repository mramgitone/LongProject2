MadCap Movie Viewer

The MadCap Movie Viewer is used to do either of the following.

  • Preview You can preview a movie before it is finished. See Previewing Movies.
  • View You can view a completed movie that was built with the MadCap Mimic Movie Format (MMF). See Viewing Movie Output.

Refer to the online Help provided with the MadCap Movie Viewer for more information.

The MadCap Movie Viewer opens when performing the following tasks.

if you are previewing a movie

  • Ribbon In the Quick Preview section, select any of the preview options.

  • Menu Select Movie>Quick-preview Movie>[preview option].
  • Local Toolbar In the local toolbar of the Frame Editor, click the face of the preview button to preview the entire movie or click the down arrow and from the drop-down menu, select a preview option.
 if you are viewing a standalone movie

  • Ribbon In the Movie Output section, select View.
  • Menu Select Movie>View.
  • Standard Toolbar In the Standard toolbar, click .
  • Keyboard Shortcut Press SHIFT+F6 on your keyboard.
 if you are viewing a collection

  • Ribbon In the Collection Output section, select View.
  • Menu Select Collection>View.
  • Standard Toolbar In the Standard toolbar, click .
