Frame Properties Dialog

This dialog is used to modify the appearance and behavior of an individual frame in a movie. To do the same for all frames in a movie, see Movie Properties Dialog.

Following are the primary tasks that you can perform in this dialog in regards to individual frames.

  • Audio (Add) Read more… See Adding Audio to Frames.
  • Audio (Record) Read more… See Recording Audio.
  • Blur Read more… After you add a blur effect to a frame, you can edit the effect. See Editing Blur Effects.
  • Border Read more… See Adding Borders.
  • Color for Background Read more… See Setting the Color for a Frame Background.
  • Condition Tags Read more… See Applying Condition Tags.
  • Images Read more… See Setting Images for Frames.
  • Master Frames (Apply) Read more… See Applying Master Frames.
  • Movie Properties Dialog (Open) Read more… You can open the Movie Properties dialog. See Movie Properties Dialog.
  • Shading Effect Read more… After you add a shading effect to a frame, you can edit the effect in a couple of ways. See Editing Shading Effects.
  • Timing Read more… See Setting Timing for Frames.
  • Transitions Read more… See Setting Transitions for Frames.
note

If you see an orange circular icon on a frame image in the Frames window pane, that means the frame is using full-motion video (FMV), which is commonly used for dragging and similar actions. If you open the properties for a frame that is using FMV, there are some things that cannot be done in that frame, such as modifying the timing of the frame.
