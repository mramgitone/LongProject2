Language Skin Editor

This editor is used to edit style settings (including text) for existing language skins. Normally, you would edit style settings for a skin in the Skin Editor. However if you want to display the output user interface in a particular language, you can use this editor instead. (The rest of the interface settings, such as which buttons to include and where to place the navigation toolbar, should still be specified in the Skin Editor.) When you generate , first looks at the style settings in the Skin Editor. If it does not find manually entered style settings in that editor, it uses those from the Language Skin Editor. If it does not find settings in either editor, it uses the default skin settings.

With the Language Skin Editor, you can modify the look of the skin and the text shown on buttons and labels. Language skins are available only with browser-based output. See About Language Support and Editing Language Skins.

Add Class

Opens the New Style dialog, which lets you create a new style class for the Toolbar Item node. After you create the new toolbar item, you can set the styles to apply an action to it or to change its look and feel. You can also apply an action to a custom toolbar item by adding custom JavaScript.

Styles

Displays the style items that you can modify for browser-based output. This includes the Flash Player (the "container" for displaying the movies), toolbars (you can have a navigation toolbar above or below the movies), and toolbar items (e.g., buttons for moving back or forward, pausing, displaying a logo). When you select an item, the relevant properties are shown to the right. If you select one of the style headings (Toolbar, Toolbar Item), as opposed to an item below it, the property values that you set will be applied to all the items below that heading.

Property Groups

Select this option to display the style properties in the appropriate groups (e.g., font properties under "Fonts," border properties under "Borders").

Alphabetical List

Select this option to display the style properties in alphabetical order.

Properties

Lists the properties relevant to the style item that is selected to the left. If the properties are displayed in property groups, expand the group to see the actual properties. The property name is displayed in the left column. The right column is used to specify the value settings for the property. Depending on the property, you might set the value by typing an entry, selecting an option from a drop-down list, or specifying settings in a small popup window.

Font/Background Color

Sets the background color of the font for the style item. Click the down arrow to open a color palette window. You can click a color shown, or you can click More colors and select a different color or create a custom color.

Font/Color

Sets the color of the font for the style item. Click the down arrow to open a color palette window. You can click a color shown, or you can click More colors and select a different color or create a custom color.

Font/Font Family

Sets the font family (e.g., Arial, Tahoma) for the style item. Click the down arrow to open a small window. In this window, select (click to add family). A drop-down list appears, from which you can select a font family. You can select additional font families to be used in case your users do not have the first font family selected. If the first font family is not found on the user's computer, it uses the second font family. If the second font family is not found, it uses the third font family, and so on. When you want to delete a font family that you have added, select it in this window and press Delete on your keyboard. When you are finished, click OK in the small window to accept the settings.

Font/Font Size

Sets the size of the font for the style item. Click the down arrow to open a small window. In this window, you can enter the size of the font in the lower-left field (either type a number or use the up and down arrows). You can select the unit of measurement (e.g., pixels, points, centimeters) using the lower-right drop-down arrow (see About Units of Measurement). You can return to the default setting by using the upper-right drop-down arrow.

Font/Font Style

Sets the style of the font for the style item. Click the down arrow and select either italic or normal.

Font/Font Weight

Sets the weight of the font for the style item. Click the down arrow and select either bold or normal.
