Topic View Window Pane

This window pane is used to view all comments and ratings submitted for a topic. See Viewing Topic Comments and Viewing Topic Ratings.

In addition, you can use the Comments tab in the window pane to change the status for a comment.

 local toolbar

Use the options in the local toolbar for various purposes, such as customizing the display of columns on the tabs, deleting comments, and setting the status for comments.

Description

Opens the live topic in a browser window. In order to use this button, you must first enter the URL of the documentation published on a web server. This can be done on the General tab of the Feedback Admin window. See Viewing Live Topics in a Browser.

Opens the Options dialog, which lets you specify the way that the columns are displayed in the window pane. This includes the ability to select certain columns to be included in charts that you create. See Customizing Column Information.

Lets you change the type of chart shown. To do this, click the chart type filter field in the local toolbar and make a selection. Initially this field displays the type of chart currently used (e.g., "Point" or "Bar"), but the label changes with each selection you make.

Lets you limit the number of rows shown in the chart. To do this, click the filter field in the local toolbar and make a selection. Initially this field displays the words "All Comments," but the label changes with each selection you make.

comments tab

Use this tab to view all comments that have been submitted for a topic. If you double-click a row, the Edit Comment dialog opens. This allows you to modify the comment's subject or body text, as well as change the status. See Editing Comments.

Description

User

Displays the username of the person submitting the comment.

Subject

Displays the comment subject entered by the user.

Comment

Displays the actual comment text entered by the user.

Date

Displays the date and time that the user submitted the comment.

Status

Displays the status of the comment (i.e., Accepted, Hidden, Pending Review).

CommentID

Displays the ID number automatically assigned to the comment.

ParentID

Displays the ID number automatically assigned to the parent. This is shown if the comment is a reply to another comment.

#

Displays the numerical order in which the comment was received.
rating tab

Use this tab to view all ratings that have been submitted for a topic.

Description

Average Rating

Rating

Displays the different rating scores, based on the number of stars that users click (5 stars=100, 4 stars=80, 3 stars=60, 2 stars=40, 1 star=20). For example the Topic View window pane might indicate that 80% is the overall average score of the topic, with users giving the topic 5 stars twice, 4 stars once, and 3 stars twice.

Rating Count

Displays the number of times the topic has received a particular score.
