Sound Editor

This editor is used to edit audio files that you have recorded. You can use this editor to accomplish many tasks, including the following.

  • Copy Audio Sections See Copying Sections of an Audio File.
  • Delete Audio Sections See Deleting Sections of an Audio File.
  • Move Audio Sections See Moving Sections of an Audio File.
  • Record New Audio See Recording New Sound for an Audio File.
  • Silent Areas (Insert) See Inserting Silent Areas into an Audio File.
  • Volume (Adjust) See Adjusting the Volume of an Audio File.
  1. Open one of the following, depending on the element for which you have recorded audio.
    • The properties dialog for an object (double-click the object)
    • The Frame Properties dialog (double-click the frame)
    • The Movie Properties dialog (select Movie>Properties in either the ribbon or menu interface)
  2. Select the Audio tab. The recorded audio WAV file should be displayed in the Audio field.
  3. Click Edit.

 

Description

File

Use this menu to select OK (to accept changes in the editor), Cancel (to close the editor without saving changes), or Save To (to save the WAV file in a specific location).

Edit

Use this menu to select basic editing options (such as undo, redo, cut, copy, paste, and delete).

View

Use this menu to open and close certain elements in the editor (such as toolbars, messages, and the status bar).

OK

Click this button to accept your changes in the editor.

Cancel

Click this button to close the editor without saving changes in the editor.

Copies the selection and sends it to the clipboard.

Cuts the selection and sends it to the clipboard.

Pastes the selection from the clipboard to the place where the cursor is inserted.

Click this button to play the audio file.

Click this button to pause the audio file.

Click this button to stop the audio file.

Click this button to open the Insert Silence dialog, which lets you add a silent portion into a particular area of the audio file. You can insert the silent portion for a specific duration based on the following.

  • An area that you have selected (just click and drag in the Sound Editor to select an area)
  • The playhead (the red vertical line that displays when you click in the Sound Editor)
  • The start of the audio file
  • The end of the audio file

Click this button to open the Record dialog, which lets you record additional audio for the sound file. The new audio is added at the beginning of the sound file, or directly after the playhead (the red vertical line that displays when you click in the Sound Editor).

Click this button to open the Adjust Volume dialog, which lets you modify the volume of the audio file.

Click this button to zoom in the display of the audio file. Each time you click this button the display is enlarged by 100%.

Click this button to zoom out the display of the audio file. Each time you click this button the display is reduced by 100%. This button is disabled if the audio file is displayed at 100%.

Displays the current zoom percentage of the audio file display.

Displays the position of the playhead in the audio file. The playhead is the red vertical line that displays when you click in the Sound Editor. For example, if "2 sec" is displayed, the playhead is located after the 2-second mark in the audio file.

Displays the time span of the selected area in the audio wave display.

Displays the total time span of the audio file.
