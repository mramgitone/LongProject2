Movie Window

This window plays the frames in the open movie. It contains a local toolbar that lets you navigate in the movie or movie collection.

  • Open a generated movie (.mcmovie) or movie collection (.mcmoviesys) file.

 

Description
Opens the previous movie (if a movie collection is playing).

Opens the next movie (if a movie collection is playing).
Lets you open a specific movie (if a movie collection is playing).
Restarts the movie from the beginning.
Goes back to the previous frame in the movie.
Goes forward to the next frame in the movie.
Pauses the movie.
Displays the current frame number in the movie.
Displays the number of seconds that the current movie frame has played.
