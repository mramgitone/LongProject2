Select Bookmark Dialog

This dialog is used to select a specific location in a topic (a bookmark or heading) when creating a link. It is used when inserting a link such as a hyperlink or cross-reference. It is also used in the Properties dialogs for tables of contents (TOCs) and browse sequences when linking an entry to a topic.

 

Description

Headings

Lists all of the headings in the topic (paragraphs associated with an <h1> through <h6> tag). You can select any of the headings as the destination for the link.

Bookmarks

Lists all of the bookmarks in the topic. You can select any of the bookmarks as the destination for the link. For more information see Inserting Bookmarks into Topics.
