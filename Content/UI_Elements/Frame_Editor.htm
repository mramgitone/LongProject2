Frame Editor

Each movie that you create using can consist of multiple frames. When a movie is open in , the frames of the movie are listed in the Frames window pane. From there, you can select a specific frame of a movie. That frame is then displayed in this editor, and you can enhance it by using the many features available in . If you have more than one movie open in , you can have more than one instance of the Frame Editor open as well—each editor showing a single frame from that particular movie.

  • Either (1) select File>Open (2) press CTRL+O on your keyboard or (3) in the Getting Started Wizard click Open. In the Open dialog, navigate to the MIMOV file and select it. Then click Open.

    OR

  • Select File>Recent Files>[Name of MIMOV file].

    OR

  • Open a collection. From the Collection Editor, click the Sections tab, select a movie, and click Open.

    OR

  • Open a collection. From the Movies window pane, double-click a movie.
