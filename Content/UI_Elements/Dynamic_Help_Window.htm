Dynamic Help Window Pane

This window pane automatically displays the Help topic related to the active element in the interface. If you want to "lock" the window pane so that it continues to show the current Help topic while you move around in the interface, click in the local toolbar of the window pane.

  • Ribbon Select the Help ribbon. In the Topics section, select Dynamic Help.

  • Menu Select Help>Dynamic Help.
  • Keyboard Shortcut Press CTRL+F3 on your keyboard.

 

Description

Keeps the current Help topic frozen in place while you move around in the interface. This allows you to follow steps or other information contained in the Help topic without the window pane changing to a different topic when you click elsewhere.

If you are distributing output using the DotNet Help format, your users can also take advantage of this feature if:

  • You distribute the latest version of the DotNet Help Viewer.

    AND

  • Your developer(s) integrate Dynamic Help into the output. See CSH Calls for DotNet Help.
