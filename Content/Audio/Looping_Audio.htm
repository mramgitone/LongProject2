Looping Audio

Use the following steps if you want the audio file to restart when it reaches the end of the playing time. This is useful, for example, if you want to play background music for the entire duration of a movie.

You can loop audio for an entire movie, a single frame, and some objects (such as cursors and buttons).

how to loop audio for a movie
  1. Open the movie.
  3. Select the Audio tab.
  4. Click Enable Audio.
  5. Use the options on the tab to add audio. See Adding Audio to Movies.
  6. Click the check box next to Loop Sound.
 how to loop audio for a frame
  1. Open the frame.
  2. Double-click the frame. The Frame Properties dialog opens.
  3. Select the Audio tab.
  4. Click Enable Audio.
  5. Use the options on the tab to add audio. See Adding Audio to Frames.
  6. Click the check box next to Loop Sound.
 how to loop audio for an object
  1. Open the frame.
  2. Double-click the object (cursor or button). The properties dialog for that object opens.
  3. Select the Audio tab.
  4. Click Enable Audio.
  5. Use the options on the tab to add audio. See Adding Audio to Objects.
  6. Click the check box next to Loop Sound.
