Deleting Sections of an Audio File

After you record an audio file, you can edit the file. One way to do this is to delete a section of the audio file.

how to delete a section of an audio file
  1. Open the movie.
  2. Open one of the following, depending on the element for which you have recorded audio.
    • The properties dialog for an object (double-click the object)
    • The Frame Properties dialog (double-click the frame)
    • The Movie Properties dialog (select Movie>Properties in either the ribbon or menu interface)
  3. Select the Audio tab. The recorded audio file should be displayed in the Audio field.
  4. Click Edit. The Sound Editor opens.
  5. Click in the audio file display and drag the mouse to select the area you want to delete. When you select an area of the audio wave, it becomes dark gray.
  7. In the Sound Editor, click OK.
  8. In the properties dialog, click OK.
