Editing Audio Files

After you record an audio file, you can edit the file in several different ways, such as adjusting the volume, deleting sections, and inserting silent areas.

how to edit an audio file
  1. Open the movie.
  2. Open one of the following, depending on the element for which you have recorded audio.
    • The Movie Properties dialog (select Movie>Properties in either the ribbon or menu interface)
    • The Frame Properties dialog (double-click the frame)
    • The properties dialog for an object (double-click the object)
  3. Select the Audio tab. The recorded audio file should be displayed in the Audio field.
  4. Click Edit. The Sound Editor opens.
  5. Use the options in the Sound Editor to do any of the following.adjust the volume of the audio file

    2. Set the options in the Adjust Volume dialog. For field descriptions, see the online Help.

    1. Click and drag in the audio wave to select the area.
    2. Press Delete.
