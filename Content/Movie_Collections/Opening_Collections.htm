Opening a Collection

There are multiple ways to open a movie collection that you have created.

how to open a collection using the open dialog

    • Ribbon Select File>Open.

    • Menu Select File>Open.
    • Standard Toolbar Click .
    • Keyboard Shortcut Press CTRL+O on the keyboard.
    • Getting Started Wizard Select Open.

  2. In the Open dialog, navigate to the MIPRJ file and select it. Unless you selected a different location when creating the collection, a collection file is stored by default in the Documents\My Movies folder.

  3. Click Open. The collection is loaded in the Collection Editor.
 how to open a recent collection using the file menu
  • Select File>Recent Files>[Name of MIPRJ file].
 how to open a collection using the start page

On the Start Page, the area labeled Open an Existing File lists the most recently opened collections (and movies).

  1. Do one of the depending on the part of the user interface you are using.

    • Ribbon In the Tool Windows section, select Start Page.

    • Menu Select View>Start Page.
    • Standard toolbar Click .
  2. Click the file you want to open. The collection is loaded in the Collection Editor.

 how to open a collection by dragging a file from windows
  1. Open Windows and navigate to a folder containing a collection file.
  2. Launch .

  3. Drag the collection file from Windows to the application window and drop it on the title bar in .

    You can also use this method to open any file type that is supported in .
