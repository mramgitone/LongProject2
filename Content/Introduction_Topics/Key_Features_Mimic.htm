Key Features

offers dozens of features to assist you in producing movies. Following are some of the key features. See the online Help for more information about each feature.

General

general

Following are some general key features in .

  • Integration with Flare Read more… is a great tool for many kinds of users (e.g., technical support, sales, marketing), but it is especially perfect for Help authors and was designed with those individuals in mind. Therefore, is tightly integrated with MadCap Flare. This means that it is very easy to start the movie-making process from Flare and to quickly incorporate the finished movie into your output. See Distributing Output and the Flare online Help.

  • Integration with Lingo Read more… One of the easiest ways to translate a collection or movie is for a translator to open that collection or movie within MadCap Lingo, which is tightly integrated with . Because of this integration, there is no need to transfer localized files outside of the actual collection or movie, which helps prevent content and formatting corruption. In addition, translators can leverage all previous translations created in other tools by importing Translation Memory eXchange (TMX) files. For more information, please refer to the documentation provided with MadCap Lingo. See About Language Support.

  • Integration with MadCap MovieSync Read more… is also integrated with MadCap MovieSync—a unique application used for quickly transferring movie and frame properties from one Mimic movie to another. These properties can include objects (e.g., shapes, cursors, typing boxes), timing, sizes, and more. MovieSync is especially useful when you are recording a new version of a movie where most of the frames are the same as those in the older version of the movie. It is designed to replace the manual work involved in copying or re-creating all of the various properties in each frame so that the new movie looks and behaves the same way as the older movie. For more information, please refer to the documentation provided with MadCap MovieSync.
  • XML File Format Read more… movie files are XML-based. This means that they can be opened and modified with third-party XML tools. See MadCap Mimic Files.
  • Unicode Support for All Left-Right Languages Read more… is fully Unicode capable, making it possible to handle the entire world’s Unicode language characters. not only supports Western European languages, but also double-byte Asian languages, Eastern European languages, and more. See About Language Support.

Creation

creation

Following are some of the unique movie creation features available in .

  • Immediate Preview Read more… With some movie-making software, you must wait for the application to process your movie when you want to preview it (or part of it). In , the preview is immediate, whether you are previewing 1 frame, 10 frames, or 100 frames. See Previewing Movies.
  • PowerPoint Import Read more… In addition to creating a movie by recording your actions, you can also create a movie by importing a Microsoft PowerPoint presentation. See Creating Movies from PowerPoint Presentations.
  • Collection-oriented Workflow Read more… In addition to creating standalone movies, works with the concept of collections. Each collection can contain several individual movies. This allows you to easily and quickly link smaller movies together to form larger ones. See Creating a Movie Collection.
  • Recording Modes Read more… When you record a movie, you can select whether to capture images automatically, manually, or with full-motion video (FMV). See Selecting the Recording Mode.
  • Snap to It Read more… Forget about having to manually resize your recording window or the application so that they are identical. In , you can automatically "snap" the recording window to an active application's size, or you can move an active application within the recording window. See Recording Movies.

Enhancement

enhancement

Following are some of the unique movie enhancement features available in .

  • Animated Typing Boxes Read more… Not only can you add boxes for users to type input during a movie, but you can also add animated typing boxes automatically or manually to movie frames. Animated typing boxes allow users to watch the typing occur before their eyes. See Adding Animated Typing Boxes.
  • Individual Frame Images Read more… Each background frame image in a movie is stored as an individual image file. This means that you can easily edit a particular background image with any image editor.
  • Input Typing Boxes Read more… If you want users to enter input into a particular movie frame, you can add an input typing box. You can also provide success and failure actions, as well as success, failure, and hint captions for the input box. See Adding Input Boxes.
  • Integrated Audio Read more… lets you easily incorporate background sound or record narration for your movie. See About Audio.
  • Special Effects Read more… You can quickly apply several different kinds of special effects to frames and objects in your movies. These include effects that let you highlight an area by applying shading, blur, or gray scale to everything around it. Another feature lets you create a 3D effect that zooms in and emphasizes a specific area in an image. See About Effects.
  • Trajectories Read more… A trajectory is the path of movement of an object (usually a cursor) on a movie frame. With , trajectories can be added automatically when you record movies, but you can also add them manually when you later edit the movie. See About Trajectories.
  • Visual Timeline Read more… Use 's visual timeline to synchronize objects in a movie frame. You can specify how long each object should be displayed in a frame, including the specific moment when it should be shown. See About Timing and Timeline Window Pane.

Single-Sourcing

single-sourcing

has numerous single-sourcing features that save you time and effort by letting you create content once and reuse it in multiple places. This includes the following features.

  • Condition Tags and Targets Read more… See About Conditions and About Targets.
  • Frame Libraries Read more… See About Frame Libraries.
  • Master Frames Read more… See About Master Frames.
  • Palettes Read more… After you create a movie, you can add various types of objects (such as shapes, lines, and cursors) to the individual frames. A palette is an element that lets you store objects for later use. See About Palettes.
  • Variables Read more… In addition to adding regular text to frame objects, you can also insert variables (re-usable text). Not only can you create variables in , but you can also link to variables that already exist in a MadCap Flare project. See About Variables.
