 images—auto-size objects with text

In Mimic, a check box has been added which allows all captions in a Movie to be autosized to fit text. When checked, this option overrides the autosize option on the shape level. When unchecked, the autosize option on the shape level becomes active.

The following might be old:

For Capture/Mimic, I’ve included an autosize check box which will size the rectangle-based shape (rectangle, callouts, etc) to fit the text inside. With this option checked the user will not be able to manually size the shape, but may adjust the shape placement. This is to allow users who do automation coupled with translation, so that upon compilation of their projects in various languages, the dimensions of the shape will adjust to the caption.
