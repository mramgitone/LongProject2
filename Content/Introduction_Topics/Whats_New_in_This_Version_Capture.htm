Cropping

Various enhancements have been made to the cropping feature in Capture.Also, can resize crop selectionxxx

  • Adjust Crop Area Previously, when you started to crop an image, the area was cropped as soon as you released the mouse button. Starting with this version, the drawn area is not immediately cropped. Instead, you need to click an Accept Crop button first. With the addition of this option, you have the opportunity to adjust the area before cropping it.

  • Blank Images You can now crop blank images (i.e., those that do not yet have a background).

  • Image Objects If you have inserted an image object into a picture, you can crop it, as well as the image itself.

  • Mouse Wheel Zoom When you start cropping an image, you can hold the CTRL key and move the mouse wheel (or scroll wheel) to zoom in and out. Previously, the mouse wheel zoom was not supported during the cropping activity.

  • Objects Shown when Cropping If you have objects on an image (e.g., shapes, image objects) and initiate the crop feature, those objects can still be seen when you draw the crop area. Previously, those objects temporarily disappeared during the cropping action.

  • Permanent CropsThere is a new option that allows you to make crops permanent in an image. This option removes the cropping history on an image.

 Objects

A few enhancements have been made to objects in Capture.

Auto-sizing to Fit Text

If you have images that contain objects with text, you can auto-size those objects to ensure none of the text is cut off. This can be done by selecting an option in the Properties dialog. With this option selected, you will not be able to manually size the object, but you can adjust its placement.

You might use this auto-size option if you plan to have your image object text translated into other languages. The translated language might require more space in order to show all of the words in the object. The auto-size option ensures that all of the words are shown.

how to auto-size objects to fit text
  1. In the image, double-click on the object holding text.
  2. In the Properties dialog, click the Rectangle tab.
  3. Select Auto-size to fit text.
 example note

If you insert Capture images into a MadCap Flare project, you also have the option of using image object auto-sizing via the Target Editor.

In previous versions you had the option to make all similar objects (e.g., all rectangles) on an image take on the same properties as the one you select. That feature is still available, but a new option now also lets you make all objects on an image look the same, even if they are not similar.

Therefore, you start by clicking that rectangle. Then in the Object ribbon, you click the down arrow on the Make Objects… button , and you select Make All Objects the Same.

 Properties—Copy to All

Capture has a feature that allows you to make all objects of the same type look the same (i.e., take on the same properties). For example, you might have three bubble shapes with a green background and one with a yellow background. You can use this feature to quickly change all of the green bubbles so that they have yellow backgrounds.

Starting with this version, this feature has been enhanced so that you can do the same thing with all objects, not just those of the same type.

Example

Let's say you have an image with a bubble, two rectangles, and an oval. Each one has a different background color, border, and font type.

You decide you want the bubble and rectangles look like the oval. Therefore, you right-click the oval and select Make all look the same>Make All Objects The Same.

 Rotation by Dragging

Previously, you could rotate objects clockwise or counter clockwise—1 degree, 45 degrees, or 90 degrees at a time. Starting with this version, you can select an option—in the Object ribbon or from the right-click context menu—that lets you click on the corner of an object and drag it, thus rotating it manually. You can also initiate the rotate mode by holding the CTRL key, then clicking and dragging the object.

 How To Use the Color Fill Tool
  1. In the local toolbar of the Capture Editor, click . If you click the face of the button, you can apply the most recently selected color. Otherwise, you can click the down arrow and select another color.

  2. Click on an area of the image background (not an object) that has the same color. That area will be filled with the color you selected.

  3. To disengage from the color fill, select another tool in the local toolbar or choose the Select Mode button .
 Eraser

A new tool lets you erase parts of an image background. When you do this, the areas erased become transparent.

How To Use the Eraser Tool

  1. In the local toolbar of the Capture Editor, click . If you click the down arrow, you can select a width for the eraser. If you click the face of the button, you can use the most recently selected width.

  2. Click on the image background (not on an object) to erase the area. If you use a single click, just one spot is erased. If you click and drag, a larger area is erased, as if you were using an eraser on a sheet of paper.

  3. To disengage from the eraser, select another tool in the local toolbar or choose the Select Mode button .

 Pencil

A new tool lets you draw on the image background as if you were using a pencil. You can select several pencil widths and colors when using this tool.

How To Use the Pencil Tool

  1. In the local toolbar of the Capture Editor, click . If you click the down arrow, you can select both a color and a width for the eraser. If you click the face of the button, you can use the most recently selected color and width.

  2. Click on the image background (not on an object) to draw with the pencil tool. If you use a single click, just one spot is drawn. If you click and drag, a larger area is drawn, as if you were using a pencil on a sheet of paper.

 Dictionaries—Import

For spell checking, you can now see which dictionaries are installed on your computer by opening the Options dialog and selecting the Dictionaries tab. See Options Dialog.

You can download more dictionaries from OpenOffice.org. After you download a new dictionary, you can install it by clicking Import Dictionaries. See Importing Dictionaries.

 Drag Sections of Images

You can use a selection tool to choose a rectangular area of the image. You can then click and drag that area to another place on the image. The moved selection becomes an image object. If your selection includes an object (or part of one), that object becomes flattened as a result.
