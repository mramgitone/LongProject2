Introduction

Welcome to the MadCap Feedback Server Admin, which lets you set up an SQL database for use with MadCap Feedback. Authors use Feedback to track live user feedback on their MadCap Flare online documentation.

The following will help you get started quickly with Feedback Server Admin.

  • Installation See detailed steps on installing Feedback Server Admin and other necessary software. See Installing MadCap Feedback Server.
  • Configuration and Maintenance After Feedback Server Admin is installed, it must be configured and maintained so that it works properly with Feedback and Flare. See Configuring the MadCap Feedback Server Admin.
