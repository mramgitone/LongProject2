Viewing Files with Changes

how to view files with changes

    • Ribbon In the Analysis section select Project Analysis>Files With Changes.

    • Menu Select View>Project Analysis>Files With Changes.

    The Project Analysis window pane opens, with the "Files With Changes" option selected from the drop-down menu at the top. The area at the bottom of the window pane displays all topics in your project where tracked changes have been made.

  2. Open a project.

    • Ribbon In the Dynamic Reports section select More Reports>Files With Changes.
    • Menu Select View>More Windows>Files With Changes.

    The Files With Changes window pane opens, displaying all topics in your project where changes have been inserted.

    • File
    • Title
    • Folder
    • Change Displays the type of change that was made.
    • Initials Displays the initials of the individual who made the change.
    • Creator Displays the user name of the individual who made the change.
    • Timestamp Displays the date and time when the change was made.

  6. You can double-click any file in the list to accept or reject the changes (see Accepting or Rejecting Tracked Changes).
  7. If you double-click a file in the list, you will see a message stating that the file has unresolved changes in it. Click Yes if you want to open the file in to accept or reject the changes.
