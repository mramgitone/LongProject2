Creating Palettes

provides you with an initial palette to help get you started. However, if you want to create additional palettes for different purposes, you can do so easily.

how to create a palette

    • Ribbon In the Tools section click .

    • Ribbon In the Tool Windows section click .

    • Menu Select View>Palettes.
    • Standard Toolbar Click .

    The Palettes window pane opens.

  2. In the local toolbar of the Palettes window pane, click . The New Palette dialog opens.
  3. Enter a name for the new palette.
  4. The new empty palette opens (by default on the left side of the user interface).

After you create a palette, you can add objects to it. See Adding Objects to Palettes.
