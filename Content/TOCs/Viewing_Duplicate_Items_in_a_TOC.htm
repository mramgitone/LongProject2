Viewing Duplicate Items in a TOC

how to view duplicate items in the toc
  1. Open a project.

    • Ribbon In the Dynamic Reports section, select More Reports>Duplicate TOC Items.

    • Menu Select View>More Windows>Duplicate TOC Items.

    The Duplicate TOC Items window pane opens, displaying all files that have been added more than once to a TOC.

    • File
    • Title
    • Folder
    • TOC Displays the name of the TOC where the multiple links to the file are found.
    • Count Displays the count.

  3. You can open any file in the list to view or modify it in an editor. You can open the file in the interface or in the Flare project.
