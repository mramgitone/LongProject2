Inserting Heading Variables Into Frames

  See About Heading Variables, About Running Head Variables, and Inserting Running Head Variables into Frames. See About Heading Variables.

Following are steps for inserting a Heading variable into a page layout frame.

how to insert a heading variable into a page layout frame
  1. Click in the Frame Contents window pane.

    • Ribbon In the Single-Source section select Variable.

    • Menu Select Insert>Variable.
    • Local Toolbar Click .
    • Keyboard Shortcut On your keyboard press CTRL+SHIFT+V.

    The Variables dialog opens, with the variable set(s) on the left and the variables associated with the selected set on the right.

  3. Select the Heading variable set.

  4. Select the variable you want to add. Much of the information that follows pertains to projects and are primarily for the authors of those projects.

  5. The variable is added to the Frame Contents window pane.
  6. The variable can now be seen in the frame.
