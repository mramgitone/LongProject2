Using the Administration Menu

Administration Menu

In the dashboard, the Administration menu is available to members of the Administrators group. With the commands in this menu, you have the ability to modify the web server settings or to run reports.

In the dashboard, the Administration menu is available to members of the Administrators group. With the commands in this menu, you have the ability to modify the web server settings or to run reports.

settings

The options in the Administration>Settings menu include:

  • Database View the database settings. See Database.
  • General View and configure the general settings for the web server. See General Settings.
  • Groups Manage the default user groups and create new groups. See Managing Groups.
  • Suggested Feeds Configure suggested feeds for your user groups. See Suggested Feeds.
  • Users Add new users and perform user administration tasks. See Managing Users.
  • Profiles Configure the elements and fields that are available for user profiles. See Configuring Profile Settings.
  • Moderation Configure the moderation feature in Pulse. You can add reasons for new posts being rejected by moderators. See Configuring Moderation—Rejection Reasons.
  • Log View server events, errors, processes and tasks. See Log.
  • Communities View a list of communities that integrate with your systems. See Communities.
  • Synonyms Manage a list of synonyms to enhance the search capabilities of a community. See Managing Synonyms.
 reports

The functions in the Administration>Reports menu give you access to a variety of usage reports for both the system and your communities. See Reports.
