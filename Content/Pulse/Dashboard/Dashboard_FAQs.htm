what version number do i have?
  1. Log in to the dashboard as an administrator.
  2. Select Administration>Settings.The version number appears in the top right-corner of the page.

Managing Accounts

managing accounts

If you are a member of the Administrators group and responsible for user administration, here are some answers to frequently asked questions.

who can disable or delete user accounts?

user accounts can be disabled or deleted by these individuals:

  • Pulse Administrators Members of the Administrators group can disable to delete user accounts from the Administration>Settings menu. Members of the Employees and Customers groups cannot delete user accounts, including their own account.
  • SQL Server Administrator If you have administrative permissions to SQL Server, you can remove user information directly from the SQL database. This will immediately remove a user from .

For more information about users, see Users.

 how can i reuse an email address?

If you delete a user account in the dashboard, access to features is immediately revoked. However, to give a user the ability to reuse an email address in , you must also delete the user from the dbo.Users and dbo.Login tables in SQL Server. If you are using SQL Server Management Studio you can right-click the user and select Delete from the context menu. Once the entries are deleted, the email can be used to create a new account.
