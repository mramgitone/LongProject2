Posting Content

Users can post content in a variety of ways, such as status updates, questions, articles, and tasks.

Users can post content in a variety of ways, such as status updates, questions, articles, and tasks.

  • Status Updates All registered users can post status updates in the feeds associated with a system. See Posting Status Updates.
  • Questions All registered users can post questions in a feed and vote on answers. See Posting Questions.
  • Articles All registered users can create and post articles in the topic feeds associated with a system. See Posting Articles.
  • Tasks All registered users can create, assign, and manage tasks with . See Assigning Tasks.
