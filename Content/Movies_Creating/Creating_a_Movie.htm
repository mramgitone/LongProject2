  • Record a Movie Simulate (record) your actions in a specified application or area, creating a movie with multiple frames. See Recording Movies.The easiest way to do this is to click in the Standard toolbar. See Recording Movies.
  • Create a Movie from a Single Image Create a movie from an existing image. After creating the movie, you can enhance the frame and insert additional frames if you want. See Creating Movies from Single Images.The easiest way to do this is to xxx, select Movie From Image File, and choose an image file on your computer.
  • Create a Movie from a Screen Capture Capture an image on your screen, creating a movie with a single frame of that image. After creating the movie, you can enhance the frame and insert additional frames if you want. See Creating Movies from Screen Captures.The easiest way to do this is to xxx, select Movie From Screenshot, position the capture area, and click the red button.
  • Create a Blank Movie Create a movie that consists of a single blank frame. You can enhance the frame and insert additional frames if you want. See Creating Blank Movies.The easiest way to do this is to xxx, select Blank Movie, and complete the fields in the Add New Movie dialog.
  • Import a Movie If you have created a collection, you can import an existing movie into it. See Importing Movies.The easiest way to do this is to open the Collection Editor, click Import Mimic Movie in the local toolbar, and complete the fields in the dialog that opens.
  • Import a PowerPoint Presentation In addition to creating a movie by recording your actions, you can also create a movie by importing a Microsoft PowerPoint presentation. See Creating Movies from PowerPoint Presentations.The easiest way to do this is to xxx, select Import PowerPoint, and complete the fields in the Import PowerPoint Wizard.
