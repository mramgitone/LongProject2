Importing Movies

In addition to creating a movie by recording your actions or capturing an image, you can also import an existing movie into a collection.

how to import a movie
  1. Open the collection that you have created.

    • Ribbon  In the File section, select Import Movie.

    • Menu Select Collection>Import Mimic Movie.
    • Local Toolbar Click .

    3. The Import Movie dialog opens.

  3. At the end of the Source File field, click .
  4. In the Open dialog, locate and double-click the movie (MIMOV file) that you want to import.
  5. (Optional) If you do not want to use the name of the movie to be imported, you can enter a new name in the File Name field.
  6. Click Import. The movie is loaded into the interface. It is also added to the Collection Editor.

After you import the movie, you can enhance it in several ways (e.g., inserting additional frames, adding effects, adding objects). When you are finished enhancing the movie, you can build it. See Enhancing a Movie and Building Movie Output.
