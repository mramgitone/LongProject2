Viewing Browser Statistics

You can use Feedback Explorer to see which browsers are being used to view your online output.

how to view browser statistics

  2. Select View>Browser Statistics.The Browser Statistics window pane opens.

    You can see statistics on all browsers used, including the number of views for each. You can also see that browser's overall percentage of views compared with other browsers.

    Browsers are grouped by their name with the major/minor version numbers.

  7. (Optional) You can change the type of information shown in the chart. To do this, click in the Chart field in the local toolbar and select either View Count or View Percentage. The text in the chart changes accordingly. In most types of charts this information is displayed on the left side of the chart.

notes

Although Feedback Explorer specifies the different versions of Internet Explorer in the Browser Statistics window pane (e.g., IE 8.0, IE 7.0), if the output is being viewed in Internet Explorer 8.0 Compatibility Mode, this will display in the window pane as IE 7.0.

You can also reset browser statistics if necessary. See Resetting Browser Statistics.
