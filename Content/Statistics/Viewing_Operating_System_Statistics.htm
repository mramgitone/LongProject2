Viewing Operating System Statistics

You can use Feedback Explorer to see which operating systems are being used to view your online output.

how to view operating system statistics

  2. Select View>Operating System Statistics. The Operating System Statistics window pane opens.

    You can see statistics on all operating systems used, including the number of views for each. You can also see that operating system's overall percentage of views compared with other operating systems.

  7. (Optional) You can change the type of information shown in the chart. To do this, click in the Chart field in the local toolbar and select either View Count or View Percentage. The text in the chart changes accordingly. In most types of charts this information is displayed on the left side of the chart.

notes

You can also reset operating system statistics if necessary. See Resetting Operating System Statistics.
