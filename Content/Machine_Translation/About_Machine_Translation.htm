Example

Let's say you are creating a new project and do not have an existing translation memory. In the Start New Projects wizard, you can select a machine translation provider to pre-translate your project and get a head start on your translations. See Pre-Translate Files.

 Machine Translate Entire Projects

Machine translation allows you to quickly translate a project using translation suggestions from a translation memory provider. Once you have selected a machine translation provider, you can translate an entire project using machine translation. See Translating Projects with Machine Translation.

 Machine Translation Suggestions

If you do not want to translate the entire project using machine translation or pre-translation, you can choose to run machine translation in the background so you will see suggestions from the machine translation provider alongside suggestions from your translation memory in the TM Suggestions window pane. You then can select any single suggestion for a target segment by selecting the segment and double-clicking a suggestion from the MT.

 Future Suggestions Based on Accepted Translations

If you are using a translation memory in your project (as recommended), and you accept a translation suggestion from the machine translation provider, adds the segment to your translation memory as a 100% match. Because some machine translation providers charge a fee based on the number of characters translated, this can save you money in the long run, since you will not need to retranslate the file using the machine translation. If you need to retranslate the file, or if you translate a file with similar segments, your translation memory will provide translation suggestions based on your accepted translations.

 Concordance Searching

If you are using machine translation, you can use the concordance to search any of the suggestions your machine translation provider has returned. After you have enabled the MT for concordance searches, you can enter any word in the concordance search window pane, and if it appears in a machine translated suggestion, the translation will appear in the window pane. This allows you to check for consistency in your project even if you do not have a translation memory installed. See About the Concordance.
