About Translation Memory eXchange

The purpose of Translation Memory eXchange (TMX) is to allow easier transmission of translation memory data between tools and/or translation vendors with little or no loss of critical data during the process.

In , you can import TMX files from tools such as Trados and CATALYST. This allows you to leverage all prior translations, whether or not they occurred in the same project or document. By importing a TMX file into , all of your past translation work will become available as suggestions when you are localizing content in . Then, when you have finished translating content in , you can export the TMX files so that they can be used again in the other tools.

See Importing TMX Files and Exporting TMX Files.
