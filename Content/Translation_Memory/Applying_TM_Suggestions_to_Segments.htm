Applying TM Suggestions to Segments

You can automatically apply matching translations from a TM database to segments in a project. You can automatically apply 100% or 101% matches from the TM database. In other words, the Target cell in the Translation Editor will be populated with the translation. This does not mean the translation is completed. You still need to confirm each one. See About Translation Memory.

You can apply TM suggestions at the project, file, or segment level.

how to apply tm suggestions to an entire project—during process of creating project
  1. Begin the process of creating a project. See Creating New Projects.

  2. On the second page of the wizard, select the TM and click Pre-translate.

  3. Finish the steps for creating the project.
 how to apply tm suggestions to an entire project—after the project is created
  1. Open the project.

    • Ribbon In the Translation Memory section select Translate project with TM.

    • Menu Select Translate>Translate project with TM.

    • Translate Toolbar Click .

    A message lets you know if segments were applied, and how many.

 how to apply tm suggestions to a file
  1. Open the file in the Translation Editor.
  2. In the local toolbar click . A message lets you know if segments were applied, and how many.
 how to apply tm suggestions to a segment
  • In the Translation Editor, click in the empty target cell. The matching suggestion is applied immediately to that cell.
 notes

To confirm a segment, click in that Target cell and press Enter.
