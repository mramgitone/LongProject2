Choosing a Translation Memory Database

See About Translation Memory.

how to choose a tm database
  1. Make sure you have created at least one TM database. See Creating a Translation Memory Database.

    • Ribbon In the Translation Memory section select Choose TM.

    • Menu Select Translate>Choose Translation Memory.

    • Translate Toolbar Click .

    The Current Translation Memory Provider dialog opens.

  3. Next to each TM in the grid that you want to associate with the project, click Use.

    You can also use the dialog to do any of the following:

    add/remove tm databases to/from grid create new tm databases edit tm database

    You can open any TM database in the grid to edit it in the Translation Memory Editor. To do this, select the row in the grid and click Edit TM. See Editing the Translation Memory Database.

         delete tm database

    You can delete a TM database by selecting the row and clicking Delete. This not only removes the TM from the grid, but it deletes it entirely.
