Spell Checking Manually

This is a feature that lets you perform spell checking for the current topic, all open topics, all topics in the same folder as the current one, or all topics in the project. As questionable words are identified, you can select an alternative spelling, ignore the spelling warning, and add words to the dictionary so that future occurrences are not flagged as possible misspellings.

    • current document Read more… checks the spelling in the active topic only.
    • whole project Read more… checks the spelling in every topic in the project. 
    • all open documents Read more… checks the spelling in all topics currently open in the interface.

  4. Click the Find Next or Start button to initiate the search.
  5. After finds and highlights the next questionable word, a list of suggested spellings is displayed at the bottom of the Spell Check window pane. If you want to change the spelling of the term, follow Step 6. If you do not want to change the spelling of the word, follow Step 7.
  6. You can type your own corrected word in the Suggestions field or you can select a word from the list (which is then automatically added to the Suggestions field). Then click one of the following buttons.
    • Change Read more… changes only that occurrence of the misspelled word.
    • Change in All Files Read more… changes that occurrence of the misspelled word, as well as any other occurrences of it in the rest of the project.
  7. If you want to keep the word as it is, click one of the following buttons.
    • Skip Read more… ignores the spelling of the word (this occurrence only) and moves on to the next misspelled word. If other occurrences of the word are found in the topic, they will be flagged as questionable spellings.
    • Add to Dictionary Read more… adds the word to the dictionary so that it is not flagged as a questionable spelling in the future. then moves on to the next misspelled word.
    • Ignore Read more… ignores the spelling of the word and moves on to the next misspelled word. also adds the spelling of this word to the Ignored Words dialog (for this topic only). This means that if other occurrences of the word are found in the topic, they will not be flagged as questionable spellings.

    • Ignore in All Files Read more… ignores the spelling of the word and moves on to the next misspelled word. also adds the spelling of this word to the Ignored Words dialog (for all topics included in the search). This means that if other occurrences of the word are found in any other topics included in the search, they will not be flagged as questionable spellings.

      note

      You can also specify words to be ignored by using the ignore spell check list. See Adding Words to the Ignore Spell Check List.
