About Spell Check

You can use the following spell checking features.

  • Spell Checking While Typing Read more… This feature lets you keep tabs on your spelling as you type in your topics. Questionable words are underlined with a jagged red line. By right-clicking on a word, you can select an alternative spelling, ignore the spelling warning, and add words to the dictionary so that future occurrences are not flagged as possible misspellings. See Spell Checking While Typing.
  • Spell Checking Manually Read more… This feature lets you open the Spell Check window pane and perform spell checking for the current topic, or all open topics, all topics in the same folder as the current one, or all topics in the project. As questionable words are identified, you can select an alternative spelling, ignore the spelling warning, and add words to the dictionary so that future occurrences are not flagged as possible misspellings. See Spell Checking Manually.
  • Adding Words to the "Ignore" Spell Check List Read more… This feature lets you manually add specific words to a list so that they are not flagged as having a questionable spelling in the topic again. However, additional occurrences of the word in other topics will still be flagged as having a questionable spelling if you use spell check in those topics. See Adding Words to the Ignore Spell Check List.
