Adding Words to the Ignore Spell Check List

You can open the Ignored Words dialog and manually add specific words to a list so that they are not flagged as having a questionable spelling in the content file again. However, additional occurrences of the word in other files will still be flagged as having a questionable spelling if you use spell check in those files.

how to add words to the "ignore" spell check list
  1. Open the content file.
  2. Open an image.
  3. Select a segment.
  4. Place your cursor in the text edit area at the bottom of the editor.

    • Ribbon In the Spell Check section select Ignored Words.

    • Menu Select Tools>Ignored Words.

    The Ignored Words dialog opens.

  6. Click in the list field and type the word that you want to ignore in the file. You can enter one word per line.
 note

Alternatively, you can use the Spell Check window pane and select the "Ignore" button when a term is flagged. As a result, that term is automatically added to the Ignored Words dialog. If you click the "Ignore in All Files" button, the term is added to the Ignored Words list for all files where that term is found. See Spell Checking Manually.
