Opening Dictionaries

You can add words to a custom dictionary when you spell check images and click the "Add to Dictionary" button for highlighted words. When a word is added to the custom dictionary, it is not flagged as a questionable spelling in the future; Capture simply moves on to the next misspelled word.

A separate dictionary is created for each language used. If you want to make changes to a custom dictionary, you can open it using the steps below.
