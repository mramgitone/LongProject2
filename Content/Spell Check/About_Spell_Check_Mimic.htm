About Spell Check

You can use spell checking features for the text in your movies. These features include the following.

  • Spell Check Movies Read more… This feature lets you open the Spell Check window pane and perform spell checking for the current movie, an entire collection, all open files, and all files in the same folder. As questionable words are identified, you can select an alternative spelling, ignore the spelling warning, and add words to the dictionary so that future occurrences are not flagged as possible misspellings. See Spell Checking Movies.
  • Add Words to the "Ignore" Spell Check List Read more… This feature lets you manually add specific words to a list so that they are not flagged as having a questionable spelling in the movie again. However, additional occurrences of the word in other movies will still be flagged as having a questionable spelling if you use spell check in those movies. See Adding Words to the Ignore Spell Check List.
  • Editing Custom Dictionaries Read more… This feature lets you modify the words that you have added to a custom dictionary. You can add words to a custom dictionary when you spell check movies and click the "Add to Dictionary" button for highlighted words. A separate dictionary is created for each language used in your collection. See Editing Dictionaries.
