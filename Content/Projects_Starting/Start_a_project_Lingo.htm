Starting a Project

The first step in translating a project after you launch is to start a project. You can create a new project or open an existing one. In addition, you can update previous projects.

  • Create New Projects You can create new projects that you are translating for the first time. See Creating New Projects.
  • Open Projects You can open projects that you have already created. See Opening Projects.
  • Update Projects You can update a Lingo project in order to translate new changes from files that you have previously localized. See Updating Projects.
