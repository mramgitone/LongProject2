Deleting Projects

To delete a project completely, you must use Windows Explorer. This operation cannot be done from .

You can also remove projects from displaying as recent projects in . This does not delete the project, but simply removes it from those shortcut locations.

how to delete a project through windows
  1. In Windows Explorer, browse to the location where you have stored the project (e.g., C:\Documents and settings\Projects\My Project).
  2. Delete the folder holding the project files.
 how to remove a project from recent projects in flare

    • Ribbon Select File>Manage Recent Projects.

    • Menu Select File>Recent Projects>Manage Recent Projects.

    The Recent Projects dialog opens.

  2. Highlight the project and click Remove.
