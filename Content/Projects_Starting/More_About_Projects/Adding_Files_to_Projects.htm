Adding Files to Projects

You can now add additional files to a project after the project has been created. You can add additional files to any project created from other individual files, such as individual Word documents or XML files.

How to Add Project Files

  1. In the local toolbar of the File List window pane or Project Editor, click .

    The Add Project Files Wizard opens.

  2. Click Add.
  3. In the dialog that opens, select the file(s) you want to add to the project. Click Open.
  4. If you need to remove a file, select it in the wizard and click Remove.
  5. (Optional) If you are adding Word, XML, or text files to your project, you can create a file type filter to import only certain parts of the file. See About File Filters.
  6. When you are done adding files, click Finish. The files are added to the project.
