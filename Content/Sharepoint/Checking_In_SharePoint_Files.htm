Checking In SharePoint Files

When you are finished working on a SharePoint file that you checked out, you can check it back in so that others can edit it.

how to check in sharepoint files

    • Ribbon In the Resources section select SharePoint Explorer.

    • Menu Select View>SharePoint Explorer.

    The SharePoint Explorer opens. See SharePoint Explorer.

  1. Locate the file(s) that you want to check in. You can use the "multi-view" to do this.

  2. Select the file(s) that you want to check in. If the window pane is split into two halves, you can select the appropriate folder on the left side and select the files on the right side.
  3. Right-click the file(s) and from the context menu select Check In.

  4. Click Check In.

    By default there is a check mark next to all files that were selected. If necessary, you can click any check box to remove the check mark, which means that file will not be checked in when you are finished.
