Synchronizing SharePoint Files

After you map project files to source files located on a SharePoint server, you can synchronize them to ensure that each file contains the same content. This process allows you to import content from SharePoint files, export content from mapped files in the project to SharePoint, or keep the most recently modified content.

how to synchronize sharepoint files note

In the list of files, you can click to see more details, such as the modified date and file size. With the modified date, keep in mind that if multiple users are exporting changes to the same source external file, it's possible that some users could end up with a dialog indicating they have the most recent version of a file, when in fact they do not. Good communication between all individuals involved is the best way to avoid such situations.
