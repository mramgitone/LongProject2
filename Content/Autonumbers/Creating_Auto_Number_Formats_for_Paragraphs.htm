Creating Autonumber Formats for Paragraphs

Autonumbers for Paragraphs

When you incorporate autonumbering into content, you do so by creating an autonumbering format, which consists of one or more commands. Some examples of commands are: CH:, {n+}, {chapnum}, {b}, and {/b}. In addition, you can add text next to commands.

example

The following steps show you how to create an autonumber format for a single paragraph. You also have the option of creating an autonumber format for a style. It is recommended that you use the style method whenever possible. A style allows you to apply the same format to multiple paragraphs throughout your project, and any changes to the format are applied automatically to all the paragraphs using that style.

For more information and several samples, see About Autonumbers and Autonumber Format Examples.
