Editing Colors for Charts

If you have previously created custom colors for charts (see Editing Colors for Charts), you can edit them by changing or deleting the colors.

how to change a color in a chart
  1. Open a window pane that has a chart.
  2. In the local toolbar click . The Chart Options dialog opens.
  3. Click the down arrow in the Color column for the row you want to change.
  4. Click Pick Color.
  5. Use the Color Picker dialog to choose a custom color. See Color Picker Dialog.
  6. When you are finished click OK. The color is changed in the Chart Options dialog.
 how to delete a color in a chart
  1. Open a window pane that has a chart.
  2. In the local toolbar click . The Chart Options dialog opens.
  3. Click on the row with the color you want to delete.
  4. In the local toolbar click .
  5. In the message that displays click Yes.
  6. In the Chart Options dialog click OK.
