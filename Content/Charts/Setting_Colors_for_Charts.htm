Setting Colors for Charts

You can set custom colors for charts, allowing you to more easily view and analyze the information in them. Using the Chart Options dialog, you can add multiple rows, assigning different colors to each. When a chart is displayed, the first data item uses the color of the first row, the second item uses the color of the second row, and so on. If you have more items in the chart than you have custom colors, the default color will be used on later items after the custom colors have all been designated.

how to set colors for a chart
  1. Open a window pane that has a chart.
  2. In the local toolbar click . The Chart Options dialog opens.
  3. Click .
  4. Use the Color Picker dialog to choose a custom color. See Color Picker Dialog.

  5. When you are finished click OK. A new row with that color is added to the Chart Options dialog.
