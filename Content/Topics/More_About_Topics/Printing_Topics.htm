Printing Topics

You can print a topic show in the interface, whether it is a topic you are writing for your own project or a topic you are viewing in the online Help.

how to print a topic
  1. Open the topic.

    • Ribbon or Menu Select File>Print.

    • Keyboard Shortcut Press CTRL+P.
  3. In the Print dialog, complete the options and select OK.
 note

At this time, it is possible to print only one topic at a time.
