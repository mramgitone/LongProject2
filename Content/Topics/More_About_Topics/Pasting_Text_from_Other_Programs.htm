Pasting Text from Other Programs

If you want to copy text from another program—such as Notepad, Microsoft Word, or a page in an Internet browser—you can paste it into , and by clicking the paste icon next to the copied content, you can specify how it should be brought into .

how to paste text from other programs
  1. After copying the text in the other program, open a in .
  2. Place your cursor where you want to paste the text.

    • Ribbon  In the Clipboardsection select Paste.

    • Menu Select Edit>Paste.
    • Standard Toolbar Click .
    • Right-Click Right-click and from the context menu select Paste.
    • Keyboard Shortcut Press CTRL+V.

    The content is pasted using the default behavior, and the paste icon appears after it. The drop-down arrow is visible when you hover over the icon.

  4. (Optional) If you want to change the way the content is added to the file, click the drop-down arrow in the paste icon and select the option that best suits your needs.

    • Paste Paragraphs Read more… Select this option if you want to paste the text as regular paragraphs. The pasted text will take on the style of the location where its being pasted. New paragraphs will become default paragraphs.
    • Paste Paragraph Block Read more… Select this option if you want to paste the text as paragraph blocks, such as <div>, <blockquote>, or <fieldset>. For more information about these types of styles, see Creating Tag Groups.
    • Paste Inline Text Read more… Select this option if you want to paste the text as inline text, so that it takes on the style of the location where it is being pasted. If you have copied multiple lines of text from the source document and use this option, the text will be pasted in one line, not multiple lines.
    • Paste Table Read more… Select this option if you want to paste the text into a table.
    • Paste List Read more… Select this option if you want to paste the text as a list.
    • Paste HTML Read more… Select this option if you want to paste text from an HTML source, such as a website.
