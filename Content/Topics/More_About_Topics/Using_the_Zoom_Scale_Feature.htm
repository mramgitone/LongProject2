Using the Zoom/Scale Feature

The bottom part of the XML Editor includes a toolbar that lets you use zoom/scale features to better see your content. The features that are available depend on whether you are working in Web Layout mode or Print Layout mode. See About Layout Modes.

Web Layout Mode

web layout mode

If you are using the Web Layout mode, the following toolbar appears at the bottom of the XML Editor:

You can use the buttons in this toolbar to adjust the size of the text displayed in the XML Editor. Please note that the font is scaled only so that you can more easily see it in the editor. The actual font size is not adjusted for the output. Also, objects such as images remain at their original scale; only the font increases or decreases in scale.

Print Layout Mode

print layout mode

If you are using the Print Layout mode, the following toolbar appears at the bottom of the XML Editor:

You can use the buttons on the left side of this toolbar for navigating through a topic that is long enough to consist of multiple pages. And you can use the Zoom/Scale button to adjust the size and number of pages shown for the open topic.
