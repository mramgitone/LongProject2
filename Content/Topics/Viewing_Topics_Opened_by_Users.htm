Viewing Topics Opened by Users

Topics

You can work with topic data in in the following ways.

  • Viewing topics opened by users See Viewing Topics Opened by Users.
  • Resetting topic views When you view topics visited by users, one of the columns in the Topic window pane displays the view count (i.e., the number of times that topic has been visited). You can reset the topic view count so that the counter starts over at 0. See Resetting Topic Views.
  • Viewing context-sensitive Help topics See Viewing Context-Sensitive Help Topics Opened by Users.
  • Resetting context-sensitive Help calls When you view topics visited by users based on context-sensitive Help (CSH) calls, one of the columns in the window pane displays the CSH count (i.e., the number of times that topic has been visited). You can reset the CSH call count so that the counter starts over at 0. See Resetting Context-Sensitive Help Calls.
  • Viewing live topics If you publish output to a web server using one of the WebHelp outputs, you can take advantage of a "Live View" button in the Topic View window pane. This button opens the live topic in a browser window so that you can view it just as users are seeing it. See Viewing Live Topics in a Browser.
