Resetting Context-Sensitive Help Calls

When you view topics visited by users based on context-sensitive Help (CSH) calls, one of the columns in the window pane displays the CSH count (i.e., the number of times that topic has been visited). You can reset the CSH call count so that the counter starts over at 0. You might use this feature, for example, if you have been testing your topic views in and want to reset the counter to 0 before you publish your output.

how to reset csh calls
  2. Open the Context-Sensitive Help Calls window pane (select View>Context-Sensitive Help Calls).
  3. Select the rows for which you want to reset the CSH calls. You can hold down your SHIFT or CTRL key and click to select a range or rows or individual rows, respectively.
  4. In the local toolbar, click the Reset CSH Call Counts button. Then click Yes.
 notes

You can also reset topic views, topic ratings, and search phrase counts. See Resetting Topic Views, Resetting Topic Ratings, and Resetting Search Phrase Counts.
