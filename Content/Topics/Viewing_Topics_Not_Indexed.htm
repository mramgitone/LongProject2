how to view topics not yet indexed

    • Ribbon In the Analysis section select Project Analysis>Topics Not In Index.

    • Menu Select View>Project Analysis>Topics Not In Index.

    The Project Analysis window pane opens, with the "Topics Not In Index" option selected from the drop-down menu at the top. The area at the bottom of the window pane displays all topics where you have not yet inserted index keywords.

  2. Open a project.

    • Ribbon In the Dynamic Reports section select More Reports>Topics Not In Index.

    • Menu Select View>More Windows>Topics Not In Index.

    The Topics Not In Index window pane opens, displaying all topics in your project where you have not yet inserted index keywords.

    • File
    • Title
    • Folder

  6. You can double-click open any topic in the list to view or modify it. You can open the file in the interface or in the project.
