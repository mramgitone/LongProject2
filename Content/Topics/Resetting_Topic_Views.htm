Resetting Topic Views

When you view topics visited by users, one of the columns in the window pane displays the view count (i.e., the number of times that topic has been visited). You can reset the topic view count so that the counter starts over at 0. You might use this feature, for example, if you have been testing your topic views in and want to reset the counter to 0 before you publish your output.

how to reset topic views

  3. Select the rows for which you want to reset the topic views. You can hold down your SHIFT or CTRL key and click to select a range or rows or individual rows, respectively.
  4. In the local toolbar, click the Reset Topic Views button. Then click Yes.
 notes

This does not remove any comments or ratings associated with topics visited. It merely resets the view counter.

You can also reset topic ratings, context-sensitive Help calls, and search phrase counts. See Resetting Topic Ratings, Resetting Context-Sensitive Help Calls, and Resetting Search Phrase Counts.
