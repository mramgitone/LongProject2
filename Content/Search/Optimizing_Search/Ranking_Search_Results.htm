Ranking Search Results

The search database generates organic search results and looks at numerous signals, factors, and metrics to determine how relevant the results are to the query terms. Understanding how searches are ranked in the different output types helps you determine strategies for optimizing your search results.

ranking results—dotnet help, webhelp, webhelp air, webhelp mobile, and html5

Below is a general explanation about the search engine:

content signals

When end users enter their query terms in output, the search engine looks at a variety of signals, such as:

  • Do the search term(s) occur as the first word (or first words) of the topic title? 
  • Do the search term(s) occur in the topic title?
  • Do the search term(s) occur in the file name?
  • Do the search term(s) occur in a heading tag?
  • Do the search term(s) occur in the body of a topic?
  • How many times do the search term(s) occur in the body of the topic?
  • Do the search term(s) occur in an index marker in the topic?
  • Do the search term(s) occur in close proximity to one another (i.e., how close together are the keywords?) in the body of a topic?
 content weighting

As with most search engines, takes a variety of factors into account to when ranking its search results. Since retrieved results are relative to the search query and indexed content, the table below describes in general terms how different content elements are weighted:

Content Elements

Description

Title Tags

The topic title lets readers know the topic of a particular page. It is placed in a <title> tag, within a <head> tag in HTML documents.

You can add a topic title by right-clicking the topic in the Content Explorer, and selecting Properties from the context menu. In the Properties dialog, click the Topic Properties tab, and enter text in the Topic Title field.

You also have the option to leave the Topic Title field blank, in which case the first topic heading will be used as the topic title by default.

H1 Heading Tags

Keywords contained in the <h1> tag are given less weight than the topic title, but they are still an important factor.

File Names

Keywords included in a file name are factored into search.

Index Keywords in Topics

The search engine looks at topics for index keywords. Index keywords in a topic have less weight than <h1> tags, but more weight than subsequent heading tags (e.g., <h2> through <h6>

note

If you do not want index keywords to be considered when retrieving search results, you can exclude them from the search. See Excluding Index Entries From Search.

Subsequent Heading (H2-H6) Tags

The search engine also looks for keywords in <h2> through <h6> tags in your topics.

In the Body Text

The search engine also gives weight to factors such as keyword density, frequency, proximity, and the use of styles or emphasis in the body of document (i.e., text contained within the <body> tag on an HTML page).
