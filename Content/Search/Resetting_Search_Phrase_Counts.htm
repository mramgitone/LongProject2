Resetting Search Phrase Counts

When you view search phrases used by readers, one of the columns in the window pane displays the view count. You can reset the search phrase count so that the counter starts over at 0. You might use this feature, for example, if you have been testing used search phrases in and want to reset the counter to 0 before you publish your output.

  4. In the local toolbar, click the Reset Search Count button. In the message that displays click Yes.
 notes

You can also reset topic views, topic ratings, and context-sensitive Help calls. See Resetting Topic Views, Resetting Topic Ratings, and Resetting Context-Sensitive Help Calls.
