Viewing Topics With Snippet Conditions

See Using Snippet Conditions.

how to view topics with snippet conditions
  1. Open a project.

    • Ribbon In the Dynamic Reports section select More Reports>Topics With Snippet Conditions.

    • Menu Select View>More Windows>Topics With Snippet Conditions.

    The Topics With Snippet Conditions window pane opens, displaying all topics in your project where snippet conditions have been applied.

    • File
    • Folder
    • Expression Displays the condition expression used.

  3. You can open any topic in the list to view or modify it. You can open the file in the interface or in the project.
