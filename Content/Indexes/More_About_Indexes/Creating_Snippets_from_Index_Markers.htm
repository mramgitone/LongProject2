Creating Snippets from Index Markers

If you want to reuse the same index marker in many topics, you can convert the index marker to a snippet. Then you can insert that snippet into as many topics as you want.

how to create a snippet from an index marker
  2. In the XML Editor, right-click the index marker that you want to use to create a snippet.
  3. From the context menu, click Select.

    • Ribbon In the Snippet section select Create Snippet.

    • Menu Select Format>Create Snippet.

    The Create Snippet dialog opens.

  5. In the Snippet File field, type a new name for the snippet.
  6. It is recommended that you leave the project folder selection as "Resources\Snippets." After the snippet is created, you can see the snippet file in the Content Explorer.
  7. Make sure that a check mark is displayed in Replace Source Content with the New Snippet.
  8. Click Create. The snippet is inserted into the topic in surrounded by brackets.
 notes

A marker can hold multiple index keywords.

You can access the actual index marker by opening the snippet from the Content Explorer. The snippet opens in its own page in the XML Editor, showing the index marker and keywords, which you can modify if necessary.
