Creating Snippets from Scripts

If you want to reuse the same script in many topics, you can convert the script to a snippet. Then you can insert that snippet into as many topics as you want.

how to create a snippet from a script
  2. In the XML Editor, right-click the script that you want to create a snippet from.
  3. From the context menu, click Select.

    • Ribbon In the Snippet section select Create Snippet.

    • Menu Select Format>Create Snippet.

    The Create Snippet dialog opens.

  5. In the Snippet File field, type a new name for the snippet.
  7. Make sure that a check mark is displayed in Replace Source Content with the New Snippet.
  8. Click Create.
 note

You can access the actual script by opening the snippet from the Content Explorer. The snippet opens in its own page in the XML Editor, showing the script icon , which you can modify if necessary.
