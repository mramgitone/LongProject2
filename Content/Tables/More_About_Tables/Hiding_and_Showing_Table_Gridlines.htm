Hiding and Showing Table Gridlines

Gridlines are the gray lines that indicate the edges of cells in a table. They appear in the XML Editor so that you can distinguish where rows, columns, and cells are located. Gridlines do not appear in the final output unless you add borders to the table.

If you prefer not to see table gridlines, you can easily turn them off. If the gridlines are not shown and you want to see them, you can easily turn them on.

how to hide or show table gridlines

    • Ribbon In the Table section click the check box next to Show Gridlines.

    • Menu Select Table>Show Gridlines.
