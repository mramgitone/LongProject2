Adding Rollover Captions

A rollover caption is an effect where you designate an area on a frame for the user to hover the mouse over. When the user does this, a caption with text that you provide appears on the frame. When the user moves the mouse away from the designated area, the caption disappears.

There are a couple of ways to create a rollover caption. You can either create the hover shape first, or you can create the caption first. Steps for both methods are given.

how to add a rollover caption (create hover shape first)
  1. Open the frame.
  2. In the Frame Editor, draw a shape (e.g., rectangle, circle) over the area of the frame to be associated with the caption. This shape represents the hover area.

    • Local Toolbar Select the hover shape. In the local toolbar of the Frame Editor, click the down arrow next to the New Button Mode button . From the drop-down list, select Rollover Caption Mode.
    • Right-Click Right-click the hover shape. In the pop-up menu, select Add Rollover Caption.

    The cursor changes to small crosshairs. Click in the frame and draw the bubble shape, which will serve as the caption.

  4. Change the text in the bubble shape by doing one of the following.
    • Click on the shape and start typing. Use the buttons and fields at the top of the small window to apply formatting to the text. When you are finished, click somewhere in the frame.
    • Double-click the shape. In the Rollover Caption Properties dialog, select the Rich Text tab and enter the text for the caption. Use the buttons and fields at the top of the tab to apply formatting to the text.
  5. (Optional) Change the look of the rollover caption or the hover shape, or both. To do this, double-click the shape. Then use the tabs in the properties dialog to change the look of the shape to meet your needs.
  6. (Optional) Move or resize either shape as necessary.
how to add a rollover caption (create caption first)
  1. Open the frame.

    • Ribbon In the Editing section, click the Rollover Caption Mode button .

    • Menu Select Insert>Action Shape>Rollover Caption Mode.
    • Local Toolbar Select the hover shape. In the local toolbar of the Frame Editor, click the down arrow next to the New Button Mode button . From the drop-down list, select Rollover Caption Mode.

    The cursor changes to small crosshairs.

  3. Click in the frame and drag the mouse to draw a bubble shape, which will serve as the caption.
  4. Change the text in the bubble shape by doing one of the following.

    • Click on the shape and start typing. Use the buttons and fields at the top of the small window to apply formatting to the text. When you are finished, click somewhere in the frame.

      OR

    • Double-click the shape. In the Rollover Caption Properties dialog, select the Rich Text tab and enter the text for the caption. Use the buttons and fields at the top of the tab to apply formatting to the text.
  5. Now use one of the buttons in the local toolbar to draw a shape (e.g., rectangle, circle) over the area of the frame to be associated with the caption. This shape represents the hover area.
  6. Right-click the hover shape. In the pop-up menu, select Pick Rollover Caption. The background of the frame turns gray.
  7. Click the rollover caption (bubble shape) that you created. The gray background disappears.
  8. (Optional) Change the look of the rollover caption or the hover shape, or both. To do this, double-click the shape. Then use the tabs in the properties dialog to change the look of the shape to meet your needs.
  9. (Optional) Move or resize either shape as necessary.
notes

If an icon like this is displayed on a rollover caption, that means the caption is not yet connected with an object. After you associate the caption with an object, the icon will disappear.

Rollover captions are not supported on some devices, such as the iPad.
