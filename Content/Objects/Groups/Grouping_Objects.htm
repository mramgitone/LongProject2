Grouping Objects

You can group multiple objects into an object group, which then acts as a single object. And at any point, you can ungroup the objects so that they are separate once again.

example

example

Let's say you have a frame an image with three objects—a rectangle, a line segment, and an image object, like this:

You can hold down your CTRL key and select all three objects.

Then you can either click in the local toolbar or right-click and select Group Selected Items, which converts the three objects into a single object group.

 how to group objects
  1. Open an image containing multiple objects.
  2. Open a frame containing multiple objects.
  3. Hold down your CTRL key and click the objects that you want to group together or click and drag across the objects you want to select.
