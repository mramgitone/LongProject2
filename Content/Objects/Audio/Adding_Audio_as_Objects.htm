Adding Audio as Objects

In addition to the ability to add audio to objects such as shapes and buttons, you can also add sounds as objects themselves. As with any other type of object, you can add as many audio objects as you want to a movie frame, and you can adjust their timing in the Timeline window pane. The Timeline window pane is also the best way to determine if a frame contains an audio object, since there is no visual indication of this in the frame itself.

how to add audio as an object
  1. Open the frame.

    • Ribbon In the Insert section, click the Insert Sound button.

    • Menu Select Insert>Sound.
    • Local Toolbar In the local toolbar of the Frame Editor, click .

    The Add Sound to Frame dialog opens.

  3. select existing mp3 or wav file from sound library
    1. If you have already imported or recorded an audio file, click the drop-down on the left side of the dialog and select Sound Library.
    2. In the field below, select an audio file.
         select system sound
    1. Click the drop-down on the left side of the dialog and select System Sounds.
    2. In the field below, select an audio file.
         import mp3 or wav file into sound library
    1. Click Import new sound.
    2. In the dialog that opens, find and double-click the audio file.
         record new mp3 file
    1. Click in the Select recording device drop-down field and choose and input source. (Depending on the input source, you may also need to attach a microphone to your computer.)
    2. Click the red record button.
    3. Speak or play the sound you want to record, and when you are finished click the square button.
    4. (Optional) In the dialog that opens, you can type a new name for the sound.
    5. Click OK.

  4. (Optional) With the file selected in the Add Sound to Frame dialog, you can click Play to listen to the audio.

  5. Click OK. The sound is added to the frame as an object, as indicated by the tan bar in the Timeline window pane.

 note

You can also add audio to existing objects. In addition you can add audio to a movie and to individual frames in a movie. See Adding Audio to Objects, Adding Audio to Movies, and Adding Audio to Frames.

After you add an audio object to a frame, you can adjust it's start and end time by dragging the tan bar in the Timeline window pane.
