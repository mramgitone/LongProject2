About Objects

An object is an element that you can add in a frame to enhance or explain areas of the frame (e.g., callouts), or to incorporate actions into the frame (e.g., buttons, cursor trajectories). Objects can be elements such as shapes, lines, cursors, buttons, or even other images. Each object is stored on its own layer in the frame. This enables you to overlap objects and edit them individually.

A collection can contain multiple movies. Each movie can be composed of many frames. And each frame of a movie can contain multiple objects.

Types of Objects

types of objects

Following are some of the objects that you can add to a movie frame.

  • Callouts Read more… A callout is simply a shape that calls attention to something in a frame. For example, you might want to instruct the user to click a certain area in the frame. A callout is a way to provide the user with the instructions, as well as point to the appropriate area in the frame. See Adding Callouts.
  • Cursors Read more… When you record a movie, bitmaps of cursors are added automatically to the frames, along with trajectories of their movements. By default, cursors are displayed in a lightly shaded oval shape. However, you can edit the appearance of the cursor shape (e.g., if you do not want the oval shape to be seen at all, you could specify that the shape should have no border or fill color). If you do not want to use the style of the default cursor, you can create your own look and set it as the default cursor style for recordings. In addition to adding cursors automatically during a recording, you can add a cursor manually to a frame. You can even set a trajectory for that cursor. See Adding Cursors.
  • Buttons Read more… A button is an object that looks and acts like a button that you might see in a software application. You can add buttons to frames in a movie as a way to include navigation and allow a user to move from one frame to another. See Adding Buttons.
  • Shapes Read more… There are various shapes that you can add to a frame, such as a bubble, polygon, oval, and rectangle. See Adding Shapes to Frames.
  • Typing Objects Read more… You can add typing to your movie through the use of animated typing boxes and user input boxes. See About Typing Objects.
    • Animated Typing Boxes Read more… When you record a movie, captures any typing actions that you perform on screen. This typing is therefore animated in the movie output. In other words, each letter of the content is typed as the user watches it. Not only are these animated text boxes added automatically when you record movies, but you can also add them manually when you edit the movie. See Adding Animated Typing Boxes.
    • Input Boxes Read more… If you want the user to enter text at a certain frame in the movie, you can accomplish this by adding an input box. You can then enter the exact content that you want the user to type. You can also format the text, as well as specify captions and actions based on the user's input. See Adding Input Boxes.
  • Rollover Captions Read more… A rollover caption is an effect where you designate an area on a frame for the user to hover the mouse over. When the user does this, a caption with text that you provide appears on the frame. When the user moves the mouse away from the designated area, the caption disappears. See Adding Rollover Captions.
  • Lines Read more… has a polyline tool that lets you create a line with one or more line segments. You can also specify whether the line should include arrows, a shadow, or other properties. See Adding Lines.
  • Image Objects Read more… In addition to adding objects such as shapes, lines, and buttons to a frame, you can also insert another image as an object. Like other objects, this image resides in its own layer on the frame. You can insert any image files of the following types: BMP, JPG, JPEG, GIF, PNG, TIF, TIFF. See Adding Images as Objects.
  • Audio Objects Read more… In addition to the ability to add audio to objects such as shapes and buttons, you can also add sounds as objects themselves. As with any other type of object, you can add as many audio objects as you want to a movie frame, and you can adjust their timing in the Timeline window pane. The Timeline window pane is also the best way to determine if a frame contains an audio object, since there is no visual indication of this in the frame itself. See Adding Audio as Objects.
  • Object Groups Read more… You can group multiple objects into an object group, which then acts as a single object. And at any point, you can ungroup the objects so that they are separate once again. See Grouping Objects.
