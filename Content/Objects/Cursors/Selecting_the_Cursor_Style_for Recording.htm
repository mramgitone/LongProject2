Selecting the Cursor Style for Recording

When you record a movie, records your movements and displays the cursor and its trajectory on each frame. Unless you specify otherwise, the default cursor style is used.

Before you begin recording a movie, you can select the style to be used for the cursors. You can select a factory style or you can select a style from any of your palettes.
