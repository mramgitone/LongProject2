Selecting an End Action for a Cursor

example

example

Let's say you have a frame containing a cursor movement, and you want to give users a choice of performing a task (e.g., clicking an area in your movie) or watching as the task is performed for them (watching the cursor move along its trajectory and click the area). In a situation such as this, you might do the following:

  1. Add a rectangle shape and place it around the area that you want the user to click. Then double-click that rectangle and select an action for it, such as "GotoNext."
  2. Add a callout, pointing it to that rectangle area. Add text to the callout, providing instructions for the user (e.g., "Click this area. Otherwise, click the button below to see it done.").
  3. Add a button. Give it an appropriate label (e.g., "Do it for me"). Double-click the button and select the "StartCursor" action for it.
  4. Select an end action for the cursor. See the steps below for details. In this example, you might select an action such as "GotoNext." The end action is necessary in case the user opts to click the "Do it for me" button.
  5. Pause the timing of the frame. To do this, double-click the frame and select "Pause movie at end of frame" on the Transition tab. This will ensure that the frame waits for the user to perform an action (either clicking in the rectangle area or clicking the button) before moving on.
 how to select an end action for a cursor
  1. Open the frame.
  2. In the Frame Editor, double-click the cursor shape. The Cursor Properties dialog opens.
  3. Select the End Action tab.

  4. Select the action that you want to apply to the cursor.
