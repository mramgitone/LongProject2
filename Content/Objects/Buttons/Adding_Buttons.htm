Adding Buttons

You can add buttons to frames in a movie. Adding a button is a way to include navigation and allow a user to move from one frame to another (by applying an action to the button).

how to add a button to a frame
  1. Open the frame.

    • Ribbon In the Editing section, click the Button Mode button .

    • Menu Select Insert>Action Shape>Button Mode.
    • Local Toolbar In the local toolbar of the Frame Editor, click the down arrow next to the New Button Mode button and from the drop-down list, select Button Mode.

    The cursor changes to small crosshairs.

  3. Click in the frame and drag the mouse to draw a rectangle shape somewhere in the frame. Release the mouse button when you are finished.

    The button is added to the frame, with the default text "Click Me." (You can change the look of the button, as well as the text displayed, by double-clicking the button and making selections in the Button Properties dialog. You can use the Button Action tab to select an action for the button.)

 notes

If an icon like this is displayed on a button, that means the button is not yet associated with an action. If an icon like this is displayed on a button, that means the action link for that button is broken and needs to be fixed. After you select a valid action for the button, the icon will disappear.
