Setting Mouse Hover Properties

how to set mouse hover properties
  1. Open a frame that has a button added to it.
  2. Double-click the button. The properties dialog for the object opens.
  3. Select the Mouse Hover tab, and adjust the settings in the following fields.
    • Background Read more… Use this area to set the color for the background of the button. In the first section, select the direction of the color gradient, or select to use a solid color.
    • Fill Start Read more… Click the down arrow to select a color for the start of the background color. If you select a different color for the "fill end," the object background will be displayed as a gradient of colors moving from the start color to the end color.
    • Fill End Read more… Click the down arrow to select a color for the end of the background color. If you select a different color for the "fill start," the object background will be displayed as a gradient of colors moving from the start color to the end color.
    • Line:Color Read more… Click the down arrow to select a color for the line of the button.
    • Width Read more… Set the width of the line for the button (in pixels).
    • Font:Color Read more… Click the down arrow to select a font color for the text in the button.
    • Bold Read more… Select this check box if you want the text in the button to display in bold.
    • Italic Read more… Select this check box if you want the text in the button to display in italics.
    • Underline Read more… Select this check box if you want the text in the button to be underlined.
 notes

You can also apply specific properties to display the button in a particular way when the button is pressed. See Setting Mouse Pressed Properties.
