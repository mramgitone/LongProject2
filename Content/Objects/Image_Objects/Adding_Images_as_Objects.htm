Adding Images as Objects

In addition to adding objects such as shapes, lines, and buttons to a frame, you can also insert another image as an object. Like other objects, this image resides in its own layer in the frame. You can insert any image files of the following types: BMP, CUR, JPG, JPEG, GIF, PNG, TIF, TIFF.

how to add an image as an object
  1. Open the frame.

    • Ribbon In the Insert section, click the face of the Insert Image button to insert an image file, or select the drop-down arrow on the Insert Image button and choose either Insert Image, or Insert Cursor File.

    • Menu Select Insert>Image File (if you want to add a cursor image, you can select Insert>Cursor File instead).

    The Open dialog opens.

  3. In the Open dialog, locate and double-click the file. That image is added to the frame as an object.
  4. To move the image object, click in the object and drag it to a new location on the frame.
