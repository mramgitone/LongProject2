Adding Callouts

A callout is simply a bubble shape that calls attention to something in a frame. For example, you might want to instruct the user to click a certain area in the frame. A callout is a way to provide the user with the instructions, as well as point to the appropriate area in the frame.

When you record a movie, callouts are added automatically to the frames. If you do not want to use the style of the default callout, you can create your own look and set it as the default callout style for recordings. You can also add a callout manually to a frame by following the steps below.

 how to add a callout to a frame
  1. Open the frame.

    • Ribbon In the Editing section, click .

    • Menu Select Insert>Bubble Mode.

    • Local Toolbar In the local toolbar of the Frame Editor, click the Bubble Mode button .

    The cursor changes to small crosshairs.

  3. Click in the frame and drag the mouse to draw a rectangle somewhere in the frame. Release the mouse button when you are finished. The bubble shape appears on the frame.
  4. Click somewhere in the middle of the bubble and drag it to the appropriate location on the frame. If you click on the small bubble icon within the shape, the entire shape moves, except the pointer. If you click elsewhere inside the shape, the entire shape moves, including the pointer.
  5. Click the point (small circle) at the tip of the pointer on the bubble shape and drag it to the appropriate location.
  6. Double-click the bubble shape. The Bubble Properties dialog opens.
  7. Select the Rich Text tab.
  8. In the bottom portion of the tab, enter the text that you want to display in the bubble shape.

    9. Use the fields at the top of the dialog to change properties such as the color, font, and size for the text.

    Alternatively, you can use the Font tab to set the font properties. The Font tab applies the format properties to all of the text in the shape, whereas the Rich Text tab applies the format properties only to selected text in the shape.

  9. Use the other tabs in the dialog to set properties for the shape and change its look (e.g., border, fill color, shadow).
  11. If necessary, you can resize the bubble shape by clicking and dragging any of the points around the edge of the shape.
