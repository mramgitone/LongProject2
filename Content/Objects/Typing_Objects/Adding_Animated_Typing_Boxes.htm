Adding Animated Typing Boxes

When you record a movie, can capture any typing actions that you perform on screen. This typing is therefore animated in the movie output. In other words, each letter of the content is typed as the user watches it. Not only can these animated text boxes be added automatically when you record movies, but you can also add them manually when you edit the movie.

how to add an animated typing box
  1. Open the frame.

    • Ribbon In the Editing section, click the Typing Box Mode button .

    • Menu Select Insert>Action Shape>Typing Box Mode.
    • Local Toolbar In the local toolbar of the Frame Editor, click the down arrow next to the New Button Mode button and from the drop-down list, select Typing Box Mode.

    The cursor changes to small crosshairs.

  3. Click in the frame and drag the mouse to draw a rectangle over the area of the frame where you want the animated typing to be displayed. Release the mouse button when you are finished. An empty typing box appears on the frame.
  4. Double-click the typing box. The Typing Box Properties dialog opens.
  5. In the Typing Box tab, complete any of the following.
    • Start Text Enter the text (if any) that you want to be displayed initially in the typing box in the output. If you do not want any start text, leave this field empty.
    • Typed Text Enter the text that you want to be typed in the box as the frame plays. This text will follow after any start text that is included.

  7. (Optional) You can make other modifications to the typing box, including the following.

    modify the look of the typing box
    1. Double-click the typing box (not the frame).
    2. In the Typing Box Properties dialog, use the various tabs to make changes to the typing box (e.g., background color, border, font, animation effects, curve of rectangle).
    increase or decrease the typing speed
    1. Double-click the typing box (not the frame).
    2. In the Typing Box Properties dialog, click the Time Span tab.

    3. In the Length field, increase or decrease the number of seconds as necessary.

      If you increase the number of seconds, be aware that the timing of the frame must also be long enough to accommodate it.

         move or resize the typing box
    • To move the entire typing box, click somewhere in the middle of the box and drag it to the new location.
    • To resize the typing box, click any of the points (i.e., small circles) around the edge of the box and drag it to adjust the height and/or width.
note

When you record in Automatic mode, you can select whether to add typing box shapes when you perform typing actions. If you select this option, it inserts a typing box object when you type during a recording session. If you disable this option and type during a recording session, the frame in question becomes a full-motion video (FMV) frame. See Adding Typing Box Shapes for Recording.
