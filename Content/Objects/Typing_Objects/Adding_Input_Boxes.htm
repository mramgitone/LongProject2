Adding Input Boxes

If you want the user to enter text at a certain frame in the movie, you can accomplish this by adding an input box. You can then enter the exact content that you want the user to type. You can also format the text, as well as specify captions and actions based on the user's input.

how to add an input box
  1. Open the frame.

    • Ribbon In the Editing section, click the Input Box Mode button .

    • Menu Select Insert>Action Shape>Input Box Mode.
    • Local Toolbar In the local toolbar of the Frame Editor, click the down arrow next to the New Button Mode button and from the drop-down list, select Input Box Mode.

    The cursor changes to small crosshairs.

  3. Click in the frame and drag the mouse to draw a rectangle over the area of the frame where you want the input box to be displayed. Release the mouse button when you are finished. The input box appears on the frame with the text "(input box)."
  4. Double-click the input box. The Input Box Properties dialog opens.
  5. Select the Input Box tab.
  6. In the Correct Entries area, enter the text that you want the user to type. You can add one line of text or multiple lines of text. If you add multiple lines of text, the user only needs to type one of those entries correctly in the output to continue.
  7. In the Options area, you can enter the number of attempts the user will be given to type the correct text.
  8. Select the Success Action tab to set an action to perform after the user types the input correctly and presses Enter. You can select any of the following actions for the tab.For descriptions of each action, see Selecting Actions for Objects.
    • GotoFrame Read more… Select this option to open a specific frame in the movie. When you select this option, the Frame Name field appears at the bottom of the tab. Enter the name of the destination frame in this field. (The name of the frame is identified above it in the Frames window pane. You can use the default number associated with the frame, or you can change the name in the Frame Properties dialog.)
    • GotoFirst Read more… Select this option to open the first frame in the movie.
    • GotoNext Read more… Select this option to open the next frame in the movie.
    • GotoPrevious Read more… Select this option to open the previous frame in the movie.

    • StartCursorRead more… Select this option to start the cursor movement (if the frame has one).

      example

      This option would be useful if you have a frame containing a cursor trajectory, and you want to give users a choice of performing a task (e.g., clicking an area in your movie) or watching as the task is performed for them (watching the cursor move along its trajectory and click the area). In a situation such as this, you might do the following.

      1. Add a rectangle shape and place it around the area that you want the user to click. Then double-click that rectangle and select an action for it, such as "GotoNext."
      2. Add a callout, pointing it to that rectangle area. Add text to the callout, providing instructions for the user (e.g., "Click this area. Otherwise, click the button below to see it done.").
      3. Add a button. Give it an appropriate label (e.g., "Do it for me"). Double-click the button and select the StartCursor action for it.
      4. Select an end action for the cursor. To do this, double-click the cursor on the frame, and select an action (e.g., "GotoNext") on the End Action tab. This is needed in case the user opts to click the "Do it for me" button.
      5. Pause the timing of the frame. To do this, double-click the frame and select "Pause movie at end of frame" on the Transition tab. This will ensure that the frame waits for the user to perform an action (either clicking in the rectangle area or clicking the button) before moving on.

    • Pause Read more… Select this option to pause the movie. The movie will remain paused until the user clicks the object again.
    • AcceptRead more… Select this option to accept the user's input. Let's say you add an input box to a frame. After users complete the input box, they can usually simply press Enter on the keyboard. However, if you want users to click say, a button, when they are finished with the input, you can create a button and apply this action to it. After users finish typing the input, they can click that button to submit the input (instead of pressing Enter on the keyboard).
    • OpenRead more… Select this option to open a remote file (such as a website). When you select this option, the URL field appears at the bottom of the tab. Enter the full URL in this field. With this option, the website or other file will open in the viewer used for displaying the movie (e.g., MadCap Movie Viewer, MadCap Help Viewer).
    • OpenWithDefaultRead more… Select this option to open a remote file (such as a website). When you select this option, the URL field appears at the bottom of the tab. Enter the full URL in this field. With this option, the website or other file will open in the default Windows application for the file type (e.g., HTM files open in Internet Explorer, DOC files open in Microsoft Word).
    note

    You can also select a failure action (on the Failure Action tab). This lets you specify an action to take if the user does not enter the correct text in the number of attempts allowed.

 notes

You can add only one typing input box per frame.

If an icon like this is displayed on an input box, that means the box is not yet associated with an action. If an icon like this is displayed on an input box, that means the action link for that box is broken and needs to be fixed. After you select a valid success action for the input box, the icon will disappear.

Rollover captions work with input boxes in the MadCap Mimic Movie Format (MMF) only; they do not work in .

After you create an input box and apply a success action, you may want to add either of the following to enhance the user experience with it.

  • Add a failure caption. Add a message to display to users if they fail to complete the input box successfully in the specified number of attempts.
  • Add a hint caption. Add a message to display to users if they hover over the input box for help.
