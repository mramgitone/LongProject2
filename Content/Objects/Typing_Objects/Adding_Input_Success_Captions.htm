Adding Input Success Captions

After you add an input box to a frame, you can add a success caption to accompany it. This is a message that displays to users after they complete the input box successfully.

how to add a success caption
  1. Open the frame where you have added an input box.

    • Ribbon In the Editing section, click the Input Success Caption Mode button .

    • Menu Select Insert>Action Shape>Input Success Caption Mode.
    • Local Toolbar In the local toolbar of the Frame Editor, click the down arrow next to the New Button Mode button and from the drop-down list, select Input Success Caption Mode.

    The cursor changes to small crosshairs.

  3. Click in the frame and drag the mouse to draw a rectangle somewhere in the frame. Release the mouse button when you are finished. A bubble shape appears on the frame and points to the input box.
  4. If necessary, click somewhere in the middle of the bubble and drag it to the appropriate location on the frame. If you click on the small bubble icon within the shape, the entire shape moves, except the pointer. If you click elsewhere inside the shape, the entire shape moves, including the pointer.
  5. If necessary, click the point (small circle) at the tip of the pointer on the bubble shape and drag it to the appropriate location.
  6. Double-click the bubble shape. The Bubble Properties dialog opens.
  7. Select the Rich Text tab.
  8. In the bottom portion of the tab, enter the text that you want to display in the bubble shape.

    9. Use the fields at the top of the dialog to change properties such as the color, font, and size for the text.

    Alternatively, you can use the Font tab to set the font properties. The Font tab applies the format properties to all of the text in the shape, whereas the Rich Text tab applies the format properties only to selected text in the shape.

  9. Use the other tabs in the dialog to set properties for the shape and change its look (e.g., border, fill color, shadow).
  11. If necessary, you can resize the bubble shape by clicking and dragging any of the points around the edge of the shape.
 notes

If an icon like this is displayed on an input success caption, that means the caption is not yet connected with an input box. After you draw an input box on the frame, the icon will disappear.

You may also want to add one of the other captions associated with an input box:

  • Add a failure caption. Add a message to display to users if they fail to complete the input box successfully in the specified number of attempts.
  • Add a hint caption. Add a message to display to users if they hover over the input box for help.
