Linking a Browse Sequence Entry to a TOC

You can link a browse sequence entry to a TOC.

how to link a browse sequence entry to a toc
  1. Open the table of contents.
  2. Select a TOC entry.
  3. In the local toolbar, click . The Properties dialog opens.
  4. Select the General tab.
  5. Click Select TOC. The Link to TOC dialog opens, displaying all the TOCs in your project.
  6. Select the TOC to which you want to link the entry and click Open.
