Output Files for the HTML5 Format

output files for the html5 format

Following are output files for the HTML5 format.

  • HTM Read more… This is the main entry file for the output in the format if the movie is to be viewed in a browser. The HTM file is named after each movie (e.g., myfirstmovie.htm, mysecondmovie.htm).

  • Support Files for Browsers Read more… In addition to the main output file, there are also various support folders and files that you will need in order for your movies to play in browsers correctly.

  • Single Video Files Read more… These are extra standalone .mp4 and .webm files that are created if you select an option to produce them. They are stored in the "videos" output subfolder and can be distributed by themselves, without any support folders or files. The regular browser-related support files are necessary to view your movies in the correct browser. On the other hand, the single video files—.webm or .mp4—can be used to play movies in a standalone player, such as Windows Media Player or QuickTime. Single MP4 files can be used to view movies on the iPhone.You can play single MP4 or WebM files on YouTube.

    In order to generate single video files for HTML5, you must have Internet Explorer (IE) 9 or higher installed. Therefore, you must be working on an operating system that supports IE 9.

Output Files for the Microsoft Silverlight and Adobe Flash Formats

output files for the microsoft silverlight and adobe flash formats

Following are output files for the Microsoft Silverlight and Adobe Flash formats.

  • SWF (Flash Only) Read more… This is the Flash output file that is created when you generate movies (whether the movies are part of a collection or standalone) in the format. If you use an embedded skin for standalone movies, the SWF file is the only file you need to distribute.

  • HTM Read more… This is the main entry file for the output in and formats (unless you used an embedded skin for Flash standalone movies). The HTM file is named after each movie (e.g., myfirstmovie.htm, mysecondmovie.htm). The output plays in a browser window.

  • Support Files Read more… In addition to the main output files, various extra files are created to support the movie output.

Output Files for the Adobe AIR Format

output files for the adobe air format

Following are output files for the Adobe AIR format.

  • AIR Read more… This is the only file that is created when you generate output from . Users can double-click this file to create the executable (EXE) file. In the future, each time users open the AIR file, the installation process repeats; a new EXE file is created or the old one is replaced.
  • EXE Read more… This is the executable file that is created when users open the AIR file. By accessing the EXE file, users can view the output. In the future, users can open the EXE file, which is placed in their Start menu, to see the output.
