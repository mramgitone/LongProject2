 how to select fonts using local formatting
  1. Select the content to which you want to apply a font.

    • Ribbon In the Font section select a font from the Font Family field.

    • Menu Select Format>Font. The Font Properties dialog opens. Select the Font tab. In the Font field, select a font.

    • Text Format Toolbar Select a font from the Font Family field.

    • Keyboard Shortcut Press CTRL+SHIFT+B. The Font Properties dialog opens. Select the Font tab. In the Font field, select a font.
