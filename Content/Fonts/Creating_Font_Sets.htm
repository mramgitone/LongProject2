Creating Font Sets

Because different types of computers and operating systems do not always have the same fonts, it is recommended that you try to include fonts that cover all of the bases.

example

example

Let's say you want to use Arial if possible. Therefore, you list that font type first in the font set. Next you might add a similar font that is found on Macintosh computers, such as Helvetica, since Macs do not use Arial. And third, you might add a generic font type, such as Sans-serif.

  6. In the Name field, enter a name for the new font set.
  7. From the list of font families on the right side of the dialog, double-click each font family that you want to add to the set. Start with the preferred font family that you want to be used, then select the second, then the third, and so on until you are finished.
  9. Click OK in the Font Family Picker dialog.
  10. Click OK in the Font Properties dialog.

If you want to create additional font sets in the Font Family Picker dialog, you need to first select an existing font set in the list. Then click Define Set, give it a the set a new name, and choose new fonts. As a result, a new font set with that name is added to the list, but the other one also remains in the list just as it was before.

After you create a font set, you can select that font set when applying fonts to content. You can also select that font set as part of the user interface list (i.e., determines which fonts are seen in 's user interface).

note

There is no need to link a font set to a stylesheet. After you create a new font set in a project, it is automatically available for use in any stylesheet you add to the project. You just open the stylesheet, select your style, click the font-family property, and apply it.
