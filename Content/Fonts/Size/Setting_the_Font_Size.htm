Setting the Font Size

You can specify the size of text, choosing from many different units of measurement.

how to set the font size on a style
  1. if using simplified view

      • In the local toolbar of the Stylesheet Editor, click the down arrow in the Font Size field and select a size.

        OR

      • Select the Font tab. From the Size field, click the down arrow in the Font Size field and select a size.

      You can also type a number and a different unit of measurement directly in the field (e.g., 5in, 25px, 10em). See About Units of Measurement.
