Inserting Horizontal rules

lets you insert rules (horizontal lines) into topics. For example, you might use a rule as a design element to separate your topic title from the content.

provides the hr tag, which you select when inserting a rule. You can always change the look of the rule by modifying this style.

If you want to have different rules for your topics, you can open add new style classes for each different horizontal line (based on the <hr> tag). For example, if you want a short rule, you can create a new style class based on the <hr> tag and name it "short." The full name for you new style would be "hr.short." You can then adjust the style property settings for "hr.short" to meet your needs.

note

The hr tag may render differently from browser to browser. For a more consistent look across all browsers, it is a good idea to set both the color and the background-color properties of the hr tag.
