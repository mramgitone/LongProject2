How Did You Do That? Styles

To make sure our content looks good with the page layouts we created, we used styles. Following are the steps we followed for the different elements. If you remove the background images from the page layouts or replace them with others that are larger or smaller, you will probably need to adjust the styles (e.g., indentation, space above/below) so that text fits nicely around the images.

body style

First, we specified some settings on the body style in our stylesheet. We did this because all of the other styles inherit those settings. For example, we wanted most of the styles in our project to use an Arial font. Therefore, it does not make sense to specify Arial on every style. Instead, we specified Arial on the body style, and as a result, all other styles and style classes use it as well. However, we wanted to use Century Gothic as the font for a few styles, so we simply specified that particular font for those styles, overriding the inherited Arial setting.

  1. use the stylesheet editor (beginners)
    1. Open your stylesheet. See Opening Stylesheets.
    4. On the left side of the Stylesheet Editor, select the body style.

    6. Choose the settings shown in the following image (most of these properties should already be set by default):

         get the css (advanced users)

    Enter this in the stylesheet (most of these properties should already be set by default):

    body

    {

    color: #404040;

    display: block;

    font-family: Arial;

    font-size: 8pt;

    line-height: 13px;

    margin: 8px;

    mc-dita-type: body;

    mc-footnote-format: {n};

    mc-footnote-number: page;

    mc-footnote-position: page;

    mc-hyphenate: always;

    mc-hyphenate-maximum-adjacent-line-count: 2;

    mc-hyphenate-shortest-prefix: 3;

    mc-hyphenate-shortest-suffix: 3;

    mc-hyphenate-shortest-word: 5;

    mc-short-line: 10;

    mc-short-line-method: never;

    mc-short-line-step: 0.5;

    mc-short-line-tighten-maximum: 3;

    orphans: 2;

    widows: 2;

    }

body text

We used several paragraph styles to control the various ways that body text appears in our template.

First, we used the standard p style to control the look of regular body text when the background is light.

Second, we created a style class called "p.NarrowLeft" to control the look of body text that has a narrow width due to a background image on the right.

Third, we created a style class called "p.BackPanel" to control the look of body text that needs special placement on the back panel of the foldout.

We used the standard li style to control the look of individual list items on a light background. And we created a style class called "li.DarkBackground" to control the look of individual list items on a dark background.
