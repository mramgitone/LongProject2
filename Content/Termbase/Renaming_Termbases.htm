Renaming Termbases

xxx Keeping in case the feature comes back later

At any point, you can rename a termbase.

how to rename a termbase

    • Ribbon Select View>Termbase.

    • Menu Select View>Termbase.

    • Standard Toolbar Click .

    The Termbase window pane opens.

  2. In the local toolbar, click Manage Termbases.The Manage Termbases dialog opens.
  3. Select the termbase that you want to rename.
  4. Click Rename.The Enter New Termbase Name dialog opens.
  5. Type a new name for the termbase.
