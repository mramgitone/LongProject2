Adding Concepts and Terms to Termbases

After you add a termbase, you can add translated concepts and terms to it. A concept is a group of related terms in different languages. You can do this in various ways.

  • Quick Add Terms (add multiple terms at once) Read more…
  • Termbase Editor (add concepts and terms with additional details) Read more…
  • Translation Editor (add single or multiple terms while translating) Read more…

Quick Add Terms Method

how to add concepts and terms via the quick add terms method

    • Ribbon Select View>Termbase.

    • Menu Select View>Termbase.

    • Standard Toolbar Click .

    The Termbase window pane opens.

  2. In the local toolbar, click . The Batch Add Terms to Termbases dialog opens.
  3. In the Language 1 field, select the first language.
  4. In the Language 2 field, select a second language.
  5. In the first row click in the Original cell and type the term in the first language and press Enter on your keyboard.
  6. In the first row click in the Translation cell and type the term in the second language and press Enter on your keyboard.
  7. In the next row repeat steps 5 and 6. Do this until you've added all the terms you need.
  8. In the field at the bottom of the dialog, select the termbase that you want to add the terms to.
  9. The next time you reopen the Termbase Editor, you will see the concept rows you added, and you can add variations of those terms and additional details for them.

You can also use the "Quick Add Terms" option by clicking in the local toolbar of the Termbase Editor.

Termbase Editor Method

how to add concepts and terms via the termbase editor

    • Ribbon Select the Resourcesribbon. In the TBX section, select Edit Termbase.

    • Menu Select xxx.

    • Translate Toolbar Click xxx.

  2. From the submenu,select the termbase you want to edit.
  3. Click Edit. The Termbase Editor opens.
  4. In the local toolbar, click . The Add Term to Termbase dialog opens.
  5. In the top field, select a language and type the term.
  6. In the next field, select a second language and type the term.
  7. The new concept row is added to the grid in the editor.

  8. In the lower-left section, you can click the plus tab to add the term in another language. You can also click to add variations of the term.

Translation Editor Method

how to add terms via the translation editor
  1. Open a file. See Viewing a List of Files Requiring Translation.
  2. In the Translation Editor, highlight a term (or terms) in a segment that you want to add to the termbase.

    • Right-click in the field and select Add New Term.

      OR

    • In the local toolbar click .

    The Add Term to Termbase dialog opens. The term that you highlighted is also highlighted in the corresponding field in this dialog.

  4. In the next field, highlight or type the corresponding target term.

  5. (Optional) If you want to add metadata to the term, click Advanced. The Add Term to Termbase dialog expands to display additional drop-downs and fields where you can add additional information to the term.

    To return to the basic editor, click Basic.

  6. The new concept row is added to the grid in the Termbase Editor.

After adding terms to a termbase, you can edit them (adding more details) and use them in your translation work. See Editing Termbases and Inserting Terms from a Termbase.
