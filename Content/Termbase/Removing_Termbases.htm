Removing Termbases

If you no longer need a termbase, you can remove it.

Important how to remove a termbase

    • Ribbon Select the Resourcesribbon. In the Termbase section, select Choose Termbase.

    • Menu Select Resources>Choose Termbase.

    • Standard Toolbar Click xxx.

      The Manage Termbases dialog opens.

  2. Select the termbase that you want to remove.
  3. Click Remove.
