Exporting Termbase Files

If you have created a termbase in , you can export it to a TermBase eXchange (TBX) file.

how to Export TBX files—recommended

    • Ribbon Select the Resourcesribbon. In the TBX section, select Export to TBX.

    • Menu Select xxx.

    • Translate Toolbar Click xxx.

      The Export to TBX File dialog opens.

  2. From the Termbase to Export drop-down, select the termbase you want to export to a TBX file. Next to the TBX File field, click . The Choose file to export to dialog opens, where you can locate an existing TBX file you want to overwrite, or enter a name for a new TBX file. Click Save.

  3. Click Start Export.
 how to Export TBX files—Alternate

    • Ribbon Select the Resourcesribbon. In the Termbase section, select Choose Termbase.

    • Menu Select Resources>Choose Termbase.

    • Standard Toolbar Click xxx.

      The Manage Termbases dialog opens.

  2. Click Export.... The Export to TBX dialog opens.

  3. From the Termbase to Export drop-down, select the termbase you want to export to a TBX file. Next to the TBX File field, click . The Choose file to export to dialog opens, where you can locate an existing TBX file you want to overwrite, or enter a name for a new TBX file. Click Save.

  4. Click Start Export.
