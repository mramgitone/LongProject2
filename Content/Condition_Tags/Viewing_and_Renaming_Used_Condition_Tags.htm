how to view used condition tags
  1. Open a project.

    • Ribbon  In the Dynamic Reports section, select Used Items>Used Condition Tags.

    • Menu Select View>Used Items>Used Condition Tags.

    The Used Condition Tags window pane opens, listing all of the condition tags used throughout the project.

    • Condition Tag
    • File
    • Title
    • Folder
    • Tag
    • Exclude Action

  3. You can open any file in the list to view or modify it in an editor. You can open the file in the interface or in the project.

 how to rename condition tags

  1. In the window pane, select all of the condition tags that you want to rename.

    note

    All condition tags that you select will be renamed to the same condition tag that you provide in the following steps.
  2. In the local toolbar, click the Rename button. The Rename Condition tag(s) dialog opens.
  3. In the New condition tag name field, enter a new name.

  5. A message lets you know how many files were changed. All of the selected condition tags are renamed.
