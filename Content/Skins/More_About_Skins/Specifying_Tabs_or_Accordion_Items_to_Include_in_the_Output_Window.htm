Specifying Navigation Elements to Include in the Output Window

You can determine which navigation elements from your project (TOC, Index, Search, Glossary, Browse Sequences, Favorites, Community) that you want users to have access to in the output. If you are creating output, these elements will display as tabs in the output window (except for a glossary, which is included in the TOC as a book). If you are creating , , these elements will display as accordion items in the output window. If you are creating output, the elements will display as simple links. The Community option is for MadCap Pulse and therefore supported only in and .

how to specify tabs or accordion items to include in the output window
  1. Open a skin.
  2. Select the General tab.
  3. In the Features section, click the check box (adding a check mark) next to each navigation element that you want to include in the output. The elements available are different, depending on the type of skin you've opened—Standard, HTML5, or Mobile.

Make sure that you associate the skin with the target. See Associating Skins with Targets.
