Specifying How the TOC Looks in HTML Help Output

how to specify the way the toc looks in html help output
  1. Open a Standard skin.
  2. Select the HTML Help Setup tab.
  3. Click the TOC Options button.

  4. In the HTML Help TOC Options dialog, select the appropriate options. The "Preview" area at the bottom of the dialog changes as you make selections. Click the OK button when you are finished.

    • Lines between items Read more… Displays dotted lines, showing the connections between TOC books and entries.
    • Lines from root Read more… Displays lines connecting TOC books and entries starting at the root.
    • Plus/minus squares Read more… Displays plus and minus squares that open and close TOC books when clicked.
    • Single click to open books Read more… Formats TOC books to open with a single click. If this option is not selected, books open with a double-click.
    • Use folder icons for books Read more… Displays a folder icon instead of a book icon for TOC book entries.
    • Always show the selection Read more… Shows the selected entry even if the TOC tab is not the currently selected item (does not have focus).
    • Binary Read more… For large compiled Microsoft HTML Help projects, a binary TOC reduces the amount of time it takes to load a TOC by creating a binary contents file. This feature does not support custom TOC icons or other customizations such as displaying topics in custom windows.
    • No border Read more… Creates the TOC without adding a border around the TOC pane.
    • Simple border Read more… Adds a single line border around the TOC pane.
    • 3D border Read more… Adds a border around the TOC pane that has a 3D appearance.

Make sure that you associate the skin with the target. See Associating Skins with Targets.
