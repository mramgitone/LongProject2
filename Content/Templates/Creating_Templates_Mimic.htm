Creating Templates

In , templates can be used for creating collections, movies, and skins. A template is simply an existing file of the same type that you are creating. Therefore, creating a template for future use in is just a matter of creating a collection, movie, or skin; providing the settings that you want; and placing it in the proper folder on your computer.

how to create a template

  1. In , create the collection, movie, or skin that you want to use as a template.

    note

    Creating a skin in is a matter of editing the skin within Mimic and then exporting it to another location. See Editing Skins and Exporting Skins.
  2. Add content or settings to the collection, movie, or skin as you normally would.
  3. Save the file(s).

  4. In Windows, find and copy the collection, movie, or skin folders/files that you created.

    note

    If creating collection or movie templates, copy the entire folder structure holding the files. If creating a skin template, only the skin file is necessary.

  5. In your "Documents" folder, create a subfolder and name it My Templates.

    This subfolder will be used to hold all of the different templates that you create in , as well as the templates for all other applications.

    note
  6. In the My Templates folder, create the following subfolders, depending on whether you are storing collections, movies, or skins:
    • For collections, name the subfolder Mimic Projects
    • For movies, name the subfolder Mimic Movies
    • For skins, name the subfolder Mimic Skins
  7. Paste your collection folder in the " Projects" subfolder. Paste your movie folder in the " Movies" subfolder. Paste your skin file in the " Skins" subfolder.

The next time you create a new collection or movie in , or import a skin, you can select My Templates in the dialog to display (and select from) the templates that you've created.
