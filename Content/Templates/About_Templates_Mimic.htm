About Templates

When you create or import some types of files in , you can use templates. A template is simply an existing file of the same type. By choosing a template, you are giving yourself a head start because the template already contains some settings for you.

provides you with some built-in templates (called "factory templates"), so you do not need to worry about coming up with your own. However, you can always create your own templates and use them for certain tasks. See Creating Templates.

Templates are used when you do any of the following.

  • Create Movie Collections Read more… When you create a new movie collection (MIPRJ file), you base your new collection on an existing MIPRJ file. provides you with the following factory collection templates. See Creating a Movie Collection.
    • Empty Read more… If you select this template, the movie collection is created but does not yet contain a movie. As you record movies with the collection open, those movies are automatically added to the collection.
    • One Frame Read more… If you select this template, the collection is created with one movie containing a single blank frame. You can edit this frame and add more frames to the movie. You can also add more movies to the collection.
  • Create Blank Movies Read more… When you create a new blank movie (MIMOV file), you base the new movie on an existing MIMOV file. provides you with a factory template called "One Frame," which creates a movie with a single blank frame. You can edit this frame or add more frames to the movie. See Creating Blank Movies.
  • Import Skins Read more… Skins (MISKN files) are used to display the look and feel of movies generated in . If you select one of these output types, you can generate the output with 's default skin. However, you also have the choice of editing that skin or importing a different MISKN file. When you go through the process of importing a skin, you can select a skin saved in your template folder (see Creating Templates for information about this folder), or you can navigate to a skin file in another location. See Importing Skins.
  • Create User-defined Language Skins Read more… If you plan to display your movie interface in a language other than English, you can use one of the language skins provided by . However, if you want to display the output in a language for which does not supply a language skin, you can create your own language skin. When you do this, you select a -supported language skin on which to base your new skin (preferably a language that is not too far removed from the language that you want to use). This -supported language skin serves as the "template" for you user-defined language skin. See Creating User-Defined Language Skins.
