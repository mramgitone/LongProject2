Applying Styles to Content

You can apply styles from a stylesheet to content in your topics. For regular stylesheets, this means choosing sections in your topics and selecting individual styles from the stylesheet. For table styles, this means inserting a table into a topic and selecting a table stylesheet during that process, or from the Table Properties dialog when editing a table.

how to apply styles from a regular stylesheet

When you insert or create certain content in a topic, the parent tag for that type of element is often automatically applied (e.g., the <li> tag is automatically applied when you create a list bullet item). The following steps show you how to select a particular style for content (perhaps a class of a parent tag, such as "li.indented," or another tag altogether).

  1. Open the topic for which you want to apply styles.

  2. When applying styles to content, you are likely to use one of the following, depending on the type of style you are applying. Most of the time, there are multiple ways to select a style. To make sure these tools are open, do the following.

    styles window pane

    • Ribbon In the Styles section select Style Window.

    • Menu Select View>Style Window.
    • Keyboard Shortcut Press F12

    The Styles window pane opens, showing styles from your stylesheet.

         styles drop-down field

    • Ribbon The styles drop-down field is located in the Styles section. It looks like this: . The current style is displayed in the drop-down—in this example, <p> is the current style.

    • Text Format Toolbar The drop-down in the toolbar looks like this: . The current style is displayed in the drop-down—in this example, <li> is the current style.

         floating style picker

    • Menu Select Format>Pick Style.

    • Keyboard Shortcut Press CTRL+SHIFT+H.

    The Style Picker displays, showing style classes from the stylesheets that are associated with the topic.

         structure bars

    In the local toolbar at the bottom of the XML Editor, click one of the buttons to turn on the structure bars . Depending on the button that you click, structure bars are displayed either to the left of the topic or table content, or above it. See About Structure Bars.

         markers

    In the local toolbar of the XML Editor, click the down arrow in the Show tags button and select Show Markers.

         example

    If you still do not see your style available for selection, try closing and re-launching .

  3. In the topic, do one of the following, depending on the type of content to which you are applying a style.

    characters (i.e., selected text)
    1. Highlight all characters within a paragraph that you want to be affected by the style, without selecting the entire paragraph.
    2. From the Home ribbon, Styles window pane, Floating Style Picker, or Text Format toolbar, select the character style.
    dynamic effects (e.g., drop-down text, expanding text, togglers, popups)
    1. Insert the dynamic effect. For example, if you are working with drop-down text, see Inserting Drop-Down Text .
    2. You can create a class for one of the parent dynamic effect styles in the Stylesheet Editor (e.g., MadCap|dropDownHead.italic). See Creating New Style Classes.
    3. For some dynamic effect styles (e.g., MadCap|dropDownHead.italic), you can place your cursor on the appropriate text in the topic and select the style from the Styles window pane or Text Format toolbar. For other dynamic effect styles (e.g., MadCap|dropDown.myimageicon), you can hover over the appropriate marker in the topic until the cursor is an arrow . Then right-click the marker and select Style Class>[Name of Style].
         footnotes
    1. Insert the footnote into the topic. See Inserting Footnotes.
    2. You can create a class for one of the parent footnote tags in the Stylesheet Editor. See Creating New Style Classes.
    3. For some footnote styles (e.g., MadCap|footnote), you can hover over the appropriate marker in the topic until the cursor is an arrow . Then right-click the marker and select Style Class>[Name of Style].
         forms
    1. Insert the form into the topic. The <form> tag can be selected after you highlight multiple paragraphs in a topic, select the Home ribbon and click , or select the Format menu and choose Group. See Creating Tag Groups.
    2. You can create a class for the <form> tag in the Stylesheet Editor. See Creating New Style Classes.
    3. Right-click the structure bar to the left that is labeled "form" and select Style Class>[Name of Style].
         headings
    1. Click somewhere in the paragraph to be used as the heading. If you highlight the text, make sure you highlight the entire paragraph. Otherwise, you will not be able to select a heading style, but rather a character style only. The exception to this is when you highlight portions of multiple paragraphs at the same time; in that case, you will be able to select a heading style to be applied to all of those paragraphs.
    2. From the Home ribbon, Styles window pane, Floating Style Picker, or Text Format toolbar, select the character style.
         links (e.g., cross-references, text hyperlinks)
    1. Insert the link into the topic.
    2. You can create a class for the link tag in the Stylesheet Editor (e.g., a.greenlink, MadCap|xref.seepage). See Creating New Style Classes.
    5. In the menu, select Style Class>[Name of Style].
         list items
    1. Create the list item in the topic. See About Lists.
    2. You can create a class for the list tag in the Stylesheet Editor (e.g., li.Indented). See Creating New Style Classes.
    3. Click somewhere in the list item. If you highlight the text, make sure you highlight the entire line. Otherwise, you will not be able to select a list style, but rather a character style only. The exception to this is when you highlight portions of multiple list items at the same time; in that case, you will be able to select a list style to be applied to all of those items.
    4. From the Home ribbon, Styles window pane, Floating Style Picker, or Text Format toolbar, select the character style.
    paragraphs
    1. Click somewhere in the paragraph. If you highlight the text, make sure you highlight the entire paragraph. Otherwise, you will not be able to select a paragraph style, but rather a character style only. The exception to this is when you highlight portions of multiple paragraphs at the same time; in that case, you will be able to select a paragraph style to be applied to all of those paragraphs.
    2. From the Home ribbon, Styles window pane, Floating Style Picker, or Text Format toolbar, select the character style.
    reusable content (e.g., variables, proxies)
    1. Insert the reusable content into the topic.
    2. You can create a class for one of the reusable content tags in the Stylesheet Editor (e.g., MadCap|variable.bluetext). See Creating New Style Classes.
    3. Hover over the reusable item in the topic until the cursor is an arrow . Then right-click the item and select Style Class>[Name of Style].
         slideshows
    1. Insert the slideshow into the topic. See Inserting and Navigating in Slideshows.
    2. You can create a class for any of the slideshow styles in the Stylesheet Editor. See Creating New Style Classes.
    3. Right-click the structure bar (either for the slideshow itself or for a slide within it) and select Style Class>[Name of Style]. See About Structure Bars.
         tables
    1. Insert the table into the topic. See Inserting Tables into Topics.
    2. You can create a class for any of the table styles in the Stylesheet Editor (e.g., th.style1, td.mystyle, tr.firsttablerow). See Creating New Style Classes.
    3. For some table styles (e.g., <caption>, <th>), you can click somewhere on the appropriate text in the table and select the style from the Styles window pane, Floating Style Picker, or Text Format toolbar.
    4. For other table styles (e.g., <table>, <tr>), you can right-click the structure bar and select Style Class>[Name of Style]. See About Structure Bars.
         notes

    For another way to select styles for table content, including the ability to use <p> tags in tables, see Selecting Styles for Table Cell Contents.

    You can also create and apply stylesheets specifically for tables. See Applying Table Stylesheets to Tables.
