Setting Style Classes for Topics

You can set style classes for one topic at a time or for multiple topics at once.

how to set style classes for one topic at a time
  1. Find and select the topic.
  2. In the local toolbar of the Content Explorer, click . The Properties dialog opens.
  3. Select the Topic Properties tab.
  4. In the Topic Style Class field, click the down arrow and select a style class from the list.
 how to set style classes for multiple topics at once
