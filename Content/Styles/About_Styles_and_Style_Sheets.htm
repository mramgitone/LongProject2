What are Styles?

Styles are elements that contain formatting settings. You can apply styles to your content to change the way it looks. works with cascading stylesheet (CSS) rules that are specified by the World Wide Web Consortium, or W3C (http://www.w3.org).

If we were to change the color to green in the style, every heading or paragraph in the project that uses that style would change to green immediately.

An important aspect of CSS is that it is based on community-wide standards set by the W3C. This means that CSS can be used for any XML-based tool, not just . Tools such as Microsoft Word and Adobe FrameMaker also use styles, but they are proprietary, which means they can be used only within that application.

Regular Stylesheets and Table Stylesheets

regular stylesheets and table stylesheets

There are two basic kinds of stylesheets in —regular stylesheets and special table stylesheets. Each kind of stylesheet uses a different editor in . See About Regular Stylesheets and About Table Stylesheets.

For the differences between regular stylesheets, table stylesheets, and local properties—and when you should use one over the other—see Regular Stylesheets, Table Stylesheets, or Properties?

Mediums

mediums

A medium is an alternative set of styles in a stylesheet that you use for different outputs. They are intended to be an exception to the default style. Mediums can be used in both regular stylesheets and table stylesheets. See About Stylesheet Mediums.

Skin Styles

skin styles

In addition to regular and table stylesheets, you can use skin styles. These let you control the look of the online output window (e.g., navigation pane, TOC or browse sequence entries, index keywords). See Skin Styles.
