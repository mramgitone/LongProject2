Basics of CSS

To understand how to use styles for maximum benefit in Flare, you should understand the basics of cascading stylesheets (CSS).

W3C and CSS Resources

The vast majority of the styles and style properties in were developed by the World Wide Web Consortium (W3C). For more thorough information about each W3C style and property, refer to http://www.w3.org and use the search feature on the website.

Another way to discover the meaning of a specific property is by using the interface. When you select a property in the Stylesheet Editor, a short description of the property is displayed at the bottom of the editor.
