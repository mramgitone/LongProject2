how to fix undefined styles

There are a few basic ways to fix an undefined style: (1) You can rename the style; (2) you can remove the style from the content; and (3) you can apply a different style class to the affected content.

to rename style classes

  1. In the window pane, select all style class occurrences that you want to rename.

    note

    All style class occurrences that you select will be renamed to the same style that you provide in the following steps.
  2. In the local toolbar, click the Rename button. The Rename Style(s) dialog opens.
  3. In the New style name field, enter a new style class.

  5. A message lets you know how many files were changed. All of the selected style class occurrences are renamed.
