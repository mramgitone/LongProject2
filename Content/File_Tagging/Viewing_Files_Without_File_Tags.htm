Viewing Files Without File Tags

You can

how to view files that do not have file tags
  1. Open a project.

    • Ribbon In the Dynamic Reports section, select More Reports>Files Without File Tags.

    • Menu Select View>More Reports>Files Without File Tags.
    • Analyzer Summary Window Pane Double-click the row that mentions files without file tags.

    The Files Without File Tags window pane opens, listing all content files that do not have file tags associated with them.

    • File Displays the name of the file.
    • Title
    • Folder

  2. You can open a file and associate file tags with it. You can open the file in the interface or in the project.

  3. Once the file is open, select File>Properties. In the Properties dialog, you can use the File Tags tab to associate file tags.
