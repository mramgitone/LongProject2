About File Tagging

You can assign "tags" to files that you are reviewing in , even folders. You can use file tags for many different purposes, such as assigning authors or milestones to topics. lets you generate reports based on the tags that are assigned. This makes project development easier to track, manage, and schedule.

Let's say an author creates tags in in order to keep track of the status of files (e.g., In Progress, Sent for Review, Completed). The author then sends a topic to you for review, with the request that you mark the file as completed after you are finished.

With the topic open, you select File>Properties. This opens the Properties dialog for that file. In that dialog you select the File Tags tab and choose the appropriate option.

You can then return the topic to the author, who adds it to the project.

Steps for Using File Tagging

steps for using file tagging

Following are the basic steps involved with using the file tagging feature in :

  1. Add File Tag Set The first step in using topic tagging is to add a file tag set. Each tag set stores possible tag values and their comments for one tag type (e.g., an "Author" tag set only contains author names, and a "Status" tag set only contains status names). A factory tag set template is available with pre-designed tag types ("Author" and "Status"). However, you can create custom tag types for any purpose you can think of. See Adding File Tag Sets.

    A file tag set has an .flfts extension and is stored in the Project Organizer under the Advanced folder.

  2. Create File Tags After you add a file tag set, you can create multiple tags within it. See Creating File Tags.
  3. Associate Tags with Files Once you have created file tags, you can associate them with the appropriate files. If you associate certain tags with a folder, all of the files in that folder will be associated with those tags. See Associating Tags with Files.
  4. (Optional) Edit File Tag Sets You might need to edit a file tag set from time to time. For example, if you have a file tag set designed to hold author names and you hire a new writer, you need to open the file tag set and add that author's name. You also might decide to change the tag type (e.g., change "Author" to "Contributor"). See Editing File Tag Sets.
  5. Create and Generate Reports At any time you can create and generate a report to see the data for topics associated with any file tag set. To do this, select the appropriate tag set option(s) in the Report Editor. These options are always named "Files With Tag: [Tag Type Name]" (e.g., Files With Tag: Author). See About Reports.

These tags are intended to be assigned to entire files, not to paragraphs or other content within a topic.
