Using the Hand Mode Feature

The hand mode feature in the local toolbar of the Capture Frame Editor lets you drag areas of an image a frame around. This is useful when you have zoomed in on an image a frame so much that you cannot see all of it in the editor.

how to use the hand mode feature
  1. Open an image a frame.

    • Ribbon  In the Editing section, click .

    • Local Toolbar Click .
    • Keyboard Shortcut Hold down the spacebar to activate hand mode. To exit hand mode, release the space bar.
    • Right-Click Right-click anywhere in the Capture Frame Editor, and from the context menu, select Hand Mode.
  3. Click on the image frame and drag your mouse to move the image frame so that you can see hidden portions of it.
