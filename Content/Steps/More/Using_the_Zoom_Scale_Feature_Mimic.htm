Using the Zoom/Scale Feature

The bottom part of the Frame Editor includes a toolbar that lets you use zoom/scale features to better see your frame .

You can use some of the buttons in this toolbar to adjust the display of the frame in the Frame Editor. Please note that the frame is scaled only so that you can more easily see it in the editor. The actual frame size is not adjusted for the output.

These options are also available from the Object ribbon and the View menu.

how to use the zoom/scale feature
