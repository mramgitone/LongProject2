About Paragraph Formatting

You can affect the look and behavior of paragraphs in various ways. These settings can be applied locally or to the style used for the paragraph. Modifying the style is typically preferable to changing the settings locally for a single paragraph.

Following are some of the more common ways that you can format paragraphs. This guide provides step and details for some of the tasks that are especially important for print-based output (i.e., those listed below with page references after them). For more information about the other tasks, including steps, see the online Help or the Styles Guide.

  • Alignment Read more…  See Aligning Paragraphs.
  • Autonumbering Read more… You can apply an autonumber format to paragraphs so that certain content and/or incremented numbering displays with it. This is useful for numbering elements such as chapters, figures, or tables. See About Autonumbers, Creating Autonumber Formats for Styles, and Creating Autonumber Formats for Paragraphs.See Creating Autonumber Formats for Styles.
  • Background Read more…  See Setting the Background for Paragraphs.
  • Borders Read more…   See Setting Borders for Paragraphs.
  • Borders—Rounded Read more… For most output formats Contribution and New Document workflows, you can use CSS3 border-radius style properties to create rounded borders on block elements, such as paragraphs and blockquotes.  See Creating Rounded Borders on Paragraphs and Tables .
  • Breaks Read more… You can apply a page, column, or frame breaks to a paragraph or heading. For example, you might do this if you want the paragraph or heading to start at the beginning of the next page or column. You might apply a frame break if you want to make sure that only certain text, such as a heading, appears in a certain body frame in a page layout. These features are used for print-based output. See Setting Page and Column Breaks and Setting Frame Breaks.
  • Drop Caps/Initial Caps Read more… You can create an effect on a paragraph so that the initial letter is different than the others and drops down to the lines below. See Creating Drop Caps Effects and Creating Initial Caps Effects.See Creating Drop Caps Effects.
  • Hanging Punctuation Read more… You can use styles to create hanging punctuation on block items such as paragraphs in . Hanging punctuation lets you "hang" certain punctuation characters outside the block element, allowing the text to align.  See Creating Hanging Punctuation.
  • Hyphenation Read more…  See Adjusting Hyphenation.
  • Indentation Read more…  See Indenting Paragraphs.
  • Line Spacing Read more…  See Setting Line Spacing.
  • Moving Paragraphs Read more…  See Moving Paragraphs and Other Content with Structure Bars.
  • Next Style Read more… You can specify that a particular style should be used when you press Enter at the end of the current style. For example, after you type text for a heading and press Enter, you might want the next style to be something like p.TopicText, rather than the main <p> tag. See Setting a "Next" Style.
  • Positioning  Read more… You can adjust the positioning of a paragraph. For example, you can float it to the left of the page layout frame. You can do this by applying a position setting on the style used by the paragraph. See Positioning Paragraphs.
  • Short Line Elimination Read more…  See Setting Short Line Elimination.
  • Spacing Above/Below Read more…  See Setting Paragraph Spacing Above and Below.
  • Widow and Orphan Control Read more… You can use widow and orphan control to avoid instances where "leftover" lines from a paragraph are shown at the top or bottom of a page or column. This feature is used for print-based output. See About Widows and Orphans and Setting Widow and Orphan Control. See Setting Widow and Orphan Control.
