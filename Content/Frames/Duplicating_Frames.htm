Duplicating Frames

how to duplicate a frame
  1. In the Frames window pane, select the frame that you want to duplicate.

    • Ribbon In the Insert section, select Duplicate.

    • Menu Select Frame>Duplicate Frame.

    • Frames Toolbar Click .

      note

      If the Frames toolbar is not open, select View>Toolbars>Frames or click in the Frame Editor's local toolbar.

    The new duplicated frame is added below the current frame.
