Removing Images from Frames

how to remove an image from a frame
  1. Open the frame.
  2. In the Frame Editor, double-click the frame. The Frame Properties dialog opens.
  3. Select the Image tab.
  4. Click Remove Image. The image is removed from the preview window in the Frame Properties dialog.

  5. The image is removed from the frame.

    Any objects that were previously added to the frame still remain.
