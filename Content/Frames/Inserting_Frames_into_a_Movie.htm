Inserting Frames into a Movie

how to insert a frame into a movie
  1. In the Frames window pane, select the frame below which you want to insert a new frame.

    • Ribbon In the Insert section, select Duplicate.

    • Menu Select Frame>Frame.

    • Frames Toolbar Click .

      note

      If the Frames toolbar is not open, select View>Toolbars>Frames or click in the Frame Editor's local toolbar.
    • Local Toolbar Click .
    • Right-Click In the Frames window pane, right-click the frame and from the context menu, select Frame.

    The new frame is added after the selected frame

 notes

You can also insert frames by recording or capturing new frames. See Recording New Frames and Capturing New Frames.
