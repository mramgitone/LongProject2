About Frames

A frame is simply an image that serves as one piece of an overall movie. A collection can contain multiple movies. Each movie can be composed of many frames (just like a movie that you watch in a theater). And each frame of a movie can contain multiple objects, effects, and actions, as well as timing and transitions.

Frames in the Interface

frames in the interface

When you record a movie, the frames are created and placed in order automatically. You can add additional frames or remove existing frames as necessary. You can also edit individual frames (e.g., add objects and effects, adjust timing, apply transitions) to enhance the movie. See Enhancing a Movie.

Full-Motion Video Frames

full-motion video frames

From time to time, you may see this on a frame in the Frames window pane:

This icon displays on any frame that is using full-motion video (FMV). This is a mode of video recording that you can use to capture actual movement (e.g., if you want to record a video playing on your computer). FMV also occurs, for example, when you drag your mouse, and optionally if you type text, during a movie recording. This creates a smoother visual in the output. The orange icon will not display in the output; it is merely a way of indicating that a particular frame is using FMV. See Selecting the Recording Mode.

The timing for FMV frames is the same length as the action when you originally captured it. You cannot edit the timing of FMV frames in this version of Mimic.
