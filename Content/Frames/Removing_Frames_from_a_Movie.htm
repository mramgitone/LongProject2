Removing Frames from a Movie

how to remove a frame
  1. In the Frames window pane, select the frame that you want to remove.

    • Ribbon In the Edit section, select Remove.

    • Menu Select Frame>Remove Frame.
    • Keyboard Shortcut Press Delete on the keyboard.
    • Local Toolbar Click .
    • Right-Click Right-click the frame and from the context menu, select Remove Frame.
