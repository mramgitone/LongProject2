Pausing Frames

  • Pause the Action You can add an object (e.g., button) to a frame and apply a "pause" action to it. If the user does not click the object in the output, the movie continues to play without interruption. If the user clicks the object, the movie pauses until the object is clicked again.
  • Pause the Frame Display Time You can disable (i.e., pause) the timing in one frame or in all frames in a movie. This is useful, for example, if you add a button to a frame and want the user to click the button before the movie continues.
how to pause the action in a frame
  1. Open the movie.
  2. Open the frame.
  3. Add the object (e.g., button, shape) that you want to use to pause the action.
  4. Double-click the object. The properties dialog for that object opens.
  5. If the object is a button, select the Button Action tab. Otherwise, select the Action tab.
  6. Select Pause.
 how to pause the display time for a single frame in a movie
  1. Open the movie.
  2. Open the frame.
  3. In the Frame Editor, double-click the frame. The Frame Properties dialog opens.
  4. Select the Transition tab.
  5. Select Pause movie at end of frame.

  7. Now you can add an object to the frame and apply an action to it (e.g., go to next frame) so that the user has a way to continue the movie. See Selecting Actions for Objects.

 how to pause the display time for all frames in a movie
  1. Open the movie.
  3. Select the Transition tab.
  4. Select Pause movie at end of frame.

  6. Now you can add objects to any frames in the movie and apply actions to them (e.g., go to next frame) so that the user has a way to continue the movie. See Selecting Actions for Objects.
