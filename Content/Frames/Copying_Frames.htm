Copying Frames

how to copy a frame
  1. In the Frames window pane, select the frame that you want to copy.

    • Ribbon In the Edit section, select Copy.

    • Menu Select Frame>Copy Frame.

    • Frames Toolbar Click .

      note

      If the Frames toolbar is not open, select View>Toolbars>Frames or click in the Frame Editor's local toolbar.
    • Right-Click In the Frames window pane, right-click the frame you want to copy and from the context menu, select Copy Frame.
    • Keyboard Shortcut Select the frame you want to copy and press CTRL+C on the keyboard.
  3. In the Frames window pane, select the frame below which you want to place the copy.

    • Ribbon In the Edit section, select Paste.
    • Menu Select Frame>Paste Frame.
    • Frames Toolbar Click .
    • Right-Click In the Frames window pane, right-click the frame you want to copy and from the context menu, select Paste Frame.
    • Keyboard Shortcut Select the frame you want to copy and press CTRL+V on the keyboard.

    The new copied frame is added after the selected frame.
