Creating Master Frames

how to create a master frame
  1. Create a frame the way you normally would (e.g., inserting a blank frame and adding objects to it).

  2. Arrange the frame is just the way you want it.

    • Ribbon In the Reuse section, select Add to Master Frames.

    • Menu Select Frame>Add Frame to Master Frames.
    • Right-Click Right-click the frame and from the context menu, select Add Frame to Master Frames.

    The frame is added to the Master Frames window pane and it opens in the Master Frame Editor, which is nearly identical to the regular Frame Editor, except that it has a blue background. You can use this editor to make any changes to the master frame.

 note

Master frames do not support image effects (e.g., zoom, blur).

After you create a master frame, you can apply it to individual frames or entire movies. See Applying Master Frames.
