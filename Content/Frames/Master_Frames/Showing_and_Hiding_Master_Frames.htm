Showing and Hiding Master Frames

You can show or hide different master frame objects in the active frame. See About Master Frames and Applying Master Frames.

how to show and hide master frames
  1. Open a frame that is using a master frame.
  2. Click the master frames drop-down field at the bottom-left of the Frame Editor and choose one of the following.
    • Show All Master Frames Select this if you want both kinds of master frame objects to be shown—those on the top level as well as the bottom level.
    • Show Bottom Layer Master Frame Select this if you want only bottom-level master frame objects to be shown.
    • Show Top Layer Master Frame Select this if you want only top-level master frame objects to be shown.
    • Hide Master Frames Select this if you want to hide all master frame objects.
