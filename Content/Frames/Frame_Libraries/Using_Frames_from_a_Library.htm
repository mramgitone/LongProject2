Using Frames from a Library

how to use a frame from a library
  1. Open a movie.
  2. Open a frame library.
    • In the Frame Libraries window pane, click on a frame that you want to add. Then in the local toolbar, click .
    • Right-click on the frame and select Add Frame to Movie.

    The frame is added as the last frame of the movie. If you want to move the frame, open the Frames window pane and drag the frame up as high as you need it.
