Creating Frame Libraries

how to create a frame library

    • Ribbon In the Tool Windows section, select Frame Libraries.

    • Menu Select View>Frame Libraries.
    • Standard Toolbar Click .

    The Frame Libraries window pane opens.

  2. In the local toolbar of the Frame Libraries window pane, click . The New Library dialog opens.
  3. Enter a name for the new library.
  4. The new empty library opens (by default on the left side of the user interface).

After you create a frame library, you can add frames to it. See Adding Frames to a Library.
