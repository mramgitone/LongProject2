Deleting Frames from a Library

Use the following steps to remove a frame from a library.

how to delete a frame from a library
  1. Open the frame library containing the frame that you want to delete.
  2. In the Frame Libraries window pane, do one of the following.
    • Right-Click Right-click the frame and select Remove Frame.
    • Keyboard Shortcut Select the frame and press Delete on your keyboard.
