Capturing an Application

Use this method to capture an open application. This method is similar to capturing the active window. The difference is that this method displays a list of all open applications, letting you select the one that you want to capture. This capture method is only available through the menu interface or hot key options.

how to capture an application
  1. (Optional)

    • Ribbon In the New Screen Capture Options section, click the down arrow next to the Current Profile field.

    • Standard Toolbar Click the down arrow next to the Current Profile field.

  2. Select Capture>Capture Application. The Select window opens.

    The Capture Application feature can also be set to a keyboard shortcut. The factory default is "none". To set this feature to a keyboard shortcut, select Tools>Configure Hot Keys or select the Hot Keys tab in the Options Dialog. See Changing Hot Keys.

    • Select an application in the list and click OK.
    • Double-click an application in the list.

    The image is captured and opens in the Capture Editor.

note

If you are working on a project in MadCap Flare and want to capture and insert an image, you can initiate an Application capture from within Flare by selecting Insert>Screen Capture.
