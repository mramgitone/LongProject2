Capturing a Window With Autoscroll

Use this method to capture an open window that is too long to be displayed in its entirety on the screen. This method grabs an image of the window, automatically scrolling to the bottom of the window for you. The image starts from the top edge of the window that can be seen on the screen and scrolls to the bottom to capture the image.

How to capture a window with autoscrollhow to capture a window with autoscroll
  1. (Optional)

    • Ribbon In the New Screen Capture Options section, click the down arrow next to the Current Profile field.

    • Standard Toolbar Click the down arrow next to the Current Profile field.

  2. Open the window that you want to capture. When you initiate this method of capture, the Capture window will be minimized, allowing you to "grab" an image of whatever is directly behind it.

  3. Open Capture.
    • Ribbon Select File>New Screen Capture>Capture Window with Autoscroll.
    • Menu Select Capture>Capture Window with Autoscroll.

    Capture minimizes and a red border surrounds each window as you move the cursor over it.

  5. Click the window that you want to capture. The image is captured and opens in the Capture Editor.
