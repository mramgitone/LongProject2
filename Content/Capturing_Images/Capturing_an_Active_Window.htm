Capturing an Active Window

Use this method to automatically take a picture of the application window that is active at that moment. This method is similar to capturing an application, which lets you select an application from a list. However, with this method, you do not select an application from a list. Instead, whichever window is currently active is automatically captured. Therefore, with this method, it's best to first open the application window that you want to capture and then press the appropriate keyboard shortcut (CTRL+SHIFT+A), rather than using the ribbon or menu option in the Capture interface.

how to capture and active window
  1. (Optional)

    • Ribbon In the New Screen Capture Options section, click the down arrow next to the Current Profile field.

    • Standard Toolbar Click the down arrow next to the Current Profile field.
  2. Open the application window that you want to capture.
  3. Press CTRL+SHIFT+A on your keyboard. The image opens in the Capture Editor.
