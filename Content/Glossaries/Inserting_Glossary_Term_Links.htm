Inserting Glossary Term Links

If you have words and phrases in topics and snippets that are to be included in a glossary, you can turn them into links. A glossary term link lets users access definitions from that location in the output. Glossary term links can display definitions in popups, hyperlinks, or expanding text.

You can create glossary terms and definitions while inserting glossary term links at the same time. Alternatively, you can first add the term and definition to a glossary file and then insert the link.

 notes

You can also configure a target to convert existing glossary terms to links automatically. For more information see Associating Glossaries with Targets.

You can insert glossary terms into topics using the <glossterm> XML element. If you add glossary entries in this manner, you can still edit their properties in the Glossary Editor.
