Viewing Active Users

how to view active users

  2. Select View>Active Users.

    The active users are listed in the bottom portion of the window pane. The user name and email is always required, so you will always see that information. However, there is additional information (e.g., name, zip code, gender) that you can also include as part of the registration process. This includes the ability to determine which fields are required for your end users and which fields are optional. You can do all of this customization within the skin in the Flare project. For more information see the Flare online Help.
