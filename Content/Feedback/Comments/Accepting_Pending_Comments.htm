Accepting Pending Comments

You can accept comments in the notification email you receive or in the Feedback Explorer interface.

how to accept pending comments in email
  1. In the email click the Accept link.
  2. In the message that opens click Yes.
 how to accept pending comments in feedback explorer
  2. Open the Comments window pane or Topic View window pane (Comments tab).
    • To open the Comments window pane, select View>Comments.
    • To open the Topic View window pane, open the Topics window pane (select View>Topics). Then double-click the appropriate topic. Select the Comments tab.
    note

  4. (Optional in Comments window pane) Initially all comments are displayed. However, you can use the filter drop-down in the local toolbar of the Comments window pane to narrow which types of comments are shown (e.g., Accepted, Pending, Hidden).

  5. Select a comment.
  6. In the local toolbar, click the down arrow in the Set comment status field and select Accepted.
  7. Click Yes. When you or any users open the online output, that comment will be displayed.
 notes

If you reply to new comments from an email notification that you receive, you do not need to go through the process of accepting that comment, or your own comment for that matter. When you reply to a comment from the email link, not only is a reply created, but the original comment and your reply comment are approved at the same time. See Replying to Comments.

If you want all new comments to automatically be approved and immediately displayed in the online output (rather than requiring approval), see Specifying Whether New Comments Need to be Reviewed.
