Hiding Comments

You can hide comments via the notification email you receive or via the Feedback Explorer interface. In the interface you can hide comments manually or automatically (i.e., after a specified number of days). You might use the automatic method, for example, if you frequently make updates to your output and all comments associated with the previous version therefore become outdated.

how to hide comments in email
  1. In the email click the Hide link.
  2. In the message that opens click Yes.
 how to hide comments in feedback explorer (manual method)

  2. Open the Comments window pane or Topic View window pane (Comments tab).
    • To open the Comments window pane, select View>Comments.
    • To open the Topic View window pane, open the Topics window pane (select View>Topics). Then double-click the appropriate topic. Select the Comments tab.
    note

  4. (Optional) Initially all comments are displayed. However, you can use the filter drop-down in the local toolbar of the Comments window pane to narrow which types of comments are shown (e.g., Accepted, Pending, Hidden).

  5. Select a comment.
  6. In the local toolbar, click the down arrow in the Set comment status field and select Hidden.
  7. Click Yes. The comment is not visible to any end users, but you can continue to view it in Feedback Explorer.
 how to hide comments in feedback explorer (automatic method)

  2. In the Feedback Admin, select the General tab.
  3. Click the check box labeled Hide comments after the following number of days so that it contains a check mark.

  4. In the field below enter a number of days.
