how to edit comments

  4. (Optional) Initially all comments are displayed. However, you can use the filter drop-down in the local toolbar to narrow which types of comments are shown (e.g., Accepted, Pending, Hidden).

    • Double-click a comment.

      OR

    • Select a comment from the list and click in the local toolbar.

      OR

    • Right-click a comment and from the context menu select Edit Comment.
  6. In the Edit Comment dialog you can change the subject or body text of the comment. You can also select a different status.
  7. Click Save.
 note

You may need to click the refresh button in the output's comment area in order to see the changes.
