Discarding Comments

You can discard comments in the notification email you receive or in the Feedback Explorer interface.

how to discard comments in email
  1. In the email click the Discard link.
  2. In the message that opens click Yes.
 how to discard comments in feedback explorer

  2. Open the Comments window pane or Topic View window pane (Comments tab).
    • To open the Comments window pane, select View>Comments.
    • To open the Topic View window pane, open the Topics window pane (select View>Topics). Then double-click the appropriate topic. Select the Comments tab.
    note

  4. (Optional) Initially all comments are displayed. However, you can use the filter drop-down in the local toolbar of the Comments window pane to narrow which types of comments are shown (e.g., Accepted, Pending, Hidden).

  5. Select a comment.
  6. In the local toolbar, click the Discard selected comments button .
  7. Click Yes. The comment is removed from the system.
