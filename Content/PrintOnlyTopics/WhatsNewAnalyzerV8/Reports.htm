Reports and Window Panes

A few new reports and window panes have been added to .

File Word Count (Report)

This report is found in the Content Files and Topics categories in the Report Editor.

It shows all of the content files in the project and the word count in each.

Files Without File Tags (Report and Window Pane)

This report is found in the Content Files and File Tags categories in the Report Editor.

It shows all of the content files in the project that do not have file tags.

From View>More Reports, you can also open the Files Without File Tags window pane to view the files in a grid.

Topics With Meta Tags (Report)

This report is found in the Topics category in the Report Editor.

It shows all of the topics in the project that have meta tags, and the attributes in each.

Topics Without Concepts (Report and Window Pane)

This report is found in the Concepts, Content Files, and Topics categories in the Report Editor.

It shows all of the content files in the project that do not have concept keywords.

From View>More Reports, you can also open the Topics Without Concepts window pane to view the files in a grid.
