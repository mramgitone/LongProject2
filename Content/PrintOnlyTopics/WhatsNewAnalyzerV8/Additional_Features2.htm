Accessibility—Empty Alternate Text

There have been a couple of changes affecting how you can apply empty alt text.

First, you can specify globally that future images should all have empty alternate text.

See Applying Empty Alt Text.

In addition to images, this also works for QR codes and equations, since those elements are converted to images in the output.

Second, when you view accessibility suggestions, you have the ability to set alt text on images that do not have it. Starting with this version, you can use the Apply Accessibility Suggestions Wizard to apply empty alt text to one or more images that you select. See Viewing Accessibility Suggestions and Resolving Accessibility Issues.
