Find and Replace

The Find and Replace feature has been enhanced. This involves a redesign of the user interface, a change in the way looks for and replaces content globally, whole word search, and the ability to look throughout a whole package (if you have a review package open). See About Find and Replace.

In addition, two file widgets have been integrated that allow you to search within the active file.

From the ribbons and menus, you can select the following features: Quick Find, Quick Replace, and Find and Replace in Files.

Quick Find and Replace Widgets

Click to select any of the following filter options:

  • Regular Expressions If you select this option, you can use regular expressions when you look for content.
