Opening a Configured Feedback File

Following are steps for opening existing Feedback files to view user activity or work in them. For information about creating the initial Feedback file, see Creating a Configured Feedback File.

 

how to open a configured feedback file

    • Select File>Open.

      OR

    • Press CTRL+O on your keyboard.

  2. Locate and double-click the Feedback file (.fbconf extension) that you want to open.

 notes

As an alternative to the above steps, you can open a Feedback file (.fbconf extension) by dragging it from Windows to the application. For example, let's say you created a Feedback file and named it MyFlareProject.fbconf. You can open the Documents\My Feedback Admin folder on your computer. Then you can launch and drag the MyFlareProject.fbconf file from the My Feedback Admin folder, dropping it on the title bar in the interface.
