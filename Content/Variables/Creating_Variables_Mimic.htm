Creating Variables

In MadCap , you can create variables for collections (in which case the variables are available to all movies within that collection) or for single movies (in which case the variables are available only for that movie).

how to create a variable for a collection
  1. Open the collection. The Collection Editor opens.
  2. Select the Variables tab.

  3. To enter a new name, definition, or comment for a new variable, click in the appropriate cell. Then press F2 on your keyboard and type the name, definition, or comment. Press Enter after you finish typing. As soon as you begin typing in one of the cells, a new blank row is added in the editor (to be used for the next variable that you create).

    Adding a comment is optional and for your internal use only.

how to create a variable for a single movie
  1. Open the movie.
  3. Select the Variables tab.

  4. To enter a new name, definition, or comment for a new variable, click in the appropriate cell. Then press F2 on your keyboard and type the name, definition, or comment. Press Enter after you finish typing. As soon as you begin typing in one of the cells, a new blank row is added in the tab (to be used for the next variable that you create).

    Adding a comment is optional and for your internal use only.

note

In addition to adding a new variable in MadCap , you can also link to existing variables in a MadCap Flare project. See Linking to Flare Projects.

After you create a variable, you can insert it into an object that has been added to a movie frame. See Inserting Variables into Objects.
