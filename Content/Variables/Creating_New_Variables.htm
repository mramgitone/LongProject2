Creating New Variables

Variables are pre-set terms or content that you can use in your project over and over. New variables are created (or added to the project) using the Variable Set Editor.

The following steps show you how to create a new variable within a variable set.

The following steps show you how to create a new variable within a variable set.

how to create a new variable

  2. Double-click the Variables folder.

  3. Double-click a variable set, such as the initial one provided by . If you do not have any variable sets in the folder, you can easily add one using the File ribbon. The Variable Set Editor opens to the right, with the variables page shown. If you use the "MyVariables" template, provides you with two variables (CompanyName and PhoneNumber) to get you started. You can delete these variables or modify them to suit your needs.

  4. To create an additional variable, click in the local toolbar. A new variable row is added, with a temporary name for the variable.
  5. To enter a new name, definition, or comment for a new variable (or for the variables already provided by ), do one of the following.

    • Double-click in a field and type the name, definition, or comment.

      OR

    • Click once in a field and press F2 on your keyboard. Then type the name, definition, or comment.
