Editing Date/Time Variables

You can edit the format for a custom variable to change how the system date and time are displayed. For example, you might want to use different date/time variables for audiences in different time zones. You can associate multiple definitions with the same variable and use the appropriate one wherever necessary.

  3. Double-click the custom variable set (such as MyVariables) that contains the variable you want to modify. The Variable Set Editor opens to the right, with the variables page shown.
  4. Click in the Definition cell for a date/time variable. The Edit Format dialog opens.

  5. You can make a word part of the definition. For example, you might want the word Date with a colon and a space before a date string (e.g., Date: October 25, 2014). The issue is that certain characters are translated automatically as specifiers. So just typing the word Date wouldn't work. The easiest solution is to put the non-specifier characters within quotation marks. Therefore, you might enter "Date:" MMMM dd, yyyy.
