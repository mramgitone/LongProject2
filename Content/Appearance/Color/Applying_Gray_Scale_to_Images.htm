Applying Gray Scale to Images

how to apply gray scale to an image using the file properties dialog
  1. Capture or open the image.
  2. In the Capture Editor, double-click the image. The File Properties dialog opens.

  3. Select the Format tab or Flare Print Format tab. The Format tab is for online output, and the other for print-based output.

    more about these two tabs
  4. Select Gray Scale so that a check mark displays in the box.
 how to apply gray scale to an image using the profiles editor
  1. Open the profile.

  2. In the Profiles Editor, select the Format tab or Flare Print Format tab. The Format tab is for online output, and the other for print-based output.

    more about these two tabs
  3. Select Gray Scale so that a check mark displays in the box.
 note

You can also add a gray scale effect around an object in an image. Doing this will display the image in gray, except for the area inside the object, which remains displayed in color. For more information see Adding Gray Scale Effects.
