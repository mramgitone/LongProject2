Setting the Rectangle Properties for an Object

how to set the rectangle properties for an object
  1. Open an image that has a shape or effect added to it.
  2. Open a frame that has a shape or effect added to it.
  3. Double-click the object. The properties dialog for the object opens.
  4. Select the Rectangle tab, and adjust the settings in the following fields.
    • X Position Set the position of the object on the "X" axis (left and right) on the object. You can also drag the object on the image frame and this number will change automatically.
    • Y Position Set the position of the object on the "Y" axis (up and down) on the object. You can also drag the object on the image frame and this number will change automatically.
    • Width Set the width of the rectangular edges of the object. You can also drag the object on the image frame and this number will change automatically.
    • Height Set the height of the rectangular edges of the object. You can also drag the object on the image frame and this number will change automatically.
    • Corner Radius Set the amount of curve for the rectangular edges of the object. The higher the number, the more curve will be applied to the edges.
