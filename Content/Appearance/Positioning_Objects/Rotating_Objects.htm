Rotating Objects

You can rotate objects after adding them to images frames. Objects can be rotated clockwise or counter clockwise—1 degree, 45 degrees, or 90 degrees at a time.

 how to rotate an object
  1. Open an image to which you have added objects.
  2. Open a frame to which you have added objects.
  3. Click the object that you want to rotate.

    • Ribbon In the Rotation section, select one of the following.

      Click the face of the button to rotate the selected object 45 degrees clockwise. Click the down arrow to select 90, 45, or 1 degree clockwise rotations. See step 5 below.

      Click the face of the button to rotate the selected object 45 degrees counter-clockwise. Click the down arrow to select 90, 45, or 1 degree counter-clockwise rotations. See step 5 below.

      Returns the object to its original setting before it was rotated.

    • Menu Select Object>Rotate.
    • Format Toolbar Click . (You can open the Format toolbar by selecting View>Toolbars>Format or by clicking in the Frame Editor's local toolbar.)
    • Keyboard Shortcut Click on an object and press CTRL+Left Arrow or Right Arrow on your keyboard to rotate an object to the left or right one degree at a time.
    • Right-Click Right-click on the object and select Rotate.

  5. Depending on the option you choose, you can select one of the following rotation options.
