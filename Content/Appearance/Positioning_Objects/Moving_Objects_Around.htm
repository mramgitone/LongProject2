Moving Objects Around

After an object is added, you can move it around the image frame to place it just where you need it. You can do this by dragging the object.

how to move an object around
  1. Open an image to which you have added objects.
  2. Open a frame to which you have added objects.
  3. Hover over the object until the cursor becomes an arrow .

  4. Click on the object and drag it to the appropriate location on the image frame.

    If you have dragged the object outside the current boundaries of the image, padding is automatically added to compensate for the space needed. Double-click the image (not the object) to open the File Properties dialog. Then, on the Appearance tab, use the fields in the Background section as necessary to meet your needs.

 notes
