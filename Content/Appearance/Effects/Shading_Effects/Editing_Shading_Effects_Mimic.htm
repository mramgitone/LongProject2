Editing Shading Effects

After you add a shading effect to a frame, you can edit the effect in the following ways.

  • Modify the Amount of Shading Applied You can do this for a single frame by using the Frame Properties dialog. You can use the Movie Properties dialog to modify the amount of shading for all frames in a movie.
  • Remove the Shading Effect from a Single Frame
how to modify the amount of shading applied to a single frame
  1. Open the frame.
  2. In the Frame Editor, double-click the frame (not the shape).
  3. In the Frame Properties dialog, select the Image Effects tab.

  4. Change the number in the Shade Factor field. The higher the number, the greater the amount of shading. The shade effect will be seen if you have enabled the effect for an object in the properties dialog for an object.

    note

    The shade color is based on the background color from the Appearance tab.
  5. (Optional) Click Same as movie if you want to use the same setting that is entered for the entire movie (in the Movie Properties dialog).
 how to modify the amount of shading applied to all frames in a movie
  1. Open the movie.
  3. In the Movie Properties dialog, select the Image Effects tab.

  4. Change the number in the Shade Factor field. The higher the number, the greater the amount of shading. The shade effect will be seen if you have enabled the effect for an object in the properties dialog for an object.

    note

    The shade color is based on the background color from the Appearance tab.
note

Doing this does not automatically apply shading to all frames in the movie. It merely determines how much shading is used when you add a shading effect to an object in the frame.

 how to remove the shading effect
  1. Open the frame.
  2. In the Frame Editor, double-click the object (not the frame).
  3. In the properties dialog for that object, select the Image Effects tab.
  4. Click in the Enable Shade Effect field to remove the check mark.
