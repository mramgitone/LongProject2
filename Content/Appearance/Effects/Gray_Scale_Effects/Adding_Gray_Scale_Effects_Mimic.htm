Adding Gray Scale Effects

example

Here is an example without the gray scale effect.

In the following example, we've drawn a rectangle with a border around the top part of Simon's propeller hat. Then we applied a gray scale effect to the rest of the image around this rectangle. Notice that the bottom portion of that hat (as well as the rest of the image) is gray.

You can quickly add a gray scale effect with a rectangle shape by using the "Gray Scale Effect Mode." Otherwise, you can use a slightly longer process and add a gray scale effect for any shape.

how to use the gray scale effect mode for a rectangle
  1. Open a frame.

    • Ribbon  In the Editing section, click .

    • Menu Select Insert>Effect>Gray Scale Effect Mode.
    • Local Toolbar Do the following.
      1. In the local toolbar, click the arrow next to the Effect button .
      2. From the drop-down list, select Gray Scale Effect Mode.

    The cursor changes to small crosshairs.

  3. Click in the frame and drag the mouse to draw the rectangle over the area of the frame that you want to remain displayed in the original color. Release the mouse button when you are finished. The rectangle appears on the frame and the area around it is displayed in gray.

  4. If necessary, you can make modifications to the effect, including the following.

    modify the look of the rectangle
    1. Double-click the shape (not the frame).
    2. In the properties dialog for the object, use the various tabs to make changes to the rectangle. To remove the border around the rectangle, select the Appearance tab and set the Line:Width field to 0.
         move or resize the rectangle
    • To move the entire rectangle, click somewhere in the middle of the rectangle and drag it to the new location.
    • To resize the rectangle, click any of the points (i.e., small circles) around the edge of the rectangle and drag it to adjust the height and/or width.
how to add a gray scale effect for any shape
  1. Open a frame.

  2. In the local toolbar of the Frame Editor, click the appropriate shape button.

  3. Click in the frame and draw the shape over the area of the frame that you want to remain displayed in the original color.

    For more information about how to draw each shape, see page Adding Shapes to Frames.

  4. Double-click the shape (not the frame). The properties dialog for the object opens.

  5. Select the Image Effects tab.
  6. Click the check box next to Enable Gray Scale Effect.

  7. The area around the shape is displayed in gray.

  8. If necessary, you can make modifications to the effect, including the following.

    modify the look of the shape
    1. Double-click the shape (not the frame).
    2. In the properties dialog for the object, use the various tabs to make changes to the shape.
    move or resize the shape
    • To move the entire shape, click somewhere in the middle of the shape and drag it to the new location.
    • To resize the shape, click any of the points (i.e., small circles) around the edge of the shape and drag it to adjust the height and/or width.
