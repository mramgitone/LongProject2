Editing Blur Effects

After you add a blur effect to a frame, you can edit the effect in the following ways.

  • Modify the Amount of Blur Applied You can do this for a single frame by using the Frame Properties dialog. You can use the Movie Properties dialog to modify the amount of blur for all frames in a movie.
  • Remove the Blur Effect from a Single Frame
how to modify the amount of blur applied to a single frame
  1. Open the frame.
  2. In the Frame Editor, double-click the frame (not the shape).
  3. In the Frame Properties dialog, select the Image Effects tab.
  4. Change the number in the Blur Factor field. The higher the number, the more the frame is blurred. The blur effect will be seen if you have enabled the effect for an object in the properties dialog for an object.
  5. (Optional) Click Same as movie if you want to use the same setting that is entered for the entire movie (in the Movie Properties dialog).
 how to modify the amount of blur applied to all frames in a movie
  1. Open the movie.
  3. In the Movie Properties dialog, select the Image Effects tab.
  4. Change the number in the Blur Factor field. The higher the number, the more the frame is blurred. The blur effect will be seen if you have enabled the effect for an object in the properties dialog for an object.
note

Doing this does not automatically apply a blur to all frames in the movie. It merely determines how much blur is used when you add a blur effect to an object in a frame.

 how to remove the blur effect
  1. Open the frame.
  2. In the Frame Editor, double-click the object (not the frame).
  3. In the properties dialog for that shape, select the Image Effects tab.
  4. Click in the Enable Blur Effect field to remove the check mark.
 note

If you want to edit the blur effect within a shape, instead of around it, see Editing Blur-Inside Effects.
