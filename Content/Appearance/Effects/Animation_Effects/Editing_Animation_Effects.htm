Editing Animation Effects

After you add an animation effect to an object, you can edit it (e.g., change the type of animation effect and its properties).

how to edit animation effects for an object
  1. Open the frame.
  2. In the Frame Editor, double-click the object that you want to edit. The Properties dialog for that object opens.
  3. Select the Animation Effects tab.
  4. In the Animation section, make a selection.
    • Move The object moves along a defined path. In order to use this animation effect, you must set a trajectory path for the object. If you set the trajectory first, this effect is automatically applied to the object.
    • Shake The object shakes back and forth when it is in view.
    • Effect Properties Opens the Effect Properties dialog, which lets you edit settings for the selected animation effect.
      • Horizontal Distance Set the distance (in pixels) that the effect moves on the X-axis (left and right) on the frame.
      • Vertical Distance Set the distance (in pixels) that the effect moves on the Y-axis (up and down) on the frame.
      • Duration Enter the number of seconds that you want the effect to be displayed.
      note

      These fields are shown only if you are changing the "Shake" animation effect. For the "Move" animation effect, these properties are controlled by the trajectory associated with the object.
