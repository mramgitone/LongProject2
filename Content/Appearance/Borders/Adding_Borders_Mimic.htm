Adding Borders

how to add a border to a single frame
  1. Open the frame.
  2. In the Frame Editor, double-click the frame. The Frame Properties dialog opens.
  3. Select the Appearance tab.
  4. In the Border section, select the type of border to add (e.g., solid, double, dashed).
  5. In the Color field, click the down arrow and select a color for the border.
  6. The border is added to the frame.
 how to add a border to all frames in a movie
  1. Open the movie.
  3. Select the Appearance tab.
  4. In the Border section, select the type of border to add (e.g., solid, double, dashed).
  5. In the Width field, enter the size of the border (in pixels).
  6. In the Color field, click the down arrow and select a color for the border.
  7. The border is added to all frames in the movie.
 how to add a border to an object
  1. Double-click the object in a frame (not the frame itself).
  2. In the properties dialog for the object, select the Appearance tab.
  3. In the Color field, click the down arrow to select a color for the border. To see advanced color options, select More colors.
  4. In the Width field, enter the size of the border (in pixels).
  5. The border is added to the object.
 note

You can change the border width for entire movies only, not for individual frames.
