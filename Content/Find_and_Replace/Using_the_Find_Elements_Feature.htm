Using the Find Elements Feature

Follow the steps below if you want to find files containing particular elements—custom tags, MadCap-specific features (e.g., drop-down text, concepts, equations), markup, inline styles, and style classes. You have the flexibility to look in all files in the project or only in the open documents. The results of the search are displayed in either the Find Results 1 window pane or in the Find Results 2 window pane, depending on your selection.

how to use the find elements feature
  1. Open at least one document.

    • Ribbon In the Editing section, select Find Elements.

    • Menu Select Edit>Find Elements.
    • Keyboard Shortcut Press CTRL+E.

    The Find Elements window pane opens.

  3. In the Find what field, select one of the available options from the drop-down.

    • Custom Tags Read more… When you select this option you can click in the "Search for" field and enter text. This lets you search for any style tags, whether they are standard HTML tags (such as "img" or "table") or tags for MadCap-specific elements (such as "MadCap:annotation" or "MadCap:concept"). Style classes searches through the class attribute for all elements. If you leave the "Search for" field blank, looks for all tags.
    • MadCap Read more… When you select this option you can click in the "Search for" field and select from dozens of elements specific to MadCap and , such as drop-down text, Help control links, concepts, proxies, equations, QR codes, and much more.
    • Markup Read more… When you select this option you can click in the "Search for" field and select from dozens of standard HTML style tags, such as <a>, <div>, <h1>, <ol>, <span>, and much more.
    • Inline Styles Read more… When you select this option you can click in the "Search for" field and enter text. This lets you search for elements with locally applied styles such as color or font. If you leave the "Search for" field blank, looks for all elements with inline styles.
    • Style Classes Read more… When you select this option you can click in the "Search for" field and enter text. This lets you search through the class attribute for all elements. If you leave the "Search for" field blank, looks for all elements that have a style class applied to them.
  4. Click in the Find in field and select one of the following.
    • Whole Project Read more… looks for the text in every document in the project.
    • All Open Documents Read more… looks for the text in all documents currently open in the interface.
  5. Click in the Search for field and either make a selection from the drop-down or enter text in the field. The kind of action you can perform in this field depends on what you already selected in the "Find what" field above.
  6. Click one of the output radio buttons.
    • Output in Find Results 1 Read more… will display the search results (list of files) in the Find Results 1 window pane.
    • Output in Find Results 2 Read more… will display the search results (list of files) in the Find Results 2 window pane. This is useful if you have already performed a search using the Find Results 1 window pane and want to keep those results open while you do another search.
  7. Click the Find All button. The search results are displayed in the window pane that you specified.
