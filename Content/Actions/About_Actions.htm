About Actions

An action is a command that you can assign to an object to affect what happens next in the movie.

Types of Actions

types of actions

You can apply regular "click" actions to any object. You can also assign an end action to a cursor object:

  • Click Actions After you add an object (such as a button) to a frame, you can assign a regular action to it (e.g., go to next section, go to previous frame, pause movie). When the user clicks (or double-clicks) the object in the output, that action will be performed. See Selecting Actions for Objects.
  • End Actions If you have cursor shapes in the frames of your movie (either added manually or automatically during the recording), what happens in the movie when a cursor movement reaches the end of its path (trajectory)? Unless you specify otherwise, the movie will simply continue to play in its normal progression. It is likely that this is often what you will want to happen. However, there may be times when you want to provide a specific action to take place when the cursor reaches the end of its path. See Selecting an End Action for a Cursor. For more details, see the online Help.

Action Definitions

action definitions

Following are explanations of the many actions that you can use:

  • GotoFrame Read more… Select this option to open a specific frame in the movie when the user clicks the button or other object. When you select this option, the Frame Name field appears at the bottom of the tab. Enter the number or name of the destination frame in this field. (The name of the frame is identified above it in the Frames window pane. You can change the name in the Frame Properties dialog.)
  • GotoFirst  Read more… Select this option to open the first frame in the movie when the user clicks the button or other object.
  • GotoNext  Read more… Select this option to open the next frame in the movie when the user clicks the button or other object.
  • GotoPrevious  Read more… Select this option to open the previous frame in the movie when the user clicks the button or other object.
  • StartCursor  Read more… Select this option to start the cursor movement (if the frame has one) when the user clicks the button or other object. example

    example

    This option would be useful if you have a frame containing a cursor trajectory, and you want to give users a choice of performing a task (e.g., clicking an area in your movie) or watching as the task is performed for them (watching the cursor move along its trajectory and click the area). In a situation such as this, you might do the following:

    1. Add a rectangle shape and place it around the area that you want the user to click. Then double-click that rectangle and select an action for it, such as "GotoNext." See Adding Shapes to Frames and Selecting Actions for Objects.
    2. Add a callout, pointing it to that rectangle area. Add text to the callout, providing instructions for the user (e.g., "Click this area. Otherwise, click the button below to see it done."). See Adding Callouts and Adding Text.
    3. Add a button. Give it an appropriate label (e.g., "Do it for me"). Then apply the correction action to the button. To do this, double-click the button and select the StartCursor action for it. See Adding Buttons.
    4. Select an end action for the cursor. Double-click the cursor on the frame, and select an action (e.g., "GotoNext") on the End Action tab. This is needed in case the user opts to click the "Do it for me" button. See Selecting an End Action for a Cursor.
    5. Pause the timing of the frame. To do this, double-click the frame and select "Pause movie at end of frame" on the Transition tab. This will ensure that the frame waits for the user to perform an action (either clicking in the rectangle area or clicking the button) before moving on. See Pausing Frames.
  • GotoFirstSection  Read more… Select this option to open the first movie in the collection when the user clicks the button or other object. This option assumes you have created a collection and added multiple movies to it.
  • GotoPreviousSection  Read more… Select this option to open the previous movie in the collection when the user clicks the button or other object. This option assumes you have created a collection and added multiple movies to it.
  • GotoNextSection  Read more… Select this option to open the next movie in the collection when the user clicks the button or other object. This option assumes you have created a collection and added multiple movies to it.

  • GotoSection  Read more… Select this option to open a specific movie in the collection when the user clicks the button or other object. When you select this option, the Section Name field appears at the bottom of the tab. Select the name of the destination movie from the drop-down list in this field. This option assumes you have created a collection and added multiple movies to it.

    notes

    If you use this action and are generating Flash output, you might encounter problems with movies correctly jumping from one movie to the next. This is due to Flash-related security issues.
  • Pause  Read more… Select this option to pause the movie when the user clicks the button or other object. When the user clicks the button or other object again, the movie will continue.
  • Accept  Read more… Select this option to accept the user's input. Let's say you add an input box to a frame. After users complete the input box, they can usually simply press Enter on the keyboard. However, if you want users to click say, a button, when they are finished with the input, you can create a button and apply this action to it. After users finish typing the input, they can click that button to submit the input (instead of pressing Enter on the keyboard).
  • Open  Read more… Select this option to open a remote file (such as a website) when the user clicks the button or other object. When you select this option, the URL field appears at the bottom of the tab. Enter the full URL in this field. With this option, the website or other file will open in the viewer used for displaying the movie (e.g., MadCap Movie Viewer, MadCap Help Viewer).
  • OpenWithDefault  Read more… Select this option to open a remote file (such as a website) when the user clicks the button or other object. When you select this option, the URL field appears at the bottom of the tab. Enter the full URL in this field. With this option, the website or other file will open in the default Windows application for the file type (e.g., HTM files open in Internet Explorer, DOC files open in Microsoft Word).
  • Perform action on "Double Click"  Read more… Select this option if you want the action to be performed only when the user double-clicks the button. Otherwise, the action will be performed when the user single-clicks the button.
  • Argument  Read more… This field will change to "Frame Name," "Section Name," or "URL" when you select "GotoFrame," "GotoSection," "Open," or "OpenWithDefault" (see below).
  • Frame Name  Read more… If you selected the "GotoFrame" option, enter the number or name of the destination frame in this field. (The name of the frame is identified above it in the Frames window pane. You can change the name in the Frame Properties dialog.)
  • Section Name  Read more… If you selected the "GotoSection" option, select the name of the destination movie from this field.
  • URL  Read more… If you selected the "Open" or "OpenWithDefault" option, enter the full URL in this field.
