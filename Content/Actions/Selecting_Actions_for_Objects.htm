Selecting Actions for Objects

how to select an action for an object
  1. Open the frame.
  2. In the Frame Editor, double-click the object. The properties dialog for the object opens.

  3. If the object is a button, select the Button Action tab. Otherwise, select the Action tab. Then select the necessary action(s).

  6. (Optional) If you want the frame to be paused until the user clicks the object on the frame, do the following.
    1. Double-click the frame.
    2. In the Frame Properties dialog, select the Transition tab.
    3. Click Pause movie at end of frame.
