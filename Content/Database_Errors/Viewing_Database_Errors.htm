how to view database errors

    • Ribbon In the Analysis section select Project Analysis>Database Errors.

    • Menu Select View>Project Analysis>Database Errors.

    The Project Analysis window pane opens, with the Database Errors option selected from the drop-down menu at the top. The area at the bottom of the window pane displays all database errors found.

  2. Open a project.
  3. If you have the Analyzer Summary window pane in view (see Summary Window Pane), double-click the row that mentions new database errors. The Database Errors window pane opens.
