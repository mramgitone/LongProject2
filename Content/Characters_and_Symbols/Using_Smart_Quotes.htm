Using Smart Quotes

By default, adds straight quotation marks (") when you type them in the XML Editor. However, you can select an option to automatically use single and double curly smart quotes (”) instead when you work in the editor. This option is disabled by default.

By default, adds straight quotation marks (") when you type them in the Translation Editor. However, you can select an option to automatically use single and double curly smart quotes (”) instead when you work in the editor. This option is disabled by default.

how to use smart quotes

    • Ribbon Select File>Options.

    • Menu Select Tools>Options.

    The Options dialog opens.

  2. Select the XML Editor tab.
  3. Select the Interface tab.
  4. Click Replace straight quotes with smart quotes.
 note

Selecting this option does not automatically replace all of the straight quotes that you already have throughout your project. In order to do that, you should use the Find and Replace in Files window pane to find all of the files in the project containing straight quotes and then enter smart quotes in their place. See Using the Find and Replace in Files Feature.
