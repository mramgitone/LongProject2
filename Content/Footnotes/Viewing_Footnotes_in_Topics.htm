Viewing Footnotes in Topics

 See About Footnotes.

how to view all footnotes for a topic
  1. Open a topic.

    • Ribbon   In the Print Layout  section, select Footnotes.

    • Menu Select Tools>Footnotes.

      The Footnotes window pane opens, listing each footnote in the topic, with information about the position, sequence, and comment text of each. You can also change the position or sequence of any of the footnotes by using the drop-down fields in the local toolbar. See Editing Footnote Comment Position and Restarting Footnote Numbering.
