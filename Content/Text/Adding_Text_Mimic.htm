Adding Text

If you want the text to appear by itself, you can edit the properties of the object accordingly so that the object border or background cannot be seen (e.g., specify no border or color for the shape). Otherwise, the object will be shown with the text inside.

how to add text to an object
  1. Open the frame.
  2. Create the object (e.g., shape, button) that will contain the text.

    • Ribbon In the Advanced section click the Object Properties button. In the dialog that opens select the Rich Text tab.

    • Menu Select Edit>Edit Text. A small popup window opens, displaying the text.
    • Local Toolbar Click and in the dialog that opens, select the Rich Text tab.
    • Keyboard Shortcut Press F2. A small popup window opens, displaying the text.
    • Right-Click Right-click the object and from the context menu select Edit Text. A small popup window opens, displaying the text.
    • Right-Click Right-click the object and from the context menu select Properties. In the dialog that opens select the Rich Text tab.
    • Double-Click Double-click the object. In the dialog that opens select the Rich Text tab.

  4. Type the content. You can use the buttons and drop-downs at the top of the popup window or tab to apply bold, italic, or underline formatting to any part of the text. You can also insert variables, or change the color, font, and size for the text.

    Alternatively, you can use the Font tab in some properties dialogs to set the font properties. The Font tab applies the format properties to all of the text in the object, whereas the Rich Text tab applies the format properties only to selected text in the object.

    notes

    Another way to change the formatting for all of the content in the object is to click the object and use 's Format toolbar.

    In the properties dialog, you can use the other tabs to set properties for the object and change its look (e.g., border, fill color, shadow). If you want to make the object invisible so that only the text displays, select the Appearance tab. Then make sure a background color is not selected, and set the Line:Width field to 0.
  5. If you used the popup window, click off of it when you are finished adding text. If you used a properties dialog, click OK when you are finished.
  6. (Optional) You can resize the object by clicking and dragging any of the points around its edge.

 note

You can later edit the text if you want. This is possible because works with layers, rather than merging the text into the frame.
