About Text

You can easily incorporate text into your images. You can easily incorporate text into your movies in order to explain the various parts of a movie or to provide instructions for end users. Following are the primary tasks regarding text.

  • Adding Text Read more… You can add text to an object (e.g., bubble callout, rectangle) by simply clicking on the object and typing. You can also use the properties dialog for the object. If you want the text to appear by itself, you can edit the properties of the object accordingly so that the object border or background cannot be seen (e.g., specify no border or color for the shape). Otherwise, the object will be shown with the text inside. See Adding TextAdding Text.
  • Aligning Text Read more… After you add text to an object, you can easily align the text within the object. See Aligning Text.
  • Editing Text Read more… After you add text to an object, you can make changes to it as necessary. See Editing Text.
  • Setting Font Properties for Text Read more… After you add text to an object, you can adjust the font properties (e.g., font size, color). In the properties dialog for the selected object, you can set the font properties by using either the Font tab, the Rich Text tab, or the Text tab (depending on the type of object selected). See Setting Font Properties for TextSetting Font Properties for Text.
    • Font Tab Read more… Use the Font tab if you want the properties to be set for all text in the object.
    • Rich Text Tab Read more… Use the Rich Text tab if you want to specify settings for only portions of the text in the object.
    • Text Tab Read more… For certain objects, such as buttons, you must use the Text tab, which is similar to the Font tab. The difference is that the Text tab also lets you add the actual text, whereas the Font tab is used only for setting text properties.
