Setting Font Properties for Text

  • Font Tab Use the Font tab if you want the properties to be set for all text in the object.
  • Rich Text Tab Use the Rich Text tab if you want to specify settings for only portions of the text in the object.
  • Text Tab For certain objects, such as buttons, you must use the Text tab, which is similar to the Font tab. The difference is that the Text tab also lets you add the actual text, whereas the Font tab is used only for setting text properties.
how to set font properties by using the font tab
  1. In the Frame Editor, double-click the shape containing text that you want to modify. The properties dialog for that shape opens.
  2. Select the Font tab.
  3. Make changes using any of the fields in the tab.
    • Color Click the down arrow to select a color to apply to the text.
    • Bold Select this check box to apply bold to the text.
    • Italic Select this check box to apply italics to the text.
    • [Font Family] Select the font family for the text.
    • [Font Size] Select a font size for the text.
 how to set font properties by using the rich text tab
  1. In the Frame Editor, double-click the shape containing text that you want to modify. The properties dialog for that shape opens.
  2. Select the Rich Text tab.
  3. In the bottom portion of the tab, select the text to which you want to apply the formatting changes.

  4. Make changes using any of the fields in the tab.

    Click this button to apply bold to the selected text.

    Click this button to apply italics to the selected text.

    Click this button to underline the selected text.

    Click the down arrow to choose a color to apply to the selected text. To see advanced color options, select More colors.

    Choose the font family for the selected text.

    Choose a font size for the selected text.
how to set font properties by using the text tab
  1. In the Frame Editor, double-click the button containing text that you want to modify. The properties dialog for that button opens.
  2. Select the Text tab.
  3. Make changes using any of the fields in the tab.
    • Color Click the down arrow to select a color to apply to the text.
    • Bold Select this check box to apply bold to the text.
    • Italic Select this check box to apply italics to the text.
    • [Font Family] Select the font family for the text.
    • [Font Size] Select a font size for the text.
  4. In the Start Text field, enter or change the actual text to be displayed in the button.
 note

You can also format the object text by selecting the object and using the buttons in the Object ribbon, the Format toolbar (View>Toolbars>Format or click the Frame Editor local toolbar) or the options from the Object menu. This will change the formatting for all text in the object (similar to the Font and Text tabs).
