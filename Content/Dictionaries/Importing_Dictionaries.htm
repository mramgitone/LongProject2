Importing Dictionaries

Capturexxx

When you create a new project in , you select a language (see About Language Support). Several of the languages listed in the Wizard (i.e., those in bold) come with built-in spell checking support. You can see which dictionaries are installed on your computer by opening the Options dialog and selecting the Dictionaries tab. See .

Green dots indicate which dictionaries are supported by spell checkers and hyphenators. A hyphenatorautomatically inserts separators in an editor based on particular word patterns, as outlined in the dictionary file. Hyphenators are important only for certain languages.

You can also use the Options dialog to import more dictionaries that you have downloaded.

how to import a dictionary
  1. Download a dictionary from OpenOffice.org.

    • Ribbon Select File>Options.

    • Menu Select Tools>Options.

    The Options dialog opens

  3. Select the Dictionaries tab.

  4. At the bottom of the tab, click Import Dictionaries.

  5. In the dialog that opens, locate and double-click the dictionary that you downloaded. The dictionary is added to the Options dialog.
 notes

For more information about hyphenation patterns in different languages, see http://hunspell.sourceforge.net/tb87nemeth.pdf.

Dictionaries are stored in your AppData Windows folder.
