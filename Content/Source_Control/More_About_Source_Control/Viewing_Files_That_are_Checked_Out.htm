Viewing Files That are Checked Out

how to view files that you and others have checked out—pending check-ins window pane

    • Ribbon In the Changes section select Pending Check-Ins.

    • Menu Select View>Pending Check-Ins.

    The Pending Check-Ins window pane opens.

  2. (Optional) You can use the Filter field to limit the files that are displayed.

  3. (Optional) You can use the following toggle buttons in the local toolbar to limit the files that are displayed.

    This filters the Pending Check-Ins window pane to show or not show files that other users have checked out.

    This filters the Pending Check-Ins window pane to show or not show files that are out of date.

  4. Take note of the Status and User columns. (You may need to use the scroll bar to view these columns.)
