Viewing the History of Source Control Files

One of the benefits of 's integrated source control is that you can view the history for a particular file, including who checked in the file and when it was checked in. You can also view differences between different versions of the file and roll back to an older version if necessary. For more information see About Source Control and Multi-Authoring, Viewing Differences in Source Control Files, and Rolling Back to an Earlier Version of a File.

how to view the history of a source control file
  1. In one of the window panes (e.g., Content Explorer, Project Organizer, File List window pane, Pending Check-Ins window pane), select the relevant files(s).
    • Pending Check-Ins Window Pane From the Source Control ribbon or View menu, open the Pending Check-Ins window pane. Select the files, right-click, and select Source Control>View History.

    • Ribbon In the File section select View History.

    • Menu Select File>Source Control>View History.

    • Standard Toolbar Click the down arrow next to the Source Control button and select View History.

  3. The History dialog opens. The following are explanations of the different parts of this dialog.

    view explanations
  4. In the History dialog, click Close.
