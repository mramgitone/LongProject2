Enabling and Disabling Instant Messaging

You can enable or disable the instant messaging feature in using the Options dialog. You also have the option of disabling instant messaging on your Exchange server. If you want to disable instant messaging entirely on your Exchange server, provide the second set of steps to your network administrator.

how to enable or disable instant messaging in the options dialog
    • Ribbon Select File>Options.
    • Menu Select Tools>Options.

    The Options dialog opens.

  2. Select the General tab.
  3. Select Enable Instant Messaging.
 how to disable instant messaging on the exchange server
  1. On your Exchange server, open the Active Directory.
  2. Select the user or group for whom you want to limit instant messaging and open the properties.
  3. Select the Exchange Advanced tab.
  4. Click Custom Attributes.
  5. Select an attribute and click Edit.
  6. In the attribute field, enter either of the following.

    • MadCap: : im=off

      This command disables instant messaging for only. To disable instant messaging for other applications, change "" to another application name.

    • MadCap: *: im=off

      This command disables instant messaging in all MadCap products.

  7. Click OK to exit out of all dialogs.
