About Instant Messaging

Following are some of the tasks you can perform when it comes to instant messaging.

  • Send an instant message Read more… See Sending Instant Messages.
  • Reply to an instant message Read more… See Replying to Instant Messages.
  • Select an instant message status Read more… See Selecting an Instant Messaging Status.
  • View instant messages Read more… See Viewing Instant Messages.
  • View other instant message users Read more… See Viewing Instant Messaging Users.
  • Request a user to check in source control files Read more… See Requesting a User to Check In Source Control Files.
  • Request a user to view source control files Read more… See Requesting a User to View Files.
  • Disable Read more… If necessary, your network administrator can disable 's instant messaging feature. This can be done for , as well as any other MadCap products that use instant messaging. See Disabling Instant Messaging.

Many of the tasks listed above can be performed from the Instant Messages window pane, which can be opened by selecting Source Control>Instant Messages. However, you do not need to have this window pane open or visible to know when you have received an instant message. You don't even need to have a project open. When another user sends you an instant message, you will receive a popup on your screen.
