About Source Control

If your project is bound to source control, you can perform the following related tasks in . For more information on each of these, see the online Help.

  • Add Files Read more…
  • Check In Files Read more…
  • Merging Read more… If you are set up in a source control application to allow multiple authors to have the same files checked out, there may be times when you need to merge changes from different authors when checking in a file. The merge occurs automatically if there are no conflicting changes (i.e., changes do not occur in the same location). If there are conflicting changes, a dialog opens, allowing you to determine how changes are merged. See Merging Source Control Files.
