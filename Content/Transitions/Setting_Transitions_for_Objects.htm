Setting Transitions for Objects

how to set the transitions for an object
  1. Open the frame.
  2. In the Frame Editor, double-click the object. The properties dialog for that object opens.
  3. Select the Animation Effects tab.

  4. Change the settings in any of the following fields.

    in effect

    These options control how the object comes into view.

    • Fade Select this option if you want the object to fade into view when it opens.
    • Reveal Select this option if you want the object to appear from a particular direction when it opens. You can set the direction by clicking the Effect Properties button.

    • Effect Properties Select this button to open the Effect Properties dialog, which lets you provide settings for the selected transition effect (e.g., duration and direction).

      • Duration Enter the number of seconds that you want the effect to be displayed.
      • Direction Select the direction that you want the effect to move.
         out effect

    These options control how the object leaves from view.

    • Fade Select this option if you want the object to fade out of view when it closes.
    • Reveal  Select this option if you want the object to leave the frame from a particular direction when it closes. You can set the direction by clicking the Effect Properties button.

    • Effect Properties Select this button to open the Effect Properties dialog, which lets you provide settings for the selected transition effect (e.g., duration and direction).

      • Duration Enter the number of seconds that you want the effect to be displayed.
      • Direction Select the direction that you want the effect to move.
note

You can also set the transitions for all frames in a movie and for individual frames in a movie. See Setting Transitions for Frames and Setting Transitions for Frames.
