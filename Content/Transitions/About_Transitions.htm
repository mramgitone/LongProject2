About Transitions

You can control the way that frames and objects come into view and leave from view (transitions). This can be done in various ways.

 tasks associated with transitions
  • Set the Transitions for Movies The frames in a movie are each displayed for a certain period of time before the next frame displays. You can control the transitions for all frames in the movie. For example, you can specify that all frames should fade in and out. See Setting Transitions for Movies.
  • Set the Transitions for Frames This is the same as setting the transitions for movies. The difference is that this method lets you set the transitions for individual frames in the movie, rather than for all frames. See Setting Transitions for Frames.
  • Set the Transitions for Objects An object added to a frame is displayed for a certain period of time. You can control the transitions used for a particular object. For example, you can specify that an object should fade in when it comes into view and fade out when it disappears. See Setting Transitions for Objects.
